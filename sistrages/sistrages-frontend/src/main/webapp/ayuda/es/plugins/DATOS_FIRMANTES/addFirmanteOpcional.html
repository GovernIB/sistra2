addFirmanteOpcional(String nif, String nombre): void

DESCRIPCIÓN

  • Método que añade un firmante opcional. De los firmantes que se añaden con el método addFirmanteOpcional(), al menos uno tendrá que firmar obligatoriamente.

PARÁMETROS DE ENTRADA

  • nif: String con el nif del firmante. Tiene que ser un NIF válido.
  • nombre: String con el nombre del firmante. Dato obligatorio.

EJEMPLOS

Ejemplo 1.- Pongamos por caso que queremos establecer dos firmantes para un formulario, uno obligatorio (A) y otro opcional (B). En este caso cumplimentariamos el script "Establecer firmantes" con el contenido siguiente:

DATOS_FIRMANTES.addFirmante(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniB, nomB);

El resultado sería que el asistente de tramitación solicitará que el formulario sea firmado por A obligatoriamente.

Exemple 2.- Si queremos establecer dos firmantes para un formulario y es indiferente quién es el firmante obligatorio (A o B), rellenaremos el script "Establecer firmantes" con el contenido siguiente:

DATOS_FIRMANTES.addFirmanteOpcional(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniB, nomB);

El resultado sería que el asistente de tramitación solicitará que el formulario sea firmado por A o por B obligatoriamente.
