esDni(String dni): boolean

DESCRIPCIÓN

  • Valida si el valor introducido posee el formato para ser un DNI válido. Se considera válido toda aquella cadena que tenga 8 números y un carácter alfanumérico que cumpla con la regla de validación.

PARÁMETROS DE ENTRADA

  • dni: cadena que representa el DNI a validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena es un DNI válido y false si la cadena no es un DNI válido.

EJEMPLOS

Ejemplo 1 Validar la cadena '79865432D'.

var valido = PLUGIN_UTILS.esDni('79865432D');

La función esDni retorna 'true', debido a que la cadena contiene un DNI válido.


Ejemplo 2 Validar la cadena '4854391C.

var valido = PLUGIN_UTILS.esDni('4854391C');

La función esDni retorna 'false', debido a que el valor de la cadena no se corresponde a un DNI válido.
