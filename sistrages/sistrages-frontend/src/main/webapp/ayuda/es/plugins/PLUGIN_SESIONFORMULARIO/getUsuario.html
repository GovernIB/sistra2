getUsuario(): DATOS_USUARIO

DESCRIPCIÓN

  • Método que retorna el objeto 'DATOS_USUARIO' con la información del usuario.
  • Sólo es posible su uso en trámites autenticados.

PARÁMETROS DE SALIDA

  • DATOS_USUARIO: Retorna el objeto 'DATOS_USUARIO' con la información del usuario.

EJEMPLO

Supongamos que deseamos obtener el nombre del usuario de la sesión actual (solo para trámites autenticados)

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado(); if (autenticado == true) { var usuario = PLUGIN_SESIONFORMULARIO.getUsuario(); }

La función getUsuario retorna el objeto 'DATOS_USUARIO' a partir del cual extraer la información que nos interesa, por ejemplo, el nombre mediante la siguiente función:

var nombre = usuario.getNombre();
