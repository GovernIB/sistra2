setContribuyente(String nif, String nombre): void

DESCRIPCIÓN

  • Método que permite establecer un contribuyente en el pago. Por defecto se usará el usuario iniciador del trámite

PARÁMETROS DE ENTRADA

  • nif: NIF del contribuyente. Tiene que ser un NIF válido.
  • nombre: Nombre del contribuyente.

EJEMPLOS

Si deseamos establecer el contribuyente de un pago, incluríamos la línea de código siguiente en el script:

DATOS_PAGO.setContribuyente('11111111H','Pedro Álvarez Sánchez');
