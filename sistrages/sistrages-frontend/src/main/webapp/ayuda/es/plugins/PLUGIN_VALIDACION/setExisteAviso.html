setExisteAviso(boolean existeAviso, String tipo)

DESCRIPCIÓN

  • Método para establecer si se genera aviso.

PARÁMETROS DE ENTRADA

  • existeAviso: establecer si se genera aviso.
  • tipo: INFO / WARNING / ERROR

EJEMPLOS

Vamos a proceder a indicar que existe un aviso de tipo WARNING:

PLUGIN_VALIDACION.setExisteAviso(true, 'WARNING');
