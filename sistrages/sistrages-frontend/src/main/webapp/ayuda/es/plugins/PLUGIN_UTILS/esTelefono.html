esTelefono(String telefono): boolean

DESCRIPCIÓN

  • Valida si el valor introducido cumple con las reglas de validación necesarias para ser un número de teléfono fijo o móvil válido.
  • Se considera válida toda aquella cadena no nula que tenga 9 números y que empiece por 6, 7 o 9.

PARÁMETROS DE ENTRADA

  • telefono: Cadena que representa el número de teléfono.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena representa un teléfono válido y false si la cadena no representa un teléfono válido

EJEMPLOS

Ejemplo 1 Validar la cadena '9711714'.

var valido = PLUGIN_UTILS.esTelefono('9711714');

La función esTelefono retorna 'false', debido a que el número de teléfono debe tener como mínimo 9 dígitos.


Ejemplo 2 Validar la cadena '971177714'.

var valido = PLUGIN_UTILS.esTelefono('971177714');

La función esTelefono retorna 'true', debido a que el teléfono se corresponde con el formato de un número de teléfono fijo.


Ejemplo 3 Validar la cadena '663467284'.

var valido = PLUGIN_UTILS.esTelefono('663467284');

La función esTelefono retorna 'true', debido a que el teléfono se corresponde con el formato de un número de teléfono móvil.
