invocarDominio(String idDominio): ValoresDominio

DESCRIPCIÓN

  • Este método permite la invocació de un domini, asociado a la versión del trámite, a partir de su identificador.
  • Este método retorna un objecto que modeliza los valores retornados por el dominio.

PARÁMETROS DE ENTRADA

  • idDominio: Identificador del dominio a invocar.

PARÁMETROS DE SALIDA

  • ValoresDominio: Objecto que modeliza los datos retornados por el dominio.

EJEMPLOS

Invocación de un dominio, con identificador Provincias, que retorna el valor de las provincias de España:

var dProvincias = PLUGIN_DOMINIOS.invocarDominio('Provincias');

Otro ejemplo podría ser la invocación de un dominio que retorne el listado de paises:

var dPaises = PLUGIN_DOMINIOS.invocarDominio('Paises');
