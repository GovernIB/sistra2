setSolicita(String textoSolicita): void

DESCRIPCIÓN

  • Método que permite establecer un texto del objeto de la solicitud. Este valor únicamente tiene sentido para solicitudes genéricas.

PARÁMETROS DE ENTRADA

  • textoSolicita: cadena de texto del objeto de la solicitud.

EJEMPLOS

Pongamos por caso que estamos desarrollando una solicitud genérica y queremos enviar a registro el texto objeto de la solicitud. Para ello podríamos referenciar un campo (p.ej. SOLICITA) del formulario (p.ej. FORM1) en el que el ciudadano ha cumplimentado dicho texto:

var textoSolicita = PLUGIN_FORMULARIOS.getValor('FORM1','SOLICITA'); DATOS_REGISTRO.setSolicita(textoSolicita);
