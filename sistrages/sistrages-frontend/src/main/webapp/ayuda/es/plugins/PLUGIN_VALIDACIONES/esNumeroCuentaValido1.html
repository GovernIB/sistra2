esNumeroCuentaValido(String cuenta): boolean

DESCRIPCIÓN

  • Valida que el valor introducido no sea nulo, y posea el formato correspondiente a un número de cuenta corriente, cumpliendo con las reglas de validación necesarias para ser un NCC (Número Cuenta Corriente) Válido. Se considera válida toda aquella cadena que tenga 20 números que cumplan con las reglas de validación correspondientes a entidad, oficina, dígito de control y número de cuenta.

PARÁMETROS DE ENTRADA

  • cuenta: Código de 20 dígitos del Número de cuenta.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena introducida es un número de cuenta válido y false si la cadena introducida no es un número de cuenta válido

EJEMPLOS

Ejemplo 1 Validar la cadena '12345678901234567890'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaValido('12345678901234567890');

La función esNumeroCuentaValido retorna 'false' debido a que la cadena no cumple con las reglas de validación, el dígito de control no es válido.


Ejemplo 2 Validar la cadena '12345678061234567890'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaValido ('12345678061234567890');

La función esNumeroCuentaValido retorna 'true', debido a que la cadena cumple los requisitos para ser un número de cuenta válido.
