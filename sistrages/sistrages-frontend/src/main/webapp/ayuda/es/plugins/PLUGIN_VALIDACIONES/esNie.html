esNie(String nie): boolean

DESCRIPCIÓN

  • Valida si el valor introducido es no nulo, y posee el formato y validaciones necesarias para ser un NIE (Número Identificación Extranjería) Válido. Se considera válida toda aquella cadena que tenga 9 números y/o carácteres alfanuméricos que cumplan con la regla de validación.

PARÁMETROS DE ENTRADA

  • nie: NIE que se quiere validar.

PARÁMETROS DE SALIDA

  • boolean : true si la cadena es un NIE válido y false si la cadena no es un NIE válido.

EJEMPLOS

Ejemplo 1 Validar la cadena 'Y23459781X'.

var valido = PLUGIN_VALIDACIONES.esNie('Y23459781X');

La función esNie retorna 'true', debido a que el valor indicado corresponde a un NIE.


Ejemplo 2 Validar la cadena 'A623459781.

var valido = PLUGIN_VALIDACIONES.esNie('A623459781');

La función esNie retorna 'false', debido a que el valor indicado no corresponde a un NIE válido.
