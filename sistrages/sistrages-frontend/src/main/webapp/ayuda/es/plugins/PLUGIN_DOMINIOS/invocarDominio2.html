invocarDominio(String idDominio, ParametrosDominio parametros): ValoresDominio

DESCRIPCIÓN

  • Método que realiza la invocación de un dominio y recupera los datos que retorna. Se utiliza este método para aquellos dominios que requieran el paso de parámetros.
  • El número de parámetros que necesita un dominio se puede identificar en la pantalla de edición del dominio.

PARÁMETROS DE ENTRADA

  • idDominio: Identificador del dominio.
  • parametrosDominio: Lista de parámetros que se necesita para invocar al dominio.

PARÁMETROS DE SALIDA

  • ValoresDominio: Corresponde a los valores del dominio.

EJEMPLOS

De manera previa a la invocación de un dominio, se ha de crear en el script la lista de parámetros necesarios para la invocación, para posteriormente añadírselos al dominio. Per ejemplo, en el supuesto que necesitemos recuperar las localidades de la provincia Islas Baleares, se identificaría la provincia con el parámetro correspondiente, en este caso 07:

var parametrosDominio = PLUGIN_DOMINIOS.crearParametros(); parametrosDominio.addParametro('1', '07');

Posteriormente se invocará el dominio Localidades añadiendo la lista de parámetros al método de invocación:

var dLocalidades = PLUGIN_DOMINIOS.invocarDominio('Localidades', parametrosDominio);
