getValorPdf(String codigo): String

DESCRIPCIÓN

  • Método que devuelve el valor de un campo de un PDF anexado. Para poder utilizar este método el PDF anexado deberá ser un formulario PDF.

PARÁMETROS DE ENTRADA

  • codigo: código del campo del pdf anexado.

PARÁMETROS DE SALIDA

  • String: Valor del campo pdf anexado. Si no existe el campo devuelve una cadena vacía.

EJEMPLO

Para poder hacer uso de esta funcionalidad debemos tener en cuenta lo siguiente:

  • El usuario habrá anexado un PDF generado a partir de una plantilla de la que conozcamos el código del campo a obtener.
  • Debemos conocer de antemano el código del campo editable que se encuentra en el PDF anexado. Por norma general, solo conoce el código de campo el creador de la plantilla.

Si el ciudadano ha adjuntado un documento PDF, en el que sabemos que existe un campo al que se le ha asignado el código 'domicilio', podemos obtener el valor de ese campo mediante la siguiente sentencia:

var domicilio = PLUGIN_VALIDACIONANEXO.getValorPdf('domicilio');
