esNumeroCuentaValido(String entidad, String sucursal, String dc, String cuenta): boolean

DESCRIPCIÓN

  • Valida si los valores introducidos no son nulos, y cumple con las reglas de validación necesarias para ser un NCC (Número Cuenta Corriente) Válido. Se consideran válidas toda aquella cadena que tengan 4 números para la entidad, 4 números para la sucursal, 2 números para los dígitos de control y 10 números para la cuenta corriente y que cumplan con las reglas de validación correspondientes a la combinación de entidad, oficina, dígito de control y número de cuenta.

PARÁMETROS DE ENTRADA

  • entidad: Código de 4 dígitos de la entidad bancaria.
  • sucursal: Código de 4 dígitos de la sucursal.
  • dc: Código de 2 dígitos para los dígitos de control.
  • cuenta: Código de 10 dígitos del Número de cuenta.

PARÁMETROS DE SALIDA

  • boolean: true si entidad+sucursal+dc+cuenta forman un número de cuenta válido y false si entidad+sucursal+dc+cuenta no forman un número de cuenta válido.

EJEMPLOS

Ejemplo 1 Validar los siguientes valores: entidad: '1234', sucursal: '5678', dc: '90', cuenta: '1234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido('1234', '5678', '90', '1234567890');

La función esNumeroCuentaValido retorna 'false' ya que los parámetros introducidos no forman un número de cuenta válido, el dígito de control no es correcto..


Ejemplo 2 Validar los siguientes valores: entidad: '1234', sucursal: '5678', dc: '06', cuenta: '1234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido ('1234', '5678', '06', '1234567890');

La función esNumeroCuentaValido retorna 'true', ya que los parámetros introducidos forman un número de cuenta válido.
