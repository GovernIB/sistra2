getIdioma(): String

DESCRIPCIÓN

  • Método que retorna el idioma de la sesión de la actual tramitación.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres que contiene el idioma usado en la sesión de la actual tramitación, siendo el valor 'es' para el español, 'ca' para el catalán o 'en' para el inglés.

EJEMPLO

Obtener el idioma usado por el usuario en la actual tramitación, en este caso catalán

var idioma = PLUGIN_SESIONFORMULARIO.getIdioma();

La función getIdioma retorna la cadena de caracteres 'ca'.
