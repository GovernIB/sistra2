formateaIntACadena(int numero, String separadorMiles): String

DESCRIPCIÓN

  • Método para convertir un número de tipo int en un string

PARÁMETROS DE ENTRADA

  • numero: Número int a convertir en String .
  • separadorMiles: Carácter con el que se identifica el separador de miles.

PARÁMETROS DE SALIDA

  • String: Cadena de carácteres que representa el numero introducido.

EJEMPLOS

Ejemplo 1 Convertir el número 123110544434.58 en una cadena de carácteres

var pCadena = PLUGIN_UTILS.formateaIntACadena(123110544434, '.');

La función formateaIntACadena retorna la cadena '123.110.544.434'.
