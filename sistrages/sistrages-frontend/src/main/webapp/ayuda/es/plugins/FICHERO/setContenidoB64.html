setContenidoB64(String contenidoB64):void

DESCRIPCIÓN

  • Método que permite establecer el contenido en Base64 de un fichero.
  • Este método está disponible desde los métodos de los plugins de DATOS_PLANTILLA.addFichero (desde el script de Datos plantilla del paso información) y ValoresDominio.getFichero (desde todos los scripts a través del plugin PLUGIN_DOMINIOS).

PARÁMETROS DE ENTRADA

  • contenidoB64: String con el contenido en Base64 del fichero.

EJEMPLOS

Pongamos por caso que queremos cambiar el contenido de un fichero retornado por un dominio 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se recupera un fichero mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fichero = valores.getFichero('PDF');

y asignamos el nuevo contenido al fichero de la siguiente forma:

fichero.setContenidoB64('xxxxxxx');
