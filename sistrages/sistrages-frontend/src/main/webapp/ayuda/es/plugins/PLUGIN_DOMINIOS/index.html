PLUGIN_DOMINIOS

DESCRIPCIÓN

  • Plugin que permite el acceso a los dominios asociados a la versión del trámite.
  • Este plugin está disponible para su utilización desde todos los scripts de versión, paso y formulario.

MÉTODOS

  • crearParametros(): ParametrosDominio
  • invocarDominio(idDominio): ValoresDominio
  • invocarDominio(idDominio, parametros): ValoresDominio
