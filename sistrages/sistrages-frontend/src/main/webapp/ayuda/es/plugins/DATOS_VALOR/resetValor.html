resetValor(): void

DESCRIPCIÓN

  • Método que elimina todos los valores asociados a este objeto.

EJEMPLOS

Para ver la funcionalidad de esta función, primero añadimos elementos al actual campo, en este caso, el campo, indica el municipio de un usuario:

var municipios = DATOS_VALOR.crearValorMultiple(); municipios.addValor('0337','Manacor'); municipios.addValor('0407','Palma de Mallorca'); municipios.addValor('2060','Eivissa'); DATOS_VALOR.setValorMultiple('municipios', municipios);

ahora el campo actual tiene 3 valores correspondientes a municipios de les Illes Balears, si en este punto podemos inferir de otros campos que el usuario no pertenece a Les Illes Balears, existe la opción de borrar todos los valores mediante la siguiente sentencia:

DATOS_VALOR.resetValor();
