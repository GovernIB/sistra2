getNombre():String

DESCRIPCIÓN

  • Método que permite recuperar el nombre con extensión de un fichero.

PARÁMETROS DE SALIDA

  • String: String con el nombre y extensión del fichero.

EJEMPLOS

Pongamos por caso que se quiere recuperar el nombre de un fichero retornado para un dominio 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se recupera un fichero mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fichero = valores.getFichero('PDF');

y se recupera su nombre de la siguiente forma:

var contenido = fichero.getNombre();
