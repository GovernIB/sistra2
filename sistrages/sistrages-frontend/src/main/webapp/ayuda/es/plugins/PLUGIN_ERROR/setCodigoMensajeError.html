setCodigoMensajeError(String codigo):void

DESCRIPCIÓN

  • Método que permite recuperar el mensaje de error a partir de un código del error, que tendrá que estar previamente definido.
  • Los mensajes se definen en la pantalla del script, en la pestaña de mensajes validación.

PARÁMETROS DE ENTRADA

  • codigo: String con el identificador del mensaje de error.

EJEMPLOS

Pongamos por caso que queremos referenciar desde el script un mensaje de error, previamente definido en la pestaña de mensajes de validación, con el identificador 'CodErrFechas'.

El mensaje de error se fijaría a nivel de script mediante el código siguiente:

PLUGIN_ERROR.setCodigoMensajeError('CodErrFechas');
