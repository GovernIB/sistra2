DATOS_PARAMETROSINICIALES

DESCRIPCIÓN

  • Por defecto los parámetros iniciales serán los pasados el parámetro "parametros" en la url de inicio del trámite. En este script se permite añadir dinámicamente parámetros iniciales (p.e. calculados a partir de los parámetros que vienen en la url de inicio).
  • Los parámetros iniciales estarán disponibles para el resto de scripts del flujo a través de PLUGIN_SESIONTRAMITACION.getParametroInicio(idParametro).

MÉTODOS

  • addParametro(codigo, valor)
