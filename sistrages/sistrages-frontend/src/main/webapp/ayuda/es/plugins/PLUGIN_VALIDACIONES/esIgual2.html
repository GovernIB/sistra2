esIgual(String cadena1, String cadena2, boolean ignoreCase): boolean

DESCRIPCIÓN

  • Valida si las dos cadenas introducidas son iguales. Se considerarán iguales todas aquellas cadenas que coincidan en todos sus caracteres, o si ambas son nulas. Mediante el parámetro ignoreCase se configura la función para que no haga distinción entre mayúsculas y minúsculas.

PARÁMETROS DE ENTRADA

  • cadena1: Primera cadena para comparar.
  • cadena2: Segunda cadena para comparar.
  • ignoreCase: Si es true no hace distinción entre mayúsculas y minúsculas. Si es false hace distinción entre mayúsculas y minúsculas.

PARÁMETROS DE SALIDA

  • boolean : true si las cadenas son iguales y false si las cadenas no son iguales.

EJEMPLOS

Ejemplo 1 Validar si las cadenas 'abc' y 'ABC' son iguales. Indicamos que no se haga distinción entre mayúsculas y minúsculas

var iguales = PLUGIN_VALIDACIONES.esIgual ('abc', 'ABC', true);

La función esIgual retorna 'true', debido a que el valor asignado para cada cadena es coincidente.


Ejemplo 2 Validar si las cadenas 'abc' y 'ABC' son iguales. Indicamos que se haga distinción entre mayúsculas y minúsculas

var iguales = PLUGIN_VALIDACIONES.esIgual ('abc', 'ABC', false);

La función esIgual retorna 'false', debido a que el valor asignado para cada cadena NO es coincidente.
