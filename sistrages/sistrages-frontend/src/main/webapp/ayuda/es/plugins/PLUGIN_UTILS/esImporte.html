esImporte(String importe): boolean

DESCRIPCIÓN

  • Valida si el valor introducido es no nulo, numérico positivo y con entre cero y dos decimales. Ej: '0,00'

PARÁMETROS DE ENTRADA

  • importe: Importe a validar.

PARÁMETROS DE SALIDA

  • boolean: true si el importe es válido y false si el importe no es válido.

EJEMPLOS

Ejemplo 1 Validar una cadena de caracteres nula.

var importe;
var valido = PLUGIN_UTILS.esImporte(importe);

La función esImporte retorna 'false', debido a que el valor no es un importe válido.


Ejemplo 2 Validar la cadena de caracteres 45,20.

var valido = PLUGIN_UTILS.esImporte('45,20');

La función esImporte retorna 'true', debido a que el valor es un importe correcto.


Ejemplo 3 Validar la cadena de caracteres 98,444.

var valido = PLUGIN_UTILS.esImporte('98,444');

La función esImporte retorna 'false', debido a que el valor no está correctamente formateado
