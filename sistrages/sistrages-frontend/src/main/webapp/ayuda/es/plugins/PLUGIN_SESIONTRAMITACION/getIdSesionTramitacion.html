getIdSesionTramitacion(): String

DESCRIPCIÓN

  • Método que devuelve el id de sesión de la tramitación.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que contiene el id de sesión de tramitación.

EJEMPLO

Obtener el id sesión de la actual tramitación

var idSesionTram = PLUGIN_SESIONTRAMITACION.getIdSesionTramitacion();

La función idSesionTramitacion retorna la cadena de caracteres que representa el identificador de la sesión de tramitación actual.
