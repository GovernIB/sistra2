PLUGIN_DATOSFORMULARIO

DESCRIPCIÓN

  • Plugin de acceso a los datos de los campos del formulario.
  • Este plugin está disponible para cualquier script dentro de un formulario.

MÉTODOS

  • getValor(campo) : String
  • getValorCompuesto(campo) : VALOR_CAMPOCOMPUESTO
  • getNumeroValoresValorMultiple(campo) : int
  • getValorMultiple(campo,indice) : VALOR_CAMPOCOMPUESTO
