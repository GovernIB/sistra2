validaFechaHoraFin(String fecha1, String fecha2): int

DESCRIPCIÓN

  • Método que permite comparar 2 fechas. Las fechas deben tener el formato dd/MM/yyyy HH:mm:ss.

PARÁMETROS DE ENTRADA

  • fecha1: Cadena de texto que representa la primera fecha. El formato de la cadena debe ser dd/MM/yyyy HH:mm:ss.
  • fecha2: Cadena de texto que representa la segunda fecha. El formato de la cadena debe ser dd/MM/yyyy HH:mm:ss.

PARÁMETROS DE SALIDA

  • int: Número entero, los posibles valores a devolver son:
    1. 0: Si fecha1 y fecha2 son iguales.
    2. 1: Si fecha1 es superior a fecha2.
    3. -1: Si fecha1 es menor a fecha2.

EJEMPLOS

Ejemplo 1 Comparar las siguientes fechas '04 de junio de 2018, 10:30:12' y '11 de junio de 2018, 7:30:15'.

var valida = PLUGIN_VALIDACIONES.validaFechaHoraFin('04/06/2018 10:30:12', '11/06/2018 7:30:15');

La función validaFechaHoraFin retorna -1, ya que fecha2 es mayor que fecha1.


Ejemplo 2 Comparar las siguientes fechas '15 de mayo de 2018' y '12 de mayo de 2018 7:30:15'.

var pValida = PLUGIN_VALIDACIONES.validaFechaHoraFin('15/05/2018 10:30:12', '12/05/2018 7:30:15');

La función validaFechaHoraFin retorna 1, ya que fecha1 es mayor que fecha2.
