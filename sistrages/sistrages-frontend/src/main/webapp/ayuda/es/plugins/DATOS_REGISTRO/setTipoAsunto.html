setTipoAsunto(String tipoAsunto): void

DESCRIPCIÓN

  • Método que permite establecer el tipo de asunto a indicar en el apunte registral de entrada de las solicitudes asociadas a la versión del trámite.

PARÁMETROS DE ENTRADA

  • tipoAsunto: Identificador del tipo de asunto.

EJEMPLOS

Pongamos por caso que el tipo de asunto que queremos utilizar al registrar la solicitud depende de un valor del formulario. En tal caso utilizariamos el script para fijar el valor de dicha propiedad. Para ello incorporaríamos la siguiente línea de código:

DATOS_REGISTRO.setTipoAsunto('idTipoAsunto');
