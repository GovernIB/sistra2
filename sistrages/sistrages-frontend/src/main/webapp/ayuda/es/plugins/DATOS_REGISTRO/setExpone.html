setExpone(String textoExpone): void

DESCRIPCIÓN

  • Método que permite establecer un texto de exposición de hechos. Este valor únicamente tiene sentido para solicitudes genéricas.

PARÁMETROS DE ENTRADA

  • textoExpone: cadena de texto con los hechos a exponer.

EJEMPLOS

Pongamos por caso que estamos desarrollando una solicitud genérica y queremos enviar a registro el texto de exposición. Para ello podríamos referenciar un campo (p.ej. EXPONE) del formulario (p.ej. FORM1) en el que el ciudadano ha cumplimentado dicho texto:

var textoExpone = PLUGIN_FORMULARIOS.getValor('FORM1','EXPONE'); DATOS_REGISTRO.setNumeroExpediente(textoExpone);
