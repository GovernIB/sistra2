DATOS_REPRESENTACION

DESCRIPCIÓN

  • Objeto que permite establecer los datos representación. En caso de existir representación, será necesario establecer el representado. De forma opcional se podrá indicar el representante y de no indicarse se tomará el presentador como representante.
  • Si no se establece el script y se accede con certificado de representación, se establecerá por defecto esta representación.

MÉTODOS

  • setActivarRepresentacion(boolean)
  • crearRepresentante():Persona
  • crearRepresentado():Persona
