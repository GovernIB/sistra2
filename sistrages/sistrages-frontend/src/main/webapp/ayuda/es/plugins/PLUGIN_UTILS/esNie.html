esNie(String nie): boolean

DESCRIPCIÓN

  • Valida si el valor introducido posee el formato y validaciones necesarias para ser un NIE (Número de Identidad de Extranjero) válido. Se considera válida toda aquella cadena que tenga 9 carácteres alfanuméricos que cumpla con la regla de validación.

PARÁMETROS DE ENTRADA

  • nie: Cadena que representa el NIE que se quiere validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena es un NIE válido y false si la cadena no es un NIE válido.

EJEMPLOS

Ejemplo 1 Validar la cadena 'Y2345978S'.

var valido = PLUGIN_UTILS.esNie('Y2345978S');

La función esNie retorna 'true', debido a que el valor indicado corresponde a un NIE.


Ejemplo 2 Validar la cadena 'A623459781.

var valido = PLUGIN_UTILS.esNie('A623459781');

La función esNie retorna 'false', debido a que el valor indicado no corresponde a un NIE válido.
