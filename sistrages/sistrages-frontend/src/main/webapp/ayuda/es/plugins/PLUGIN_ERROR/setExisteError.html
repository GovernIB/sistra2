setExisteError(boolean existe):void

DESCRIPCIÓN

  • Método que permite marcar la existencia de un error.

PARÁMETROS DE ENTRADA

  • existe: booleano que ha de tenir el valor true si se quiere marcar el script con error, o false en caso contrario.

EJEMPLOS

Pongamos por caso que se quiere marcar el script como erróneo ante una situación concreta. El script se marcaría como erróneo de la manera siguiente:

PLUGIN_ERROR.setExisteError(true)
