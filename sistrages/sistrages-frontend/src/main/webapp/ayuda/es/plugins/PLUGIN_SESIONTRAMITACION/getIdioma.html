getIdioma(): String

DESCRIPCIÓN

  • Método que devuelve el idioma de tramitación.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que contiene el idioma de tramitación, siendo el valor 'es' para el español, 'ca' para el catalán o 'en' para el inglés.

EJEMPLO

Obtener el idioma con el cuál se está tramitando. En este caso Español

var idioma = PLUGIN_SESIONTRAMITACION.getIdioma();

La función getIdioma retornará 'es'.
