getMensaje(): String

DESCRIPCIÓN

  • Método que permet recuperar el valor del mensaje.

PARÁMETROS DE SALIDA

  • String: String con el texto del mensaje.

EJEMPLOS

Pongamos por caso que queremos recuperar el texto de un mensaje de validación, por ejemplo MENSAJE_001, previamente creado. Los pasos a seguir serían los siguientes:

En la pantalla de script se define un mensaje, en la pestaña de mensajes validación, con el identificador MENSAJE_001 con valor:

El valor del certificado no se corresponde con el valor cancelado.

Desde el script se puede referenciar ese mensaje de la siguiente forma:

var msg = PLUGIN_MENSAJES.crearMensaje('MENSAJE_001');

Para obtener el valor del mensaje en el script:

var mensaje = msg.getMensaje();
