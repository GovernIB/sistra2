ANEXO_DINAMICO

DESCRIPCIÓN

  • Objeto que modeliza un anexo que se puede añadir de forma dinámica.

MÉTODOS

  • setIdentificador(String identificador): Identificador del anexo (Obligatorio).
  • setDescripcion(String descripcion): Descripción del anexo (Obligatorio).
  • setExtensiones(String extensiones): Extensiones permitidas para el anexo. Lista de valores separados por comas. (Obligatorio).
  • setTamanyoMax(Strint tamMax): Tamaño máximo del anexo. Es opcional. Si se indica, se debe añadir la unidad al final del mismo. Valores posibles:
    1. KB
    2. MB
  • setUrlPlantilla(String urlPlantilla): Url en la que se puede descargar la plantilla del anexo. Dato opcional.
  • setObligatorio(boolean obligatorio): Variable booleana que permite marcar si el anexo es obligatorio o no.
  • setConvertirPDF(boolean convertirPDF): Variable booleana que permite marcar si el anexo se debe convertir a pdf.
  • setFirmar(boolean firmar): Variable booleana que permite marcar si se debe firmar o no el anexo.
  • setMaxInstancias(int maxInstancias): Variable entera que permite establecer el número máximo de instancias por anexo.
