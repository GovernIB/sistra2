esEntero(String numero): boolean

DESCRIPCIÓN

  • Verifica que el número introducido no sea nulo, ni vacío y que sea un número entero.

PARÁMETROS DE ENTRADA

  • numero: Número entero a validar.

PARÁMETROS DE SALIDA

  • boolean : True si el número es un número entero y falso si no es un número entero.

EJEMPLOS

Ejemplo 1 Validar una cadena de caracteres vacía.

var valido = PLUGIN_UTILS.esEntero('');

La función esEntero retorna 'false', debido a que el valor introducido no es un número.


Ejemplo 2 Validar una cadena de caracteres nula.

var entero;
var valido = PLUGIN_UTILS.esEntero(entero);

La función esEntero retorna 'false', debido a que el valor introducido es nulo


Ejemplo 3 Validar la cadena de caracteres '3A'.

var valido = PLUGIN_UTILS.esEntero('3A');

La función esEntero retorna 'false', debido a que el valor introducido no es un número entero.


Ejemplo 4 Validar la cadena de caracteres '12,45'.

var valido = PLUGIN_UTILS.esEntero('12,45');

La función esEntero retorna 'false', debido a que el valor introducido no es un número entero.


Ejemplo 5 Validar la cadena de caracteres '12'.

var valido = PLUGIN_UTILS.esEntero('12');

La función esEntero retorna 'true', debido a que el valor introducido es un número entero.


Ejemplo 6 Validar la cadena de caracteres '-23'.

var valido = PLUGIN_UTILS.esEntero('-23');

La función esEntero retorna 'true', debido a que el valor introducido es un número entero.
