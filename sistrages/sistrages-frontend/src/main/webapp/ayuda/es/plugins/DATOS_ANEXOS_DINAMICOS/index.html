DATOS_ANEXOS_DINAMICOS

DESCRIPCIÓN

  • Objeto que permite establecer los anexos de forma dinámica, indicando la siguiente información por anexo: identificador, descripción, extensiones permitidas, tamaño máximo, url plantilla, obligatoriedad, convertir a PDF y firmar.
  • Este objeto está disponible para el script de Lista dinámica de anexos del paso anexar documentos.

MÉTODOS

  • addAnexo(String identificador, String descripcion, String extensiones, String tamanyoMax, String urlPlantilla, boolean obligatorio, boolean convertirPDF, boolean firmar)
