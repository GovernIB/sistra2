setValorCompuesto(campo, codigo, descripcion): void

DESCRIPCIÓN

  • Método que establece el valor de un campo compuesto del formulario actual.

PARÁMETROS DE ENTRADA

  • campo: Identificador del campo al que se desea asignar el valor.
  • codigo: Código que se desea asignar al campo.
  • descripcion: Descripción que se desea asignar al campo.

EJEMPLOS

Asignar al campo 'provincia' el valor compuesto por el código '07' y la descripción 'Illes Balears':

DATOS_VALOR.setValorCompuesto('provincia', '07', 'Illes Balears');
