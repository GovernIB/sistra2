setEmail(email): void

DESCRIPCIÓN

  • Método que permite indicar un correo de email.

PARÁMETROS DE ENTRADA

  • email: parámetro de tipo texto que incluye el email.

EJEMPLOS

Por ejemplo, supongamos deseamos establecer un email destino cuando se finaliza de rellenar un formulario:

DATOS_AVISO.setEmail('responsable@caib.es');
