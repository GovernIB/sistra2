rtrim(String cadena): String

DESCRIPCIÓN

  • Método que elimina los espacios situados a la derecha en la cadena.

PARÁMETROS DE ENTRADA

  • cadena: Cadena de texto a tratar

PARÁMETROS DE SALIDA

  • String: cadena de texto sin espacios por la derecha.

EJEMPLOS

Tratar la cadena '  cadena ejemplo    ', para eliminar los espacios de la derecha.

var pCadenaLimpia = PLUGIN_UTILS.rtrim('  cadena ejemplo    ');

La función rtrim retorna '  cadena ejemplo'.
