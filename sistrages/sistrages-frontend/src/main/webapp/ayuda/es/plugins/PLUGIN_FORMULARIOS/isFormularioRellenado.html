  • isFormularioRellenado

isFormularioRellenado(String idFormulario): boolean

DESCRIPCIÓN

  • Método que indica si el formulario esta cumplimentado.

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.

PARÁMETROS DE SALIDA

  • boolean: 'true' si el formulario está cumplimentado, y 'false' en caso contrario.

EJEMPLO

Validar si el formulario con identificador 'Form1' está cumplimentado (en este caso sí que lo está)

var relleno = PLUGIN_FORMULARIOS.isFormularioRellenado('Form1');

La función isFormularioRellenado retornará 'true'.
