setValorMultiple(valor): void

DESCRIPCIÓN

  • Método que establece un valor múltiple a un campo del formulario actual.

PARÁMETROS DE ENTRADA

  • valor: Objeto de tipo ValorCampoMultiple el cual contiene los diferentes valores a asignar al campo.

EJEMPLOS

Para añadir múltiples valores a un campo, por ejemplo una lista desplegable, primero debemos crear una variable mediante crearValorMultiple. Esto permitirá añadirle valores mediante addValor:

var municipios = DATOS_VALOR.crearValorMultiple(); municipios.addValor('0337','Manacor'); municipios.addValor('0407','Palma de Mallorca'); municipios.addValor('2060','Eivissa');

Una vez creada la variable municipios, se puede asignar al campo actual mediante la siguiente sentencia

DATOS_VALOR.setValorMultiple(municipios);
