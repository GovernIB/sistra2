setValorMultiple(idFormulario, campo, valorMultiple): void

DESCRIPCIÓN

  • Método que establece un valor a la lista de valores de un campo de selección múltiple. Modificar un formulario implica que se marcará con estado incorrecto para forzar su revisión por parte del usuario. Solo se podrá modificar el formulario actual o posteriores.

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario en el que quiere modificar el valor del campo.
  • campo: Campo al que se desea asignar el valor múltiple.
  • valorMultiple: Valor múltiple a establecer en el campo.

EJEMPLOS

Por ejemplo, supongamos un campo que es una lista desplegable que contiene información sobre un municipio. Podemos inicializar una variable múltiple con diferentes datos fijos sobre municipios de les Illes Balears de la manera siguiente:

var municipios = DATOS_FORMULARIOS.crearValorMultiple(); municipios.addValor('0337','Manacor'); municipios.addValor('0407','Palma de Mallorca'); municipios.addValor('2060','Eivissa');

una vez creada la variable, podríamos asignar los valores al campo del formulario (FORM1):

DATOS_FORMULARIOS.setValorMultiple('FORM1', 'municipios', municipios);
