addParametro(String codigo, String valor): void

DESCRIPCIÓN

  • Método que permite pasar un nuevo parámetro a un formulario.

PARÁMETROS DE ENTRADA

  • codigo: String con el código del parámetro a añadir.
  • valor: String con el valor del parámetro a añadir.

EJEMPLOS

Supongamos por caso que queremos pasar un parámetro provincia (prov) a un formulario, indicando como valor el código 07 perteneciente a las Baleares. Para ello incluiríamos la siguiente línea de código:

var params = DATOS_PARAMETROSFORMULARIO.addParametro('prov','07');
