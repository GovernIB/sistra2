getApellido2(): String

DESCRIPCIÓN

  • Método que devuelve el segundo apellido del usuario autenticado.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el segundo apellido del usuario autenticado.

EJEMPLO

Si se desea obtener el segundo apellido del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener su segundo apellido mediante la función getApellido2:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellido = usuario.getApellido2();
}

La funció getApellido2 retornarà la cadena 'Rodriguez'.
