DATOS_INSTRUCCIONES

DESCRIPCIÓN

  • Plugin que permite establecer la información del paso 'Debe saber' de forma dinámica.
  • Este script sobrescribirá la información introducida en "Instrucciones de inicio (HTML)"
  • Dentro del script, hay dos formas de introducir las instrucciones del paso 'Debe Saber', a través de los mensajes de validación, con el método DATOS_INSTRUCCIONES.setCodigoMensaje(texto), o introduciedno directamente las instrucciones con el método DATOS_INSTRUCCIONES.setTextoMensaje(texto). Tendrá preferencia, en caso de que se utilicen los dos en un mismo script, el método de DATOS_INSTRUCCIONES.setCodigoMensaje(texto).

MÉTODOS

  • addParametroMensaje(parametro)
  • setCodigoMensaje(codigo)
  • setTextoMensaje(texto)
