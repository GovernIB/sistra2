getContenidoB64(): String

DESCRIPCIÓN

  • Método que permite recuperar contenido de un fichero en Base64.

PARÁMETROS DE SALIDA

  • String: Contenido del fichero en Base64.

EJEMPLO

En el caso de que necesitemos recuperar el contenido de un fichero y asignarlo a una variable, podemos usar esta función, obtendremos una cadena de caracteres en Base64 que contiene el fichero en cuestión.

var contenido = PLUGIN_VALIDACIONANEXO.getContenidoB64();
