getXml(String idFormulario): String

DESCRIPCIÓN

  • Método que devuelve el XML con los valores del formulario.

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que representa el XML con los valores del formulario.

EJEMPLO

Obtener el XML del formulario con identificador 'Form1'

var xml = PLUGIN_FORMULARIOS.getXml('Form1');

La función getXml retornará una cadena de texto cuyo contendido será el XML del formulario con sus valores.
De este XML se podría extraer el valor de un determinado campo, por ejemplo si sabemos que en el XML existe un campo dni con el siguiente xpath '/xml/persona/dni' podríamos obtener el valor del campo DNI mediante:

var dni = PLUGIN_VALIDACIONES.extraerValorXml(xml, '/xml/persona/dni');

La función extraerValorXml retornará el valor del campo dni dentro del XML, en este caso '11111111H'.
