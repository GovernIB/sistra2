esNumeroCuentaIbanValido(String cuenta): boolean

DESCRIPCIÓN

  • Valida que el valor introducido no sea nulo, y posea el formato correspondiente a un número de cuenta iban, cumpliendo con las reglas de validación necesarias para ser un IBAN (International Bank Account Number) Válido. Se considera válida toda aquella cadena que tenga 24 números que cumplan con las reglas de validación correspondientes a clave del pais, código de control e identificador NCC (entidad, oficina, dígito de control y número de cuenta).

PARÁMETROS DE ENTRADA

  • cuenta: Código de 24 dígitos del Número de cuenta.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena introducida es un número de cuenta válido y false si la cadena introducida no es un número de cuenta válido

EJEMPLOS

Ejemplo 1 Validar la cadena 'ES6621000418461234567891'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaIbanValido('ES6621000418461234567891');

La función esNumeroCuentaIbanValido retorna 'false' debido a que la cadena no cumple con las reglas de validación, el dígito de control no es válido (el CC es 40, no 46).


Ejemplo 2 Validar la cadena 'ES6621000418401234567891'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaIbanValido ('ES6621000418401234567891');

La función esNumeroCuentaIbanValido retorna 'true', debido a que la cadena cumple los requisitos para ser un número de cuenta IBAN válido.
