getNombre(): String

DESCRIPCIÓN

  • Método que devuelve el nombre del usuario autenticado.

PARÁMETROS DE SALIDA

  • String: cadena de caracteres con el nombre del usuario autenticado.

EJEMPLO

Si se desea obtener el nombre del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación, obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener su nombre mediante la función getNombre:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nombre = usuario.getNombre();
}

La función getNombre retornará la cadena 'Jose'.
