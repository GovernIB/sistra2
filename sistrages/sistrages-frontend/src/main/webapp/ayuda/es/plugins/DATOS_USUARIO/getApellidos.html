getApellidos(): String

DESCRIPCIÓN

  • Método que devuelve los apellidos del usuario autenticado.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con los apellidos del usuario autenticado (primer y segundo apellido, concatenados con un espacio).

EJEMPLO

Si se desean obtener los apellidos del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener los apellidos mediante la función getApellidos:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellidos = usuario.getApellidos();
}

La función getApellidos retornará la cadena 'Pérez Rodriguez'.
