setOficina(String oficina): void

DESCRIPCIÓN

  • Método que permite establecer la oficina de registro sobre la que se hará el apunte registral de entrada de las solicitudes asociadas a la versión del trámite.

PARÁMETROS DE ENTRADA

  • oficina: Identificador de la oficina de registro.

EJEMPLO

Pongamos por caso que la oficina sobre la que queremos registrar la solicitud depende de un valor del formulario. En tal caso utilizariamos el script para fijar el valor de dicha propiedad. Para ello incorporaríamos la siguiente línea de código:

DATOS_REGISTRO.setOficina('idOficina');
