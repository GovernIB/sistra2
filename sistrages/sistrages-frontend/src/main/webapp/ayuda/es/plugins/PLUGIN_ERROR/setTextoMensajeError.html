setTextoMensajeError(String texto):void

DESCRIPCIÓN

  • Método que permite fijar directamente por script el texto del mensaje de error.

PARÁMETROS DE ENTRADA

  • texto: String con el texto descriptivo que identifica el motivo del error.

EJEMPLOS

Pongamos por caso que queremos establecer un mensaje de error fijo y en un único idioma. Lo podríamos hacer a través del script de la manera siguiente:

PLUGIN_ERROR.setTextoMensajeError('Debe corregir las fechas ingresadas ya que no corresponden con valores correctos');
