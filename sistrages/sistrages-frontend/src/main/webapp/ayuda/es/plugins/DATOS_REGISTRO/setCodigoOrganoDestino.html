setCodigoOrganoDestino(String codigoOrganoDestino): void

DESCRIPCIÓN

  • Método que permite establecer el código del organismo destino donde se hace el apunte registral de entrada de las solicitudes asociadas a la versión del trámite.

PARÁMETROS DE ENTRADA

  • codigoOrganoDestino: Código de órgano destino. Dependerá de la solución de registro, pero típicamente será un código DIR3.

EJEMPLO

Pongamos por caso que queremos hacer el apunte registral en un órgano destino establecido en un campo (p.ej ORGANO_DESTINO) de un formulario (p. ej. FORM-1). Para ello estableceremos las siguientes líneas en el script:

var codigoOrganoDestino = PLUGIN_FORMULARIOS.getValor('FORM-1', 'ORGANO_DESTINO'); DATOS_REGISTRO.setCodigoOrganoDestino(codigoOrganoDestino);
