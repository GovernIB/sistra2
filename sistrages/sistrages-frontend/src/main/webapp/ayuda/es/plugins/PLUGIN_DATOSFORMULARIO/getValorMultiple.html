getValorMultiple(String campo, String indice): VALOR_CAMPOCOMPUESTO

DESCRIPCIÓN

  • Método que devuelve el valor de un campo múltiple.

PARÁMETROS DE ENTRADA

  • campo: Cadena que contiene el identificador del campo.
  • indice: Valor del índice dentro del campo múltiple. Tener en cuenta que los índices comienzan por 0.

PARÁMETROS DE SALIDA

  • VALOR_CAMPOCOMPUESTO: Devuelve el objeto VALOR_CAMPOCOMPUESTO ubicado en el índice del campo múltiple. Este objeto está formado por 2 campos, código y descripción, los valores de los cuales se pueden obtener mediante las funciones getCodigo() y getDescripcion() respectivamente.

EJEMPLO

Obtener el último valor del campo múltiple 'campoTitulaciones', para ello primero se debe obtener el total de valores:

var total = PLUGIN_DATOSFORMULARIO.getNumeroValoresValorMultiple('campoTitulaciones');

La función getNumeroValoresValorMultiple retornará 8 (total de valores de ese campo). Con este valor podemos calcular el índice del último elemento de la siguiente manera:

if (total > 0) { var valorCompuesto = PLUGIN_DATOSFORMULARIO.getValorMultiple('campoTitulaciones', total -1); }

La función getValorMultiple retornará el objeto valorCompuesto ubicado en la posición indicada. Para obtener los datos del valor compuesto podemos pedir su código o descripción tal como se muestra a continuación:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En este caso la función getCodigo retornará la cadena 'GRAD.SUP.INFOR' y la función getDescripcion retornará la cadena 'Grado superior en informática'.
