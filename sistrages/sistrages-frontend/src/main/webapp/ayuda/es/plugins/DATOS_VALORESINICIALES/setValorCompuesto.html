setValorCompuesto(campo, codigo, descripcion): void

DESCRIPCIÓN

  • Método que permite establecer el valor de un campo compuesto del formulario actual.

PARÁMETROS DE ENTRADA

  • campo: Campo al que se desea asignar el valor.
  • codigo: Código que se desea asignar al campo.
  • descripcion: Descripción que se desea asignar al campo.

EJEMPLOS

Un ejemplo de uso podría ser la asignación de un valor compuesto al campo "provincia" del formulario actual:

DATOS_VALORESINICIALES.setValorCompuesto('provincia', '07', 'Illes Balears');
