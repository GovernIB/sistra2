lpad(String texto, int longitudMinima, String cadenaRelleno): String

DESCRIPCIÓN

  • Método que, si es necesario, rellena una cadena, añadiendo a la izquierda los caracteres especificados en el parámetro cadenaRelleno, hasta alcanzar la longitud determinada por el parámetro longitudMinima.
  • Si la cadena de relleno tiene más de 1 carácter, se van usando los caracteres de la cadena secuencialmente para rellenar los espacios necesarios. Es decir, si tenemos la cadena de relleno '123' se irán usando secuencialmente los caracteres 1 2 y 3 para rellenar los espacios necesarios hasta completar la longitud mínima.

PARÁMETROS DE ENTRADA

  • texto: Cadena original que se desea rellenar.
  • longitudMinima: Longitud mínima que debe tener el texto.
  • cadenaRelleno: Cadena que se usará para rellenar el texto.

PARÁMETROS DE SALIDA

  • String: Texto con la longitud mínima establecida. Si el texto original no alcanza la longitud mínima, habrá sido completado, añadiendo por la izquierda la cadena cadenaRelleno, hasta alcanzar longitudMinima. Si la longitud de la cadena texto original es mayor que longitudMinima, retorna la cadena de texto original.

EJEMPLOS

Ejemplo 1 Supongamos que deseamos forzar que un determinado valor, mínimo tenga 9 dígitos. Para este caso podemos usar esta función para rellenar con 0 a la izquierda ese valor y obtener como mínimo 9 dígitos. por ejemplo, si tenemos la cadena '963467' y añadimos la siguiente sentencia:

var pRelleno = PLUGIN_UTILS.lpad('963467', 9, '0');

La función lpad retornará '000963467'.


Ejemplo 2 Supongamos que se desea que la cadena '12345' pase a tener 10 caracteres, y se rellene por la izquierda con la cadena 'aA':

var pRelleno = PLUGIN_UTILS.lpad('12345', 10, 'aA');

La función lpad retornará 'aAaAa12345'.
