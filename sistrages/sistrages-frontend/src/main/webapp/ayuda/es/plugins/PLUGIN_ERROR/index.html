PLUGIN_ERROR

DESCRIPCIÓN

  • Plugin que permite establecer errores en la ejecución de un script.
  • Se forzará la salida del trámite indicando un mensaje de error genérico. El mensaje del error establecido a través del plugin se visualizará en la ventana de debug (desde el navegador con la sesión abierta de la tramitación fallida, se deberá abrir otra pestaña en el navegador y acceder a la dirección siguiente: /sistramitfront/debug/logSesionTramitacion.html)

MÉTODOS

  • addParametroMensajeError(parametro)
  • setCodigoMensajeError(codigo)
  • setExisteError(existe)
  • setTextoMensajeError(texto)
