Plugin Error

DESCRIPCIÓN

  • Plugin para marcar el script con error
  • Este error generará una excepción de script excepto en los scripts que sean de validación, en los que el mensaje establecido en el plugin será el que se muestre al usuario.

MÉTODOS

  • addParametroMensajeError(parametro)

  • setTextoMensajeError(pTextoMensajeError)
