DATOS_FORMULARIOS

DESCRIPCIÓN

  • Objeto que permite modificar los datos de un formulario antes de guardarlo.
  • Según el tipo de campo tendrá un tipo de valor:
    • Valor simple: Es un texto. Tendrán este valor los campos de tipo texto y checkbox.
    • Valor compuesto: Es un código y un texto. Tendrán este valor los selectores de tipo desplegable y de selección única.
    • Valor múltiple: Es una lista de valores compuestos. Tendrán este valor los selectores de tipo selección múltiple.
