addParametro(String codigo, String valor): void

DESCRIPCIÓN

  • Método que permite añadir un nuevo parámetro a la versión de un trámite.

PARÁMETROS DE ENTRADA

  • codigo: String con el código del parámetro a añadir a la versión del trámite.
  • valor: String con el valor del parámetro a añadir a la versión del trámite.

EJEMPLOS

Supongamos que queremos añadir un parámetro, por ejemplo provincia, a la versión de un trámite. A dicho parámetro le queremos asignar el código 07 referido a la provincia Islas Baleares. Para ello haríamos lo siguiente:

var params = DATOS_PARAMETROS_INICIALES.addParametro('prov','07');
