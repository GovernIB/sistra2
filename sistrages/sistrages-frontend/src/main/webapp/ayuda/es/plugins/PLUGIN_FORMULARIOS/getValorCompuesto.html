getValorCompuesto(String idFormulario, String campo): VALOR_CAMPOCOMPUESTO

DESCRIPCIÓN

  • Método que devuelve el valor de un campo compuesto.

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.
  • campo: Cadena que contiene el identificador del campo.

PARÁMETROS DE SALIDA

  • VALOR_CAMPOCOMPUESTO: retorna el objeto VALOR_CAMPOCOMPUESTO. Este objeto está formado por 2 campos, código y descripción,los valores de los cuales se pueden obtener mediante las funciones getCodigo() y getDescripcion() respectivamente.

EJEMPLO

Obtener el valor del campo compuesto 'campoProvincia' del formulario con id 'Form1':

var valorCompuesto = PLUGIN_FORMULARIOS.getValorCompuesto('Form1','campoProvincia');

La función getValorCompuesto retornará el objeto valorCompuesto. Para obtener los datos del valor compuesto podemos pedir su código o descripción tal como se muestra a continuación:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En este caso la función getCodigo retornará la cadena '07' y la función getDescripcion retornará la cadena 'Illes Balears'.
