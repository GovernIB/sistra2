esCif(String cif): boolean

DESCRIPCIÓN

  • Valida si el cif introducido es un cif legalmente válido. Se considerará CIF válido toda aquella cadena que cumpla con el formato de número de carácteres correcto y cumpla con las reglas respectivas a la correspondencia alfanumérica.

PARÁMETROS DE ENTRADA

  • cif: Valor del Cif.

PARÁMETROS DE SALIDA

  • boolean: true si el cif es válido y false si el cif no es válido.

EJEMPLOS

Ejemplo 1 Se verifica una cadena vacía ''.

var valido = PLUGIN_UTILS.esCif('');

La función esCif retorna 'false', ya que el valor pasado como cif no se corresponde con el formato de un cif.


Ejemplo 2 Se verifica una cadena con valor nulo.

var cif; var valido = PLUGIN_UTILS.esCif(cif);

La función esCif retorna 'false', ya que el valor pasado como cif, es nulo y por lo tanto no se corresponde con el formato de un cif.


Ejemplo 3 Se verifica el valor 'C23568945'.

var valido = PLUGIN_UTILS.esCif('C23568945');

La función esCif retorna 'false', ya que el valor pasado como cif no se corresponde con el formato correcto para el mismo (no tiene la letra de control).


Ejemplo 4 Se verifica el valor 'C6141464E'.

var valido = PLUGIN_UTILS.esCif('C6141464E');

La función esCif retorna 'true', ya que el valor pasado como cif se corresponde con el formato correcto para el mismo.
