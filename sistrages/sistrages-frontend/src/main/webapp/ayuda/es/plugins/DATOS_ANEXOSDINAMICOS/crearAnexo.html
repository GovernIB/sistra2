crearAnexo(): AnexoDinamico

DESCRIPCIÓN

  • Método que crea una variable de tipo Anexo Dinámico para establecer sus valores.

PARÁMETROS DE SALIDA

  • anexo: variable de tipo Anexo Dinámico para establecer sus valores.

EJEMPLOS

Si deseamos añadir un nuevo anexo de manera dinámica a la versión de un trámite, ejecutaríamos lo siguiente:

var anex = DATOS_ANEXOSDINAMICOS.crearAnexo();
anex.setIdentificador('DIN1');
anex.setDescripcion('Anexo dinamico');
anex.setExtensiones('pdf');
anex.setTamanyoMaximo('1MB');
anex.setObligatorio(true);
DATOS_ANEXOSDINAMICOS.addAnexo(anex);

El resultado de la ejecución del script "Lista dinámica de anexos" con esta sentencia será un anexo que es obligatorio, que debe subirse en formato pdf, con un tamaño máximo de 1MB.
