DATOS_INICIALES_FORMULARIO

DESCRIPCIÓN

  • Objeto que permite establecer los datos iniciales en el formulario actual.
  • Este objeto está disponible para el script "Carga de datos iniciales" desde un formulario.

MÉTODOS

  • setValorCompuesto(campo, codigo, descripcion)
  • setValorMultiple(campo, valores)
