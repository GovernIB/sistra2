addParametro(String codigo, String parametro): void

DESCRIPCIÓN

  • Método que permite añadir parámetros requeridos para la invocación de un dominio.

PARÁMETROS DE ENTRADA

  • codigo: String con el código del parámetro a añadir.
  • parametro: String con el valor del parámetro a añadir.

EJEMPLOS

Pongamos por caso que queremos invocar un dominio con identificador 'MUNICIPIOS' que requiere pasar por parámetro la provincia, cuyo código de párametro es 'prov' y al que daremos como valor '07', que es el código INE de Islas Baleares. Para incorporar dicho parámetros procederíamos de la manera siguiente:

var params = PARAMETROS_DOMINIO.addParametro('prov','07');
