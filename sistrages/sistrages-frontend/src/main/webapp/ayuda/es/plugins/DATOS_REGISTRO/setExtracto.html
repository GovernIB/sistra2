setExtracto(String extracto): void

DESCRIPCIÓN

  • Método que permite establecer un texto resumen o extracto del asiento registral. Si no se fija valor se tomará por defecto el título del trámite para el que se realiza la solicitud.

PARÁMETROS DE ENTRADA

  • extracto: cadena de texto resumen del asiento registral.

EJEMPLOS

Pongamos por caso que queremos fijar un extracto del asiento basado en la cumplimentación realizada por el ciudadano. Para ello podríamos referenciar un campo (p.ej. ASUNTO) del formulario (p.ej. FORM1) en el que el ciudadano ha cumplimentado dicho texto:

var textoExtracto = PLUGIN_FORMULARIOS.getValor('FORM1','ASUNTO'); DATOS_REGISTRO.setExtracto(textoExtracto);
