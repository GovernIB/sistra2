getNumeroValoresValorElementoMultiple(idFormulario,campoListaElementos,campo): int

DESCRIPCIÓN

  • Método que retorna el número de valores de un campo con valor múltiple asociado a una lista de elementos a partir del identificador de la lista de elementos y del identificador del campo.

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario.
  • campoListaElementos: Identificador del campo lista de elementos.
  • campo: Identificador del campo dentro del formulario de la lista de elementos.

PARÁMETROS DE SALIDA

  • int: Recupera la cantidad de valores del campo.

Ejemplo

Si se desea obtener el número de elementos que tiene internamente un campo asociado a una lista de elementos en un formulario concreto, utilizaremos la función de la siguiente forma (suponiendo que el identificador de la lista de elementos es 'LISTA_ELEM1' y que dentro tiene un componente selector con el identificador 'SELECT1'):

var numeroElementos = PLUGIN_FORMULARIOS.getNumeroValoresValorElementoMultiple('FORM1','LISTA_ELEM1', 'SELECT1');

La función getNumeroValoresValorElementoMultiple() devolverá el total de elementos que tiene dicho campo dentro de la lista de elementos. En caso de no tener ninguno, devolverá 'null'.
