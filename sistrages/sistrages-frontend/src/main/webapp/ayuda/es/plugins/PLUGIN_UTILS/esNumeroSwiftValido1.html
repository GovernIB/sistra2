esNumeroSwiftValido(String swift): boolean

DESCRIPCIÓN

  • Valida que el valor introducido no sea nulo y cumpla con las reglas de validación necesarias para ser un SWIFT (código de identificación bancaria para realizar transferencias) Válido. Se considera válida toda aquella cadena que tenga 8-11 caracteres números que cumplan con las reglas de validación correspondientes a código del banco, clave del país, localidad y sucursal (la sucursal es opcional, en caso de no introducirse, será sede central). El código del banco y del país debe ser sólo mayúsculas, el código de lugar debe ser sólo mayúsculas o sólo números mientras que el código de sucursal puede ser mayúsculas y números.

PARÁMETROS DE ENTRADA

  • swift: Código de 8-11 caracteres del código Swift.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena introducida es un número de Swift válido y false si la cadena introducida no es un número swift válido

EJEMPLOS

Ejemplo 1 Validar la cadena 'BBVAXESMM'.

var valido = PLUGIN_UTILS.esNumeroSwiftValido('BBVAXESMM');

La función esNumeroSwiftValido retorna 'false' debido a que la cadena no cumple con las reglas de validación, son 9 caracteres y no 8 o 11.


Ejemplo 2 Validar la cadena 'BBVAESMM'.

var valido = PLUGIN_UTILS.esNumeroSwiftValido ('BBVAESMM');

La función esNumeroSwiftValido retorna 'true', debido a que la cadena cumple los requisitos para ser un número SWIFT válido.
