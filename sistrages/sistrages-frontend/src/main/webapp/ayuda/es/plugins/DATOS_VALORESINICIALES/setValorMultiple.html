setValorMultiple(campo, valor): void

DESCRIPCIÓN

  • Método que permite establecer un valor múltiple a un campo del formulario actual.

PARÁMETROS DE ENTRADA

  • campo: Campo al que se desea asignar el valor.
  • valor: Valor que se asigna al campo.

EJEMPLOS

Por ejemplo, inicializamos una variable múltiple con diferentes datos fijos sobre provincia, municipio y código postal:

var datos = DATOS_VALORESINICIALES.crearValorMultiple(); datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');

Una vez creada la variable se podrá asignar al campo correspondiente

DATOS_VALORESINICIALES.setValorMultiple('datosGeografico', datos);
