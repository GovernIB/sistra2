getDescripcion():String

DESCRIPCIÓN

  • Método que permite recuperar la descripcion de un fichero.

PARÁMETROS DE SALIDA

  • String: String con la descripción del fichero.

EJEMPLOS

Pongamos por caso que se quiere recuperar la descripción de un fichero retornado por un dominio 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se recupera un fichero mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fichero = valores.getFichero('PDF');

y se recupera su descripción de la siguiente forma:

var descripcion = fichero.getDescripcion();
