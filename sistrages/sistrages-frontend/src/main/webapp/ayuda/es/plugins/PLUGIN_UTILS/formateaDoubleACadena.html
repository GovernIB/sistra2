formateaDoubleACadena(double numero, String separadorMiles, String separadorDecimales, int numeroDecimales): String

DESCRIPCIÓN

  • Método para convertir un número de tipo double en un string

PARÁMETROS DE ENTRADA

  • numero: Número double a convertir en String .
  • separadorMiles: Carácter con el que se identifica el separador de miles.
  • separadorDecimales: Carácter con el que se identifica el separador de decimales. Dato obligatorio.
  • separadorDecimales: Número de decimales.

PARÁMETROS DE SALIDA

  • String: Cadena de carácteres que representa el numero introducido.

EJEMPLOS

Ejemplo 1 Convertir el número 123110544434.58 en una cadena de carácteres

var pCadena = PLUGIN_UTILS.formateaDoubleACadena(123110544434.58, '.', ',', 2);

La función formateaDoubleACadena retorna la cadena '123.110.544.434,58'.


Ejemplo 2 Convertir el número 123110544434.58 en una cadena de carácteres sin incluir los decimales

var pCadena = PLUGIN_UTILS.formateaDoubleACadena(123110544434.58, '.', ',', 0);

La función formateaDoubleACadena retorna la cadena '123.110.544.435', redondeando el número.
