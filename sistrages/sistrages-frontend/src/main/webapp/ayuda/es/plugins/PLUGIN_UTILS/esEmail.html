esEmail(String email): boolean

DESCRIPCIÓN

  • Valida si la dirección de e-mail introducida es una dirección válida. Se considerará dirección de e-mail válida toda aquella cadena que cumpla con el formato y las reglas que rigen el formato de un correo electrónico.

PARÁMETROS DE ENTRADA

  • email: Dirección de correo electrónico a validar.

PARÁMETROS DE SALIDA

  • boolean: true si el email está correctamente formado y false si el email no está correctamente formado.

EJEMPLOS

Ejemplo 1 Validar la cadena de caracteres ''.

var valido = PLUGIN_UTILS.esEmail('');

La función esEmail retorna 'false', debido a que el email introducido no tiene un formato válido.


Ejemplo 2 Validar una cadena de caracteres nula.

var email;
var valido = PLUGIN_UTILS.esEmail(email);

La función esEmail retorna 'false', debido a que el email introducido no tiene un formato válido.


Ejemplo 3 Validar la cadena de caracteres 'pepe@gmail'

var valido = PLUGIN_UTILS.esEmail('pepe@gmail');

La función esEmail retorna 'false', debido a que el email introducido no tiene un formato válido.


Ejemplo 4 Validar la cadena de caracteres 'pepe@gmail.com'.

var valido = PLUGIN_UTILS.esEmail('pepe@gmail.com');

La función esEmail retorna 'true', debido a que el email introducido tiene un formato válido.
