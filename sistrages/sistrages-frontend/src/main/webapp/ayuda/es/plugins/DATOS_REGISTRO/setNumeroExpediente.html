setNumeroExpediente(String numeroExpediente): void

DESCRIPCIÓN

  • Método que permite establecer un número de expediente. Este valor aplica a trámite de continuación, no a los de inicio.

PARÁMETROS DE ENTRADA

  • numeroExpediente: Identificador del numero de expediente.

EJEMPLOS

Pongamos por caso que queremos subsanar un expediente iniciado. En este caso podríamos vincular el registro de entrada a un número de expediente utilizando lo siguiente:

DATOS_REGISTRO.setNumeroExpediente('idNumeroExpediente');
