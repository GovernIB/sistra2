Plugin Avisos

DESCRIPCIÓN

  • Plugin para generar un mensaje de aviso.
  • Este plugin estara disponible en los scripts que implementen validación y servirá para mostrar un mensaje de aviso* (info/warning) que no para la ejecución.

MÉTODOS

  • setExisteAviso(pExisteAviso,pTipo)
  • addParametroMensajeAviso(parametro)
  • setCodigoMensajeAviso(pCodigoMensajeAviso)
  • setTextoMensajeAviso(pTextoMensajeAviso)
