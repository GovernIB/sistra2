getNombreColumnas(): List<String>

DESCRIPCIÓN

  • Método que permite obtener una lista con los nombres de las columnas que retorna un dominio.

PARÁMETROS DE SALIDA

  • List<String>: Retorna una lista de objetos String que representan los nombres de las columnas que retorna el dominio.

EJEMPLOS

Pongamos por caso que se quiere recuperar el nombre de las columnas que retorna el domini 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

y se recupera la lista de nombres de las columnas de la manera siguiente:

var listColumns = valores.getNombreColumnas();

Una vez obtenida la lista, se puede recuperar el nombre de cada columna como se indica a continuación:

for (i=0; i<listColumns.size(); i++) {
var nomcol = listColumns.get(i);
