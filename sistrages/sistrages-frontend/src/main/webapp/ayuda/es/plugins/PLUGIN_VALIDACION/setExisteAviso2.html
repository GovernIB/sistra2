setExisteAviso(boolean existeAviso, String tipo, String campo)

DESCRIPCIÓN

  • Método para establecer si se genera aviso para usarse en el script de validación de página y poder indicar el campo de formulario al que aplica la validación.

PARÁMETROS DE ENTRADA

  • existeAviso: establecer si se genera aviso.
  • tipo: INFO / WARNING / ERROR
  • campo: Identificador campo formulario

EJEMPLOS

Vamos a proceder a indicar que existe un aviso de tipo WARNING sobre el campo txtValor:

PLUGIN_VALIDACION.setExisteAviso(true, 'WARNING', 'txtValor');
