getNumeroFilas(): int

DESCRIPCIÓN

  • Método que permite obtener el número de filas retornadas como resultado de la invocación de un dominio.
  • Este método está disponible desde el método PLUGIN_DOMINIO.invocarDominio (accesible desde todos los scripts de versión, paso y formulario).

PARÁMETROS DE SALIDA

  • int: Entero que representa el número de filas retornadas por el dominio.

EJEMPLOS

Pongamos por caso que se quiere recuperar el número de filas que retorna el dominio 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

y se recupera el número de filas de la manera siguiente:

var numFilas = valores.getNumeroFilas();
