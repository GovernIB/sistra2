esNulo(Object objeto): boolean

DESCRIPCIÓN

  • Valida si el parámetro objeto es nulo.

PARÁMETROS DE ENTRADA

  • objeto: Objeto a validar.

PARÁMETROS DE SALIDA

  • boolean : true si el objeto es nulo y false si el objeto no es nulo.

EJEMPLOS

Ejemplo 1 Validar una variable inicializada a null.

Object object = null;
var vacio = PLUGIN_UTILS.esNulo(object);

La función esNulo retorna 'true', debido a que el contenido de la variable es null.


Ejemplo 2 Validar una cadena de caracteres vacía.

var vacio = PLUGIN_UTILS.esNulo('');

La función esNulo retorna 'false', debido a que el valor objeto no es null.
