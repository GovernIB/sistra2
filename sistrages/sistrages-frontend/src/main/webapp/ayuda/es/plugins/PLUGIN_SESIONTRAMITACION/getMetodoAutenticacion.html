getMetodoAutenticacion(): String

DESCRIPCIÓN

  • Método que devuelve el método de autenticación de la actual sesión.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que contiene el método de autenticación. Los valores posibles son:
    • 'ANONIMO'
    • 'CLAVE_CERTIFICADO'
    • 'CLAVE_PIN'
    • 'CLAVE_PERMANENTE'

EJEMPLO

Obtener el método de autenticación de la actual sesión (ANONIMO):

var metodo = PLUGIN_SESIONTRAMITACION.getMetodoAutenticacion();

La función getMetodoAutenticacion retorna la cadena de caracteres 'ANONIMO'.
