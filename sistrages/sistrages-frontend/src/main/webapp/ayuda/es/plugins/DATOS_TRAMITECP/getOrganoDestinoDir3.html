getOrganoDestinoDir3(): String

ÓRGANO DE DESTINO DIR3

  • Método que retorna el órgano de destino DIR3 asociado al trámite en el catálogo de procedimentos

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres (de longitud máxima 150), con el órgano de destino DIR3 asociado al trámite en el catálogo de procedimentos.

Ejemplo

Si se desea obtener el órgano de destino DIR3 asociado al trámite en el catálogo de procedimentos, utilizaremos la función de la siguiente forma:

var tramite = PLUGIN_SESIONFORMULARIO.getTramiteCP();
var organodestinodir3_tramite = tramite.getOrganoDestinoDir3();

La función getOrganoDestinoDir3() devolverá el órgano de destino DIR3 asociado al trámite en el catálogo de procedimentos. En caso de no tener ninguna, devolverá 'null'.
