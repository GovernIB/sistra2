getAccionPersonalizada(String idFormulario): String

DESCRIPCIÓN

  • Método que devuelve el valor de una acción personalizada para un formulario (en caso de tenerla).

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres que representa la acción personalizada. En caso de no tenerla, devuelve cadena vacía.

EJEMPLO

Obtener la acción personalizada del formulario con identificador 'Form1'

var accion = PLUGIN_FORMULARIOS.getAccionPersonalizada('Form1');

La función getAccionPersonalizada retornará la acción personalizada del formulario 'Form1'.
