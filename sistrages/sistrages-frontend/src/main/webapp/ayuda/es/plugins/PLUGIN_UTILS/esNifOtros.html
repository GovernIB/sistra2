esNifOtros(String nif): boolean

DESCRIPCIÓN

  • Valida si el valor introducido posee el formato requerido para ser un NIF para españoles menores de 14 años o extranjeros sin NIE que de forma transitoria necesitan una identificación válida. Se considera válido toda aquella cadena que tenga 7 números, un carácter alfanumérico al principio para marcar que es un tipo NIF (letra K/L/M) y un carácter alfanumérico que cumpla con la regla de validación.

PARÁMETROS DE ENTRADA

  • nif: cadena que representa el NIF para españoles menores de 14 años o extranjeros sin NIE que de forma transitoria necesitan una identificación a validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena es un NIF para españoles menores de 14 años o extranjeros sin NIE que de forma transitoria necesitan una identificación válida y false si la cadena no se corresponde con dicho formato.

EJEMPLOS

Ejemplo 1 Validar la cadena 'K2481154Y'.

var valido = PLUGIN_UTILS.esNifOtros('K2481154Y');

La función esNifOtros retorna 'true', debido a que la cadena contiene un NIF para españoles menores de 14 años o extranjeros sin NIE que de forma transitoria necesitan una identificación válida.


Ejemplo 2 Validar la cadena 'J2411154Y'.

var valido = PLUGIN_UTILS.esNifOtros('J2411154Y');

La función esNifOtros retorna 'false', debido a que el valor de la cadena no se corresponde a un NIF para españoles menores de 14 años o extranjeros sin NIE que de forma transitoria necesitan una identificación válida.
