esNatural(String numero): boolean

DESCRIPCIÓN

  • Valida si el valor introducido es no nulo, numérico, entero y positivo.

PARÁMETROS DE ENTRADA

  • numero: Número natural a validar.

PARÁMETROS DE SALIDA

  • boolean : true si el número es un numero natural y false si el número no es un número natural.

EJEMPLOS

Ejemplo 1 Validar si la cadena 4,0 es un número natural.

var valido = PLUGIN_VALIDACIONES.esNatural('4,0');

La función esNatural retorna 'false', debido a que la cadena no representa un número natural.


Ejemplo 2 Validar si la cadena -77 es un número natural.

var valido = PLUGIN_VALIDACIONES.esNatural('-77');

La función esNatural retorna 'false', debido a que la cadena no representa un número natural.


Ejemplo 3 Validar si la cadena 5 es un número natural

var valido = PLUGIN_VALIDACIONES.esNatural('5');

La función esNatural retorna 'true', debido a que la cadena representa un número natural.
