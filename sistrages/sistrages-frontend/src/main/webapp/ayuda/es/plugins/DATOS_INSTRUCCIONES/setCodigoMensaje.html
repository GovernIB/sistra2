setCodigoMensaje(String codigo):void

DESCRIPCIÓN

  • Método que permite recuperar la información a partir de un código, que tendrá que estar previamente definido.
  • LLa información se define en la pantalla del script, en la pestaña de mensajes validación.

PARÁMETROS DE ENTRADA

  • codigo: String con el identificador del mensaje de error.

EJEMPLOS

Pongamos por caso que queremos referenciar desde el script una información, previamente definida en la pestaña de mensajes de validación, con el identificador 'CodFechas'.

La información se fijaría a nivel de script mediante el código siguiente:

DATOS_INSTRUCCIONES.setCodigoMensaje('CodFechas');
