DATOS_PARAMETROSINICIALES

DESCRIPCIÓN

  • Objeto que permite establecer los parámetros iniciales de la versión de un trámite.
  • Este objeto está disponible desde las propiedades de la versión de un trámite, para el script "Script parámetros iniciales".

MÉTODOS

  • addParametro(codigo, valor)
