crearValorMultiple(): ValorCampoMultiple

DESCRIPCIÓN

  • Método para crear un valor múltiple para después poder utilizarlo en un campo del formulario actual.

PARÁMETROS DE SALIDA

  • ValorCampoMultiple: Crea el valor del campo múltiple.

EJEMPLOS

Por ejemplo, si deseamos generar un conjunto de datos geográficos predefenidos, primero crearíamos la variable:

var datos = DATOS_FORMULARIOS.crearValorMultiple();

y luego añadiríamos los datos a la variable, siendo el primer parámetro de entrada el código y el segundo parámetro el valor:

datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');
