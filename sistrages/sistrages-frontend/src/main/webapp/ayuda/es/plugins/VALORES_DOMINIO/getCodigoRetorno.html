getCodigoRetorno():String

DESCRIPCIÓN

  • En la recuperación del dominio se puede establecer de forma opcional un código de retorno, siempre y cuando se produzca correctamente la resolución del dominio.

PARÁMETROS DE SALIDA

  • String: Objeto String que modeliza el código de retorno.

EJEMPLOS

Pongamos por caso que se quiere recuperar el código de retorno que devuelve el domini 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

y se recupera el código de retorno de la siguiente manera:

var retCode = valores.getCodigoRetorno();
