  • esFecha

esFecha(String fecha, String formatoFecha): boolean

DESCRIPCIÓN

  • Valida que la fecha introducida no sea nula ni vacía.

PARÁMETROS DE ENTRADA

  • fecha: Fecha a validar.
  • formatoFecha: Formato de la fecha. Es opcional y el formato por defecto es dd/MM/yyyy .

PARÁMETROS DE SALIDA

  • boolean : true si la fecha es válida y false si la fecha no es válida.

EJEMPLOS

Ejemplo 1 Validar una cadena de caracteres vacía ''

var valido = PLUGIN_UTILS.esFecha('', null);

La función esFecha retorna 'false', debido a que el valor indicado no es una fecha válida.


Ejemplo 2 Validar una cadena de caracteres nula.

var valido = PLUGIN_UTILS.esFecha(null, null);

La función esFecha retorna 'false', debido a que el valor indicado no es una fecha válida.


Ejemplo 3 Validar la cadena de caracteres '10/06/2012'.

var valido = PLUGIN_UTILS.esFecha('10/06/2012', 'yyyy-MM-dd');

La función esFecha retorna 'false', debido a que el valor indicado no es una fecha válida, no sigue el formato yyyy-MM-dd.


Ejemplo 4 Validar la cadena de caracteres '10/06/2012'.

var valido = PLUGIN_UTILS.esFecha('10/06/2012', 'dd/MM/yyyy');

La función esFecha retorna 'true', debido a que el valor indicado es una fecha válida con el formato dd/MM/yyyy.


Ejemplo 5 Validar la cadena de caracteres '2012-12-09'.

var valido = PLUGIN_UTILS.esFecha('2012-12-09', null);

La función esFecha retorna 'true', debido a que el valor indicado es una fecha válida.
