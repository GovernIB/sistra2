getValor(String campo): String

DESCRIPCIÓN

  • Método que devuelve el valor de un campo simple.

PARÁMETROS DE ENTRADA

  • campo: Cadena que contiene el identificador del campo.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres que representa el valor del campo.

EJEMPLO

Obtener el valor del campo simple 'campoNif':

var valor = PLUGIN_DATOSFORMULARIO.getValor('campoNif');

La función getValor retornará el valor del campoNif, en este caso '11111111H'.
