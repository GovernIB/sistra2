getValorElemento(idFormulario,listaElementos,numElemento,campo): String

DESCRIPCIÓN

  • Método que retorna el valor de un elemento de un campo de tipo lista de elementos.

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario.
  • listaElementos: Identificador del campo de la lista de elementos.
  • numElemento: Número de elemento (empieza en 0).
  • campo: Nombre del campo en la lista de elementos.

PARÁMETROS DE SALIDA

  • String: Recupera el valor del campo.

Ejemplo

Si se desea obtener el valor del campo del primer elemento, en la lista de elemento de un formulario concreto, utilizaremos la función de la siguiente forma:

var valor = PLUGIN_FORMULARIOS.getValorElemento('FORM1','ID_LISTAELEMENTOS1',0,'ID_TEXTO1');
