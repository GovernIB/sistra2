setFormularioIncorrecto(idFormulario): void

DESCRIPCIÓN

  • Método que marca un formulario ya completado para que tenga que ser revisado

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario.

EJEMPLOS

Si por ejemplo, mediante una validación, detectamos que un campo está erróneamente introducido en un formulario (identificador 'FORM1'), podemos obligar al usuario a revisar los datos. La línea de código a introducir en el script sería:

DATOS_FORMULARIOS.setFormularioIncorrecto('FORM1');
