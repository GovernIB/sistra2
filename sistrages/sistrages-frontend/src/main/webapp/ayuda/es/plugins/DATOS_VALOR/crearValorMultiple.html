crearValorMultiple(): ValorCampoMultiple

DESCRIPCIÓN

  • Método para crear un valor múltiple para después poder utilizarlo en un campo del formulario actual.

PARÁMETROS DE SALIDA

  • ValorCampoMultiple: Retorna el objeto ValorCampoMultiple inicializado, el cual permite añadir nuevos valores

EXEMPLES

Por ejemplo, para generar un conjunto de datos geográficos predefenidos, usaremos esta función para inicializar la variable:

var datos = DATOS_VALOR.crearValorMultiple();

de esta manera podemos añadir los nuevos valores a la variable usando el metodo "addValor(codigo, descripcion)", donde el primer parámetro indica el código y el segundo parámetro el valor:

datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');
