setValor(valor): void

DESCRIPCIÓN

  • Método que establece el valor de un campo simple del formulario actual.

PARÁMETROS DE ENTRADA

  • valor: Valor que se asigna al campo.

EJEMPLOS

Establecer el valor '07' al campo con identificador 'codProvincia', :

DATOS_VALOR.setValor('07');
