getApellidosNombre(): String

DESCRIPCIÓN

  • Método que devuelve el nombre completo del usuario autenticado con el siguiente formato: 'apellido1 apellido2, nombre'.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el nombre completo del usuario autenticado.

EJEMPLO

Si se desea obtener el nombre completo del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener el nombre completo mediante la función getApellidosNombre:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nombreCompleto = usuario.getApellidosNombre();
}

La función getApellidos retornará la cadena 'Pérez Rodríguez, Jose'.
