setLibro(String libro): void

DESCRIPCIÓN

  • Método que permite establecer el libro de registro sobre el que se hará el apunte registral de entrada de las solicitudes asociadas a la versión del trámite.

PARÁMETROS DE ENTRADA

  • libro: Identificador del libro de registro.

EJEMPLOS

Pongamos por caso que el libro sobre el que queremos registrar la solicitud depende de un valor del formulario. En tal caso utilizariamos el script para fijar el valor de dicha propiedad. Para ello incorporaríamos la siguiente línea de código:

DATOS_REGISTRO.setLibro('idLibro');
