getApellido1(): String

DESCRIPCIÓN

  • Método que devuelve el primer apellido del usuario autenticado.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el primer apellido del usuario autenticado.

EJEMPLO

Si se desea obtener el primer apellido del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener su primer apellido mediante la función getApellido1:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellido = usuario.getApellido1();
}

La función getApellido1 retornará la cadena 'Pérez'.
