setValor(idFormulario, campo, valor): void

DESCRIPCIÓN

  • Método que permite establecer el valor de un campo simple. Modificar un formulario implica que se marcará con estado incorrecto para forzar su revisión. Solo se podrán modificar campos del formulario actual o posteriores a este

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario en el que quiere modificar el valor del campo.
  • campo: Campo al que se desea asignar el valor.
  • valor: Valor que se asigna al campo.

EJEMPLOS

Podemos querer establecer el valor de un campo del formulario actual, con identificador FORM1, y rellenarlo con un valor concreto. Para ello deberemos utilizar la línea de código siguiente:

DATOS_FORMULARIOS.setValor('FORM1', 'codProvincia', '07');
