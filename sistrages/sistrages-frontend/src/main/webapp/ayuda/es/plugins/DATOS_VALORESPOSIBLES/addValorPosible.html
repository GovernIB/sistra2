addValorPosible(String codigo, String descripcion): void

DESCRIPCIÓN

  • Método que permite añadir un nuevo valor al campo actual.

PARÁMETROS DE ENTRADA

  • codigo: Cadena de texto que representa el código del elemento
  • descripcion: Cadena de texto que representa la descripcion del elemento.

EXEMPLES

Añadir valores posibles al campo actual, por ejemplo, si el campo actual es un selector de municipios:

DATOS_VALORESPOSIBLES.addValorPosible('46','Provincia de Valencia'); DATOS_VALORESPOSIBLES.addValorPosible('03','Provincia de Alicante');

De esta manera estos valores quedarán añadidos al campo selector.
