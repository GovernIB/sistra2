esNifPersonaJuridica(String nif): boolean

DESCRIPCIÓN

  • Valida si el valor introducido posee el formato requerido para ser un NIF de persona jurídica y entidades (antiguo CIF) válido. Se considera válido toda aquella cadena que tenga 1 letra según la naturaleza jurídica de la entidad, 7 números y un carácter alfanumérico que cumpla con la regla de validación (que según la naturaleza jurídica de la entidad es un número o una letra).

PARÁMETROS DE ENTRADA

  • nif: cadena que representa el NIF de persona jurídica a validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena es un NIF de persona jurídica válido y false si la cadena no es un NIF de persona jurídica válido.

EJEMPLOS

Ejemplo 1 Validar la cadena 'B88018098'.

var valido = PLUGIN_UTILS.esNifPersonaJuridica('B88018098');

La función esNifPersonaJuridica retorna 'true', debido a que la cadena contiene un NIF de persona jurídica válido.


Ejemplo 2 Validar la cadena '4854391C.

var valido = PLUGIN_UTILS.esNifPersonaJuridica('4854391C');

La función esNifPersonaJuridica retorna 'false', debido a que el valor de la cadena no se corresponde a un NIF de persona jurídica válido.
