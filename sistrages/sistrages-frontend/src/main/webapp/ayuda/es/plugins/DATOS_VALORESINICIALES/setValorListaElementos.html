setValorListaElementos(campo,valorCampoListaElementos): void

DESCRIPCIÓN

  • Método que establece un valor lista elemento a un campo del formulario actual.

PARÁMETROS DE ENTRADA

  • campo : Identificador del campo del formulario.
  • valorCampoListaElementos : Objeto de tipo ValorListaElementos el cual contiene los diferentes valores a asignar al campo.

EJEMPLOS

Se inicializa una variable para asignarla a un campo multirregistro del formulario actual e ir añadiendo valores mediante la función addValor:

var datos = DATOS_VALOR.crearValorListaElementos(); var valorElemento=datos.addElemento(); valorElemento.addValor('codigo', '07073'); valorElemento.addValor('descripcion', 'Mallorca');

Una vez creada la variable, creado el elemento y asignados los valores, se podrá asignar al campo correspondiente.

DATOS_VALOR.setValorListaElementos('TABLA_MUNICIPIOS',datos);
