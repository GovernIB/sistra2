getValor(String idFormulario,String campo): String

DESCRIPCIÓN

  • Método que retorna el valor de un campo simple.

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.
  • campo: Cadena que contiene el identificador del campo.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres que representa el valor de un campo simple.

EJEMPLO

Obtener el valor del campo simple 'campoNif' ubicado en el formulario con id 'Form1':

var valor = PLUGIN_FORMULARIOS.getValor('Form1','campoNif');

La función getValor retornará el valor '11111111H', el cual corresponde con el contenido del campo 'campoNif' del formulario 'Form1'.
