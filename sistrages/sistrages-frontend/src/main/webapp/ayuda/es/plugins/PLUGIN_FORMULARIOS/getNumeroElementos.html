getNumeroElementos(idFormulario,campoListaElementos): int

DESCRIPCIÓN

  • Método que retorna el número de elementos de un campo lista elementos.

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario.
  • campoListaElementos: Identificador del campo lista de elementos.

PARÁMETROS DE SALIDA

  • int: Recupera el número de elementos.

Ejemplo

Si se desea obtener el número de elementos que tienen internamente un campo lista de elementos en un formulario concreto, utilizaremos la función de la siguiente forma (suponiendo que el identificador de la lista de elementos es 'LISTA_ELEM1'):

var numeroElementos = PLUGIN_FORMULARIOS.getNumeroElementos('FORM1','LISTA_ELEM1');

La función getNumeroElementos() devolverá el total de elementos que tiene dicho lista de elementos. En caso de no tener ninguno, devolverá 'null'.
