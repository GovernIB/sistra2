setActivarRepresentacion(false)

DESCRIPCIÓN

  • Método que permite activar o desactivar la representación (por defecto habilidada).

EJEMPLOS

Por ejemplo, supongamos deseamos desactivar la representación aunque introduzcamos el script:

DATOS_REPRESENTACION.setActivarRepresentacion(false);
