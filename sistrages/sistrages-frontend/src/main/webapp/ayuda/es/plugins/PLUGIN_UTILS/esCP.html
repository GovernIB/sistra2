esCP(String codigoPostal, String provincia): boolean

DESCRIPCIÓN

  • Valida si el código postal introducido es un código Postal válido. Se considerará código postal válido toda aquella cadena que cumpla con el formato de número de carácteres correcto y las reglas respectivas a la correspondencia alfanumérica.
  • Si se indica un número de provincia, se validará además que el CP pertenezca a la provincia indicada

PARÁMETROS DE ENTRADA

  • codigoPostal: Cadena de 5 caracteres que contiene el código postal.
  • provincia: Opcional. Es el código de la Provincia, p.e. el 07 para Illes Balears o el 46 para Valencia. Si se informa, se valida que sea numérico y que el CP pertenezca a la provincia indicada

PARÁMETROS DE SALIDA

  • boolean: true si el codigoPostal es válido y false si el codigoPostal es no válido.

EJEMPLOS

Ejemplo 1 Verificar si el código Postal 07007 es válido. Se indica un código de provincia nulo

var valido = PLUGIN_UTILS.esCP('07007', null);

La función esCP retorna 'true', ya que el valor pasado como código postal es válido.


Ejemplo 2 Verificar si el código Postal 07007 es válido. Se indica un código de provincia vacío

var valido = PLUGIN_UTILS.esCP('07007', '');

La función esCP retorna 'true', ya que el valor pasado como código postal es válido.


Ejemplo 3 Verificar si el código Postal 07007 es válido para la provincia 'Illes Balears' (07).

var valido = PLUGIN_UTILS.esCP('07007', '07');

La función esCP retorna 'true', ya que el valor pasado como código postal es válido y se corresponde con la provincia Illes Balears.


Ejemplo 4 Verificar si el código Postal 96009 es válido.

var valido = PLUGIN_UTILS.esCP('96009', '');

La función esCP retorna 'false', ya que el valor pasado como código postal no es un código postal válido.


Ejemplo 5 Verificar si el código Postal 07007 es válido para la provincia 'Las palmas' (35).

var valido = PLUGIN_UTILS.esCP('07007', '35');

La función esCP retorna 'false', ya que el valor pasado como código postal es válido, pero no se corresponde con la provincia 'Las Palmas'.
