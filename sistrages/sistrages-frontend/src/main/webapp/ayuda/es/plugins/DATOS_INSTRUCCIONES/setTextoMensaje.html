setTextoMensaje(String texto):void

DESCRIPCIÓN

  • Método que permite fijar directamente por script el texto de la información.

PARÁMETROS DE ENTRADA

  • texto: String con el texto pertinente a la información.

EJEMPLOS

Pongamos por caso que queremos establecer una instrucción directamentee desde el script. Lo podríamos hacer a través del script de la manera siguiente:

DATOS_INSTRUCCIONES.setTextoMensaje('Debe rellenar todos los pasos);
