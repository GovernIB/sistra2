getValor(int numFila, String nomColumna):String

DESCRIPCIÓN

  • Método que obtiene el valor de una columna a una fila. Si el valor del campo es nulo, se retorna una cadena vacía.
  • Este método está disponible desde el método PLUGIN_DOMINIO.invocarDominio (accesible desde todos los scripts de versión, paso y formulario).

PARÁMETROS DE ENTRADA

  • numFila: Número de fila (comienzan en 1).
  • nomColumna: Nombre de la columna.

PARÁMETROS DE SALIDA

  • String: Objeto de tipo String con el valor de la columna para la fila recuperada.

EJEMPLOS

Pongamos por caso que se quiere recuperar el valor de la columna 'column1' para la primera fila retornada en la invocación de un dominio 'DOMINIO'. Los pasos a efectuar serían los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var listColumns = valores.getNombreColumnas();
var column1 = listColumns.get(0);

y se recupera el valor de la columna 'column1' para la fila 1 de la manera siguiente:

var valorC1F1 = valores.getValor(1, column1);
