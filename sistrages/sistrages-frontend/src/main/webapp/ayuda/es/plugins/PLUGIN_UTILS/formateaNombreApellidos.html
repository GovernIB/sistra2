formateaNombreApellidos(String formato, String nombre, String apellido1, String apellido2): String

DESCRIPCIÓN

  • Método que permite combinar el nombre y los apellidos en función del parámetro formato.

PARÁMETROS DE ENTRADA

  • formato: Cadena de caracteres que indica el formato a usar. Los valores posibles son:
    • AN : apellido1 apellido2, nombre
    • NA : nombre apellido1 apellido2
  • nombre: Nombre completo.
  • apellido1: Primer apellido.
  • apellido2: Segundo apellido.

PARÁMETROS DE SALIDA

  • String: Cadena con el nombre y los apellidos formateados según el parámetro formato.

EJEMPLOS

Ejemplo 1 Supongamos que se desea dar formato (primer_apellido segundo_apellido, nombre) a los siguientes datos: Nombre:Pedro, Apellido1: Rosas Apellido2: Fuentes.

var pCadena = PLUGIN_UTILS.formateaNombreApellidos('AN', 'Pedro', 'Rosas', 'Fuentes');

La función formateaNombreApellidos retorna la siguiente cadena de caracteres 'Rosas Fuentes, Pedro'.


Ejemplo 2Supongamos que se desea dar formato (nombre primer_apellido segundo_apellido) a los siguientes datos: Nombre:Sonia, Apellido1: Arizmendi Apellido2: Cardenas.

var pCadena = PLUGIN_UTILS.formateaNombreApellidos('NA', 'Sonia', 'Arizmendi', 'Cardenas');

La función formateaNombreApellidos retorna la siguiente cadena de caracteres 'Sonia Arizmendi Cardenas'.
