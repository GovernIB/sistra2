esURL(String url): boolean

DESCRIPCIÓN

  • Valida que el valor introducido no sea nulo y cumpla con las reglas de validación necesarias para ser una URL válida. Se considera válida toda aquella cadena que cumpla con las reglas de validación correspondientes a una URL.

PARÁMETROS DE ENTRADA

  • url: Cadena que representa la URL (dirección de internet).

PARÁMETROS DE SALIDA

  • boolean: true si la cadena representa una URL válida y false si la cadena no representa una URL válida.

EJEMPLOS

Ejemplo 1 Validar la cadena 'http://www.caib.es/'

var valida = PLUGIN_VALIDACIONES.esURL('http://www.caib.es/');

La función esUrl retorna 'true', debido a que la cadena representa una URL válida.


Ejemplo 2 Validar la cadena 'http://www.caib.sedeelectronica/'

var valida = PLUGIN_VALIDACIONES.esURL('http://www.caib.sedeelectronica');

La función esUrl retorna 'false', debido a que la cadena no representa una URL válida.
