validaFechaFin(String fecha1, String fecha2): int

DESCRIPCIÓN

  • Método que permite comparar 2 fechas. Las fechas deben tener el formato yyyy-MM-dd.

PARÁMETROS DE ENTRADA

  • fecha1: Cadena de texto que representa la primera fecha. El formato de la cadena debe ser yyyy-MM-dd.
  • fecha2: Cadena de texto que representa la segunda fecha. El formato de la cadena debe ser yyyy-MM-dd.

PARÁMETROS DE SALIDA

  • int: Número entero, los posibles valores a devolver son:
    1. 0: Si fecha1 y fecha2 son iguales.
    2. 1: Si fecha1 es superior a fecha2.
    3. -1: Si fecha1 es menor a fecha2.

EJEMPLOS

Ejemplo 1 Comparar las siguientes fechas '04 de junio de 2018' y '11 de junio de 2018'.

var valida = PLUGIN_UTILS.validaFechaFin('2018-06-04', '2018-06-11');

La función validaFechaFin retorna -1, ya que fecha2 es mayor que fecha1.


Ejemplo 2 Comparar las siguientes fechas '15 de mayo de 2018' y '12 de mayo de 2018'.

var valida = PLUGIN_UTILS.validaFechaFin('2018-05-15', '2018-05-12');

La función validaFechaFin retorna 1, ya que fecha1 es mayor que fecha2.
