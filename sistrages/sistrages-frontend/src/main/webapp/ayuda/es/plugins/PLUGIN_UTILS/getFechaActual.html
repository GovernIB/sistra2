getFechaActual(String formatoFecha): String

DESCRIPCIÓN

  • Método que devuelve la fecha actual del sistema. Indicar que si se quiere pasar esta fecha a campos tipo 'Fecha', el formato de la fecha obtenida debe ser el que viene por defecto. Por otra parte, si se quiere pasar a un campo tipo 'Texto', la fecha puede pasarse con el formato que se desee.

PARÁMETROS DE ENTRADA

  • formatoFecha: Formato de la fecha. Es opcional y el formato por defecto es dd/MM/yyyy.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que representa la fecha actual del sistema, con el formato especificado (por defecto, dd/MM/yyyy).

EJEMPLOS

Ejemplo 1 Supongamos que hoy es 7 de enero de 2019, y deseamos recuperar la fecha actual del sistema (con el formato por defecto)

var hoy = PLUGIN_UTILS.getFechaActual(null);

La función getFechaActual retorna '07/01/2019'.


Ejemplo 2 Supongamos que hoy es 7 de enero de 2019, y deseamos recuperar la fecha actual del sistema (con el formato 'yyyy-MM-dd')

var hoy = PLUGIN_UTILS.getFechaActual('yyyy-MM-dd');

La función getFechaActual retorna '2019-01-07'.


Ejemplo 3 Supongamos que hoy es 7 de enero de 2019, y deseamos recuperar la fecha actual del sistema (con el formato 'MM/dd/yyyy')

var hoy = PLUGIN_UTILS.getFechaActual('MM/dd/yyyy');

La función getFechaActual retorna '01/07/2019'.


Ejemplo 4 Supongamos que hoy es 7 de enero de 2019 a las 10:53:12, y deseamos recuperar la fecha actual del sistema (con el formato 'MM/dd/yyyy HH:mm:ss')

var hoy = PLUGIN_UTILS.getFechaActual('dd/MM/yyyy HH:mm:ss');

La función getFechaActual retorna '07/01/2019 10:53:12'.


Exemple 5 Esta función también puede retornar solo la hora con la presición que se desee. Por ejemplo, si queremos obtener la hora y minutos actuales:

var hoy = PLUGIN_UTILS.getFechaActual('HH:mm');

La función getFechaActual retorna la hora y minutos actuales.
