getFechaActual(): String

DESCRIPCIÓN

  • Método que devuelve la fecha actual del sistema.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que representa la fecha actual del sistema, con el formato yyyy-MM-dd.

EJEMPLOS

Supongamos que hoy es 7 de enero de 2019, y deseamos recuperar la fecha actual del sistema

var hoy = PLUGIN_UTILS.getFechaActual();

La función getFechaActual retorna '2019-01-07'.
