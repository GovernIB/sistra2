hayDistanciaAnyos(String fechaIni, String fechaFin, String formatoFecha, int distanciaAnyos): boolean

DESCRIPCIÓN

  • Valida que la distancia mínima entre dos fechas sea como mínimo del parámetro de distancia que se pasa por entrada.

PARÀMETROS DE ENTRADA

  • fechaIni: Fecha inicial menor que la fechaFin.
  • fechaFin: Fecha final mayor que la fechaIni.
  • formatoFecha: Parámetro opcional que indica el formato de las fechas para convertirlo en una variable de tipo fecha. Si se pasa nulo, el formato fecha es 'dd/MM/YYYY' (por ejemplo, una fecha en ese formato sería '30/12/2019').
  • distanciaAnyos: Número mínimo de años de distancia.

PARÀMETROS DE SALIDA

  • boolean: true si la distancia en años entre ambas fechas es como mínimo el parámetro añosDistancia, false si no cumple el mínimo en anyos de distancia entre ambos.

EJEMPLOS

Ejemplo 1 Validar si la distancia entre dos fechas es de al menos 3 años.

var valido = PLUGIN_UTILS.hayDistanciaAnyos('01/06/2019','05/07/2015', null,3);

La función hayDistanciaAnyos devuelve 'true' porque la distancia de años entre ambas fechas es mayor o igual a 3 años.


Ejemplo 2 Validar si la distancia entre dos fechas es de al menos 5 años.

var valido = PLUGIN_UTILS.hayDistanciaAnyos('01/01/2018','05/07/2015','dd/MM/YYYY',5);

La función hayDistanciaAnyos devuelve 'false' porque la distancia de años entre ambas fechas no es mayor e igual a 5 años.
