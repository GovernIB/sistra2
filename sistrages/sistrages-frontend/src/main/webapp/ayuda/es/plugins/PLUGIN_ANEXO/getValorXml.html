getValorXml(String xpath): String

DESCRIPCIÓN

  • Método que devuelve el valor de un campo del XML anexado. Este método únicamente funcionará para anexos con formato XML.

PARÁMETROS DE ENTRADA

  • xpath: xpath del campo del XML anexado.

PARÁMETROS DE SALIDA

  • String: Contenido del campo del XML anexado. Si el campo no existe, devuelve una cadena vacía.

EJEMPLO

Para poder hacer uso de esta funcionalidad, es necesario conocer de antemano la estructura del XML.

Supongamos que deseamos validar que un determinado XML cumple con una estructura concreta, validando por ejemplo que incluye un campo 'dni'. En este caso podríamos usar la siguiente sentencia para obtener el campo 'dni' ubicado en el XML, y posteriormente verificar que sea un DNI válido.

<datos>
<dni>11111111H</dni>
<nombre>Nombre Apellido1 Apellido2</nombre>
</datos>

var dni = PLUGIN_ANEXO.getValorXml('//datos/dni');
