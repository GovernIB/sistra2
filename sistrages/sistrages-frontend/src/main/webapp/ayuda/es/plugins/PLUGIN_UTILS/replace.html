replace(String texto, String oldChar, String newChar): String

DESCRIPCIÓN

  • Método usado para reemplazar todas las ocurrencias de una determinada cadena de texto por otra cadena de texto.

PARÁMETROS DE ENTRADA

  • texto: Cadena de texto.
  • oldChar Cadena a reemplazar por newchar
  • newChar: Cadena de reemplazo.

PARÁMETROS DE SALIDA

  • string: cadena de texto en la que todas las ocurrencias de oldChar han sido reemplazadas por newchar.

EJEMPLOS

Ejemplo 1 Reemplazar, en la cadena de texto 'aba', la cadena 'a' por la cadena 'z'.

var cadenaFinal = PLUGIN_UTILS.replace('aba', 'a', 'z');

La función replace retorna la cadena 'zbz'.


Ejemplo 2 Reemplazar, en la cadena de texto 'adbad', la cadena 'ad' por la cadena 'mnm'.

var cadenaFinal = PLUGIN_UTILS.replace('adbad', 'ad', 'mnm');

La función replace retorna la cadena 'mnmbmnm'.
