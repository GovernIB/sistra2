esNumeroCuentaIbanValido(Strinbg iban, String entidad, String sucursal, String dc, String cuenta): boolean

DESCRIPCIÓN

  • Valida si los valores introducidos no son nulos, y cumple con las reglas de validación necesarias para ser un IBAN (International Bank Account Number) Válido. Se consideran válidas toda aquella cadena que tengan 4 números iban, 4 números para la entidad, 4 números para la sucursal, 2 números para los dígitos de control y 10 números para la cuenta corriente y que cumplan con las reglas de validación correspondientes a la combinación de código iban (clave del país y código de control), entidad, oficina, dígito de control y número de cuenta.

PARÁMETROS DE ENTRADA

  • iban: Código de 4 dígitos del iban siendo los 2 primeros dígitos el código del país ('ES' para España) y los otros 2 son código de control.
  • entidad: Código de 4 dígitos de la entidad bancaria.
  • sucursal: Código de 4 dígitos de la sucursal.
  • dc: Código de 2 dígitos para los dígitos de control.
  • cuenta: Código de 10 dígitos del Número de cuenta.

PARÁMETROS DE SALIDA

  • boolean: true si iban+entidad+sucursal+dc+cuenta forman un número de cuenta válido y false si entidad+sucursal+dc+cuenta no forman un número de cuenta válido.

EJEMPLOS

Ejemplo 1 Validar los siguientes valores: iban: 'ES66', entidad: '2100', sucursal: '0418', dc: '46', cuenta: '1234567891'.

var valido = PLUGIN_UTILS.esNumeroCuentaIbanValido('ES66', '2100', '0418', '46', '1234567891');

La función esNumeroCuentaIbanValido retorna 'false' ya que los parámetros introducidos no forman un número de cuenta válido, el dígito de control no es correcto..


Ejemplo 2 Validar los siguientes valores: iban: 'ES66', entidad: '2100', sucursal: '0418', dc: '40', cuenta: '1234567891'.

var valido = PLUGIN_UTILS.esNumeroCuentaIbanValido ('ES66', '2100', '0418', '40', '1234567891');

La función esNumeroCuentaIbanValido retorna 'true', ya que los parámetros introducidos forman un número de cuenta válido.
