setDatosPersona(String nif, String nombre, String apellido1, String apellido2): void

DESCRIPCIÓN

  • Método que permite establecer los datos de una persona (NIF, nombre y apellidos) que actuará como interesado en el trámite, ya sea como presentador, representante o como representado.

PARÁMETROS DE ENTRADA

  • nif: NIF de la persona. Tiene que ser un NIF válido.
  • nombre: Nombre de la persona. Dato obligatorio.
  • apellido1: Primer apellido de la persona. Dato opcional.
  • apellido2: Segundo apellido de la persona. Dato opcional.

EJEMPLOS

Supongamos por caso que queremos establecer el presentador de una solicitud. Para ello incluiríamos el código siguiente en el script "Establecer presentador":

DATOS_PERSONA.setDatosPersona('1111111H', 'Pedro', 'Pérez', null);

De esta manera se establecería que el interesado de la presentación es Pedro Pérez con NIF 11111111H
