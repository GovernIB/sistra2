PLUGIN_LOG

DESCRIPCIÓN

  • Plugin que permite utilizar el log de los scripts para debugarlos en tiempo de desarrollo
  • Para que aparezca estos mensajes deberá estar habilitado el check de 'Habilitar debug' en la pantalla de control de acceso de la versión del trámite
  • Estos mensajes son visibles si en la misma sesión del navegador (abriendo una pestaña en la ventana en la que estamos tramitando), accedemos a la siguiente dirección: /sistramitfront/debug/logSesionTramitacion.html.

MÉTODOS

  • debug(mensaje)
  • retardo(timeout)
