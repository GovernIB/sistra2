esNif(String nif): boolean

DESCRIPCIÓN

  • Valida si el valor introducido es no nulo, y posee el formato y validaciones necesarias para ser un NIF válido. Se considera válido toda aquella cadena que tenga 8 números y un carácter alfanumérico que cumpla con la regla de validación.

PARÁMETROS DE ENTRADA

  • nif: Cadena que representa el NIF a validar.

PARÁMETROS DE SALIDA

  • boolean:true si la cadena es un NIF válido y false si la cadena no es un NIF válido.

EJEMPLOS

Ejemplo 1 Validar la cadena '79865432D'.

var valido = PLUGIN_UTILS.esNif('79865432D');

La función esNif retorna 'true', debido a que la cadena contiene un NIF válido.


Ejemplo 2 Validar la cadena '4854391C.

var valido = PLUGIN_UTILS.esNif('4854391C');

La función esNif retorna 'false', debido a que el valor de la cadena no se corresponde a un nif válido
