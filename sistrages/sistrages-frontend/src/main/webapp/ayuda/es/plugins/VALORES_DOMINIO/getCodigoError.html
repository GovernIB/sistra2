getCodigoError():String

DESCRIPCIÓN

  • Método que permite recuperar el código de error, siempre y cuando se produzca un error en la resolución del dominio.

PARÁMETROS DE SALIDA

  • String: Objeto String que modeliza el código de error producido.

EJEMPLOS

Pongamos por caso que se quiere recuperar el código del error producido al invocar el domini 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

y se recupera el código de error de la siguiente manera:

var errCode = valores.getCodigoError();
