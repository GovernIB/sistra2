setValorCompuesto(idFormulario, campo, codigo, descripcion): void

DESCRIPCIÓN

  • Método que establece un valor a la lista de valores de un campo de selección compuesto. Modificar un formulario implica que se marcará con estado incorrecto para forzar su revisión por parte del usuario. Solo se podrá modificar el formulario actual y posteriores.

PARÁMETROS DE ENTRADA

  • idFormulario: Identificador del formulario en el que quiere modificar el valor del campo.
  • campo: Campo al que se desea asignar el valor.
  • codigo: Código que se desea asignar al campo.
  • descripcion: Descripción que se desea asignar al campo.

EJEMPLOS

Pongamos por caso que queremos asignar a un campo de selección múltiple (provincia), del formulario actual (FORM1) un valor concreto formado por código y descripción. La línea de código a introducir sería la siguiente:

DATOS_FORMULARIOS.setValorCompuesto('FORM1', 'provincia', '07', 'Illes Balears');
