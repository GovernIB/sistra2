setDetallePago(String modelo, String concepto, String tasa, int importe): void

DESCRIPCIÓN

  • Método que permite establecer el detalle del pago.

PARÁMETROS DE ENTRADA

  • modelo: Modelo de pago.
  • concepto: Concepto de pago.
  • tasa: Código de la tasa.
  • importe: Importe del pago en céntimos. Tiene que ser un número natural (numérico, entero y positivo).

EJEMPLOS

Por ejemplo, si se desea establecer el detalle para una tasa definida en la versión de un trámite, se debería hacer de la manera siguiente:

DATOS_PAGO.setDetallePago('Modelo1', 'Tasa de admisión a examen', 'TASA1', '8950');

El resultado es un pago del modelo 'Modelo1' con un importe de 89 euros con 50 céntimos con el codigo de tasa TASA1 con el concepto 'Tasa de admisión a examen'. El modelo y el código de TASA a indicar dependerá de la plataforma de pago a utilizar
