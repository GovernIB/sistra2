isAutenticado(): boolean

DESCRIPCIÓN

  • Método que indica si un usuario está autenticado en la sesión actual.

PARÁMETROS DE SALIDA

  • boolean: Retorna 'true' si el usuario está autenticado en la sesión actual y 'false' en caso contrario.

EJEMPLO

Si estamos en un trámite en el que el usuario puede acceder de manera autenticada y sin autenticar, y deseamos comprobar si un usuario está autenticado usaremos esta función:

var autenticado = PLUGIN_SESIONTRAMITACION.isAutenticado();

La función isAutenticado retornará 'true' si el usuario de la sesión está autenticado.
