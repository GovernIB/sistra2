distanciaDiasHabiles(String fecha1, String fecha2, String formatoFecha): int

DESCRIPCIÓN

  • Método que devuelve el número de días hábiles (sin fines de semana) que hay entre una fecha y otra. No tiene en cuenta festivos.

PARÁMETROS DE ENTRADA

  • fecha1: Cadena que contiene la fecha inicial.
  • fecha2: Cadena que contiene la fecha final.
  • formatoFecha: Formato de las fechas. Es opcional y el formato por defecto es dd/MM/yyyy.

PARÁMETROS DE SALIDA

  • int: Número de días hábiles que hay de distancia entre una fecha y otra.

EJEMPLOS

Ejemplo 1 Si queremos averiguar el número de días hábiles, es decir sin contar fines de semana, que hay entre 2 fechas (se deben especificar usando el siguiente formato yyyy-MM-dd), por ejemplo, entre el 23/04/2012 y 13/06/2012 podemos usar la siguiente sentencia:

var diasHabiles = PLUGIN_UTILS.distanciaDiasHabiles('2012-04-23', '2012-06-13', 'yyyy-MM-dd');

Ejemplo 2 Si queremos averiguar el número de días hábiles, es decir sin contar fines de semana, que hay entre 2 fechas (especificando el formato dd/MM/yyyy), por ejemplo, entre el 23/04/2012 y 13/06/2012 podemos usar la siguiente sentencia:

var diasHabiles = PLUGIN_UTILS.distanciaDiasHabiles('23/04/2012', '13/06/2012', 'dd/MM/yyyy');
