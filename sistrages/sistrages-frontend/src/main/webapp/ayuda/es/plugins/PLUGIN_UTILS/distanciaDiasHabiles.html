distanciaDiasHabiles(String fecha1, String fecha2): int

DESCRIPCIÓN

  • Método que devuelve el número de días hábiles (sin fines de semana) que hay entre una fecha y otra. No tiene en cuenta festivos.

PARÁMETROS DE ENTRADA

  • fecha1: Cadena que contiene la fecha inicial, con el siguiente formato yyyy-MM-dd.
  • fecha2: Cadena que contiene la fecha final, con el siguiente formato yyyy-MM-dd.

PARÁMETROS DE SALIDA

  • int: Número de días hábiles que hay de distancia entre una fecha y otra.

EJEMPLOS

Si queremos averiguar el número de días hábiles, es decir sin contar fines de semana, que hay entre 2 fechas (se deben especificar usando el siguiente formato yyyy-MM-dd), por ejemplo, entre el 23/04/2012 y 13/06/2012 podemos usar la siguiente sentencia:

var diasHabiles = PLUGIN_UTILS.distanciaDiasHabiles('2012-04-23', '2012-06-13');
