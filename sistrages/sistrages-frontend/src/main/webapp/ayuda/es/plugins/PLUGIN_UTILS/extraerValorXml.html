extraerValorXml(String xml, String xpath): String

DESCRIPCIÓN

  • Extrae el valor del nodo XML representado por el xpath indicado.
  • El XML debe estar bien formado, y el xpath debe contener una ruta válida a un nodo del XML

PARÁMETROS DE ENTRADA

  • xml: Cadena con estructura de XML. Debe contener un XML bien formado
  • xpath:Cadena que contiene el xpath. Contiene una secuencia de tags del XML, que marcan la ruta hacia el nodo del que queremos extraer el valor.

PARÁMETROS DE SALIDA

  • String: Valor del nodo indicado en el parámetro xpath. Si la ruta indicada en el xpath no existe retorna un error

EJEMPLOS

Ejemplo 1 Extraer el valor del nodo 'dni' en el XML. Para ello, debemos revisar la estructura completa del XML, y obtener el xpath adecuado. En este caso el xpath será '/xml/persona/dni'

var xml = '<xml><persona><nombre>juan</nombre><dni>79865432D</dni></persona></xml>';
var dni = PLUGIN_UTILS.extraerValorXml(xml, '/xml/persona/dni');

La función extraerValorXml retorna '79865432D', que es el valor del nodo dni.


Ejemplo 2 Puede ocurrir que el nodo del que intentamos obtener el valor no exista en el XML. Por ejemplo, si intentamos usar el siguiente xpath '/xml/persona/apellido' usando el mismo XML del anterior ejemplo:

var xml = '<xml><persona><nombre>juan</nombre><dni>79865432D</dni></persona></xml>';
var persona = PLUGIN_UTILS.extraerValorXml(xml, '/xml/persona/apellido');

La función extraerValorXml retorna un error, ya que '/xml/persona/apellido' no existe como nodo en el XML facilitado.
