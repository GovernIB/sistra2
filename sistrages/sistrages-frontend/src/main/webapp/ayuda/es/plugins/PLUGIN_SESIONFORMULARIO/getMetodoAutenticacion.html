getMetodoAutenticacion(): String

DESCRIPCIÓN

  • Método que devuelve el método de autenticación.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que contiene el método de autenticación. Los valores posibles son:
    • 'ANONIMO'
    • 'CLAVE_CERTIFICADO'
    • 'CLAVE_PIN'
    • 'CLAVE_PERMANENTE'

EJEMPLO

Obtener el método de autenticación de la actual sesión, en este caso es un usuario no autenticado:

var metodo = PLUGIN_SESIONFORMULARIO.getMetodoAutenticacion();

La función getMetodoAutenticacion retornará la cadena 'ANONIMO'.
