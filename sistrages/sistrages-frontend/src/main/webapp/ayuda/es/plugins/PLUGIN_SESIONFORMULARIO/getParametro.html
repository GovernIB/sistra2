getParametro(String idParametro): String

DESCRIPCIÓN

  • Método que permite recuperar el valor de los parámetros que se establecen en el script de parámetros de apertura.

PARÁMETROS DE ENTRADA

  • idParametro:Cadena del identificador del parámetro de entrada.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el valor del parámetro.

EJEMPLO

Si en el paso rellenar establecemos en el script de parámetros de apertura del formulario un parámetro llamado "tasa" con el valor "TASA1", luego puede recuperarse en los scripts del formulario:

var valor = PLUGIN_SESIONFORMULARIO.getParametro('tasa');

La función getParametro retornará la cadena 'TASA1', y podremos usar ese valor para utilizarlo en los scripts del formulario.
