getParametro(String idParametro): String

DESCRIPCIÓN

  • Método que permite recuperar el valor de los parámetros enviados a través de la URL al iniciar el trámite. Si el parámetro no existe, devuelve una cadena vacía.

PARÁMETROS DE ENTRADA

  • idParametro:Cadena del identificador del parámetro de entrada.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el valor del parámetro.

EJEMPLO

Supongamos que tenemos un trámite que permite el pago de tasas, y queremos que en función del enlace desde el que se acceda se precargue una determinada tasa en un campo del formulario. En este caso podríamos usar esta función para que recupere un parámetro de la URL, en este caso el parámetro 'tasa'. Si la URL de acceso es como sigue: http://tramite.pruebas.com/sistra/tramiteEjemplo?tasa=TASA1:

var valor = PLUGIN_SESIONFORMULARIO.getParametro('tasa');

La función getParametro retornará la cadena 'TASA1', y podremos usar ese valor para mostrarlo en un campo del formulario.
