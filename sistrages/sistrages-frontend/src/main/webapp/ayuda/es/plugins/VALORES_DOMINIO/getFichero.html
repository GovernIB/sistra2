getFichero(id): Fichero

DESCRIPCIÓN

  • Método que permite recuperar un fichero retornado por un domini.

PARÁMETROS DE ENTRADA

  • Id: Cadena de texto que representa el identificador del fichero a recuperar.

PARÁMETROS DE SALIDA

  • Fichero: Objeto Fichero que modeliza el fichero retornado por el dominio.

EXEMPLES

Pongamos por caso que se quiere recuperar un fichero con identificador 'PDF' retornado por el 'DOMINIO'. Los pasos a efectuar son los siguientes:

Primero se invoca el dominio mediante PLUGIN_DOMINIOS:

var valores = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

y se recupera el fichero, a partir de su identificador, de la siguiente manera:

var file = valores.getFichero('PDF');
