addFirmante(String nif, String nombre): void

DESCRIPCIÓN

  • Método que añade un firmante.

PARÁMETROS DE ENTRADA

  • nif: String con el nif del firmante. Tiene que ser un NIF válido.
  • nombre: String con el nombre del firmante. Dato obligatorio.

EJEMPLOS

Pongamos por caso que queremos establecer un firmante para un formulario. En este caso cumplimentariamos el script "Establecer firmantes" con el contenido siguiente:

DATOS_FIRMANTES.addFirmante('37289620L','Pedro Pérez');

El resultado sería que el asistente de tramitación solicitará que el formulario sea firmado por la persona con NIF 37289620L.
