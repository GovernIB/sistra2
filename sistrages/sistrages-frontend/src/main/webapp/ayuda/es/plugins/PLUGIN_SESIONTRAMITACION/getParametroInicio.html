getParametroInicio(String idParametro): String

DESCRIPCIÓN

  • Método que permite recuperar el valor de los parámetros enviados a través de la URL al iniciar el trámite (o bien establecidos dinámicamente mediante el script de parámetros iniciales).
  • Para incluir parámetros en la URL de inicio del trámite se deben de añadir con el siguiente formato: param-_-valorParam (p.e. mediante la url de inicio https://servidor/sistramitfront/asistente/iniciarTramite.html?tramite=TRAM1&version=1&idioma=es&idTramiteCatalogo=123&parametros=param1-_-valor1-_-param2-_-valor2) se pasan los parámetros param1 y param2).
  • Si el parámetro no existe, devuelve una cadena vacía.

PARÁMETROS DE ENTRADA

  • idParametro: Cadena del identificador del parámetro de entrada.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres con el valor del parámetro.

EJEMPLO

Si la URL de acceso es como sigue: https://servidor/sistramitfront/asistente/iniciarTramite.html?tramite=TRAM1&version=1&idioma=es&idTramiteCatalogo=123&parametros=expediente-_-EXP1234

var valor = PLUGIN_SESIONTRAMITACION.getParametroInicio('expediente');

La función getParametroInicio retornará la cadena 'EXP1234', y podremos usar ese valor para recuperar datos de dicho expediente.
