formateaCadenaAInt(String numero, String separadorMiles: int

DESCRIPCIÓN

  • Método para convertir un string en un número int.

PARÁMETROS DE ENTRADA

  • numero: Cadena a convertir a int.
  • separadorMiles: Carácter con el que se identifica el separador de miles.

PARÁMETROS DE SALIDA

  • int: Número en formato int. Si no se puede convertir lanza un error.

EJEMPLOS

Ejemplo 1 Convertir la cadena '123.110.544.434' a int.

var nInt = PLUGIN_UTILS.formateaCadenaAInt('123.110.544.434', '.')

La función formateaCadenaAInt retorna el número 123110544434 en formato int


Ejemplo 2 Convertir la cadena '123,110,544,434' a int.

var nInt = PLUGIN_UTILS.formateaCadenaAInt('123,110,544,434', ',')

La función formateaCadenaAInt retorna el número 123110544434 en formato int.
