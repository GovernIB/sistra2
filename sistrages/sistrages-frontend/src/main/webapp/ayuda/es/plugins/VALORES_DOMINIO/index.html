VALORES_DOMINIO

DESCRIPCIÓN

  • Objeto que modeliza los valores retornados por un dominio.
  • Este objeto lo retornan los métodos de invocación del plugin PLUGIN_DOMINIO.

MÉTODOS

  • getCodigoRetorno():String
  • getFichero(id): Fichero
  • getNombreColumnas():List<String>
  • getNumeroFilas():int
  • getValor(numFila, nomColumna):String
