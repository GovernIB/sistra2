getEntorno(): String

DESCRIPCIÓN

  • Método que devuelve el entorno de la tramitación.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que contiene el entorno de tramitación, siendo el valor 'DEV' para desarrollo, 'PRE' para preproducción, 'PRO' para producción y 'SE' para servicios estables.

EJEMPLO

Obtener el entorno de la actual tramitación

var entorno = PLUGIN_SESIONTRAMITACION.getEntorno();

La función getEntorno retornará el entorno de la tramitación actual.
