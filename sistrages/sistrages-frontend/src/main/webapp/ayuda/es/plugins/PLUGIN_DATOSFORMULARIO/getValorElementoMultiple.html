getValorElementoMultiple(listaElementos,numElemento,campo,indice): ValorCampoCompuesto

DESCRIPCIÓN

  • Método que retorna el valor de un campo multiple de una lista de elementos.

PARÁMETROS DE ENTRADA

  • listaElementos: Id entificador del campo de la lista de elementos.
  • numElemento: Número de elemento (empieza en 0).
  • campo: Nombre del campo en la lista de elementos.
  • indice: Valor del índice dentro del campo múltiple. Tener en cuenta que los índices empiezan por 0.

PARÁMETROS DE SALIDA

  • ValorCampoCompuesto: Recupera el valor compuesto de un índice del campo múltiple.

Ejemplo

Obtener el último valor del campo múltiple 'campoTitulaciones' del primer elemento de la lista de elementos. Para eso, primero se ha de obtener el total de valores:

var total = PLUGIN_DATOSFORMULARIO.getNumeroValoresValorElementoMultiple('LISTA_ELEM1', 'campoTitulaciones');

La función getNumeroValoresValorElementoMultiple retornará 8 (total de valores de este campo). Con este valor podemos calcular el índex del último elemento de la siguiente manera:

if (total > 0) { var valorCompuesto = PLUGIN_DATOSFORMULARIO.getValorElementoMultiple('LISTA_ELEM1', 0, 'campoTitulaciones', total -1); }

La función getValorElementoMultiple retornará el objeto valorCompuesto situado en la posición indicada. Para obtener los datos del valor compuesto podemos obtener su código o descripción tal como se muestra a continuación:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En este caso la función getCodigo retornarà la cadena 'GRAD.SUP.INFOR' y la función getDescripcion retornarà la cadena 'Grau superior en informàtica'.
