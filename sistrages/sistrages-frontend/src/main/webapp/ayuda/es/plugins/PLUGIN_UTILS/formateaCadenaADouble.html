formateaCadenaADouble(String numero, String separadorMiles, String separadorDecimales): double

DESCRIPCIÓN

  • Método para convertir un string en un número double.

PARÁMETROS DE ENTRADA

  • numero: Cadena a convertir a doble.
  • separadorMiles: Carácter con el que se identifica el separador de miles.
  • separadorDecimales: Carácter con el que se identifica el separador de decimales. Dato obligatorio.

PARÁMETROS DE SALIDA

  • double: Número en formato double. Si no se puede convertir lanza un error.

EJEMPLOS

Ejemplo 1 Convertir la cadena '123.110.544.434,58' a double.

var nDoble = PLUGIN_UTILS.formateaCadenaADouble('123.110.544.434,58', '.', ',')

La función formateaCadenaADouble retorna el número 123110544434.58 en formato double


Ejemplo 2 Convertir la cadena '123,110,544,434.58' a double.

var nDoble = PLUGIN_UTILS.formateaCadenaADouble('123,110,544,434.58', ',', '.')

La función formateaCadenaADouble retorna el número 123110544434.58 en formato double.
