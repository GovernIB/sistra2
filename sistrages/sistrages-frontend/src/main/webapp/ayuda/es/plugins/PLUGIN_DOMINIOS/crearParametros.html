crearParametros():ParametrosDominio

DESCRIPCIÓN

  • Método que permite crear la variable de parámetros de dominio, que se tendrá que añadir y pasar posteriormente al invocar el dominio, siempre y cuando dicho dominio requiera el paso de parámetros.
  • La utilitzación de este método NO es obligatoria al consumir un dominio, únicamente será necesaria si el dominio requiere el paso de parámetros para su resolución

PARÁMETROS DE SALIDA

  • ParametrosDominio: Corresponde a la lista de parámetros que se necesita para invocar al dominio.

EJEMPLOS

Para invocar al dominio que requiere el pas de parámetros, se debe crear previamente la lista de parámetros de la siguiente forma:

var params = PLUGIN_DOMINIOS.crearParametros();

Posteriormente, antes de invocar el dominio, se tendrán que añadir los parámetros que requiera el dominio.
