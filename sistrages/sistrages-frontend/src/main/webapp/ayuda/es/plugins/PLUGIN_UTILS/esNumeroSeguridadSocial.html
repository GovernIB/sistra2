esNumeroSeguridadSocial(String nss): boolean

DESCRIPCIÓN

  • Valida que el valor introducido no sea nulo, que posea el formato correspondiente a un número de la seguridad social y cumpla con las reglas de validación necesarias para ser un NSS (Número Seguridad Social) Válido. Se considera válida toda aquella cadena que tenga 11 o 12 números que cumplan con las reglas de validación correspondientes a número y dígito de control.

PARÁMETROS DE ENTRADA

  • nss: cadena que representa el número de la seguridad social a validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena introducida es un NSS válido y false si la cadena introducida no es un nss válido.

EJEMPLOS

Ejemplo 1 Validar la cadena '12/01234567/89'.

var valido = PLUGIN_UTILS.esNumeroSeguridadSocial('12/01234567/89');

La función esNumeroSeguridadSocial retorna 'false', debido a que la cadena no representa un número de seguridad social válido.


Ejemplo 2 Validar la cadena '120123456787.

var valido = PLUGIN_UTILS.esNumeroSeguridadSocial('12/01234567/87');

La función esNumeroSeguridadSocial retorna 'true', debido a que la cadena sí que representa un número de seguridad social válido.
