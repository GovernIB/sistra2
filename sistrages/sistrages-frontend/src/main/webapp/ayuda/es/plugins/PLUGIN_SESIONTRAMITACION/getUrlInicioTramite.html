getUrlInicioTramite(): String

DESCRIPCIÓN

  • Método que retorna la URL con la que se inició el trámite.

PARÁMETROS DE SALIDA

  • string: Cadena de caracteres que representa la URL con la que se inició el trámite.

EJEMPLO

Obtener la URL con la que se inició el trámite:

var url = PLUGIN_SESIONTRAMITACION.getUrlInicioTramite();

La función getUrlInicioTramite retorna una URL similar a : 'www.caib.es/sistramitfront/asistente/iniciarTramite.html?tramite=ID_TRAMITE&version=1&idioma=es&idTramiteCatalogo=32'
