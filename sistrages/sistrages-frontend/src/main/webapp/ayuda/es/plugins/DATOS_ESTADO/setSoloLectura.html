setSoloLectura(soloLectura): void

DESCRIPCIÓN

  • Método que permite indicar si un campo es de tipo lectura o editable.

PARÁMETROS DE ENTRADA

  • soloLectura: parámetro de tipo booleano, que si es true indica que el campo es de sólo lectura, mientras que si es false establece el campo como editable.

EJEMPLOS

Por ejemplo, supongamos que el valor de un campo se calcula a partir del valor de ciertos campos rellenados con anterioridad, y queremos que la persona que rellena el formulario no pueda editarlo (porque no hace falta y los datos dejarían de ser coherentes si se alterase el valor):

DATOS_ESTADO.setSoloLectura(true);

El resultado sería el establecimiento del campo como de sólo lectura, con lo que no podría editarse.
