getNombreApellidos(): String

DESCRIPCIÓN

  • Método que retorna el nombre completo del usuario autenticado con el siguiente formato 'nombre apellido1 apellido2'.

PARÁMETROS DE SALIDA

  • String: Cadena de caracteres con el nombre completo del usuario autenticado.

EJEMPLO

Si se desea obtener el nombre completo del usuario de la sesión actual, en este caso 'Jose Pérez Rodriguez', primero se debe comprobar si la sesión es autenticada, a continuación obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener el nombre completo mediante la función getNombreApellidos:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nombreCompleto = usuario.getNombreApellidos();
}

La función getNombreApellidos retornará la cadena 'Jose Pérez Rodríguez'.
