esTelefonoFijo(String telefono): boolean

DESCRIPCIÓN

  • Valida si el valor introducido cumple con las reglas de validación necesarias para ser un número de teléfono fijo válido.
  • Se considera válida toda aquella cadena no nula que tenga 9 números y que empiece por 9.

PARÁMETROS DE ENTRADA

  • telefono: Cadena que representa el número de teléfono.

PARÁMETROS DE SALIDA

  • BOOLEAN:true si la cadena representa un teléfono fijo válido y false si la cadena no representa un teléfono fijo válido

EJEMPLOS

Ejemplo 1 Validar la cadena '971177714'.

var valido = PLUGIN_UTILS.esTelefonoFijo('971177714');

La función esTelefonoFijo retorna 'true', debido a que el teléfono se corresponde con el formato de un número de teléfono fijo.


Ejemplo 2 Validar la cadena '663467284'.

var valido = PLUGIN_UTILS.esTelefonoFijo('663467284');

La función esTelefonoFijo retorna 'false', debido a que el teléfono no se corresponde con el formato de un número de teléfono fijo. La cadena corresponde a un número de teléfono móvil.
