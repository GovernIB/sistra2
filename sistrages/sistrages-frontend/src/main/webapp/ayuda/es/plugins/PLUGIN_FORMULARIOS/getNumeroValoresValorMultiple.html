getNumeroValoresValorMultiple(String idFormulario, String campo): int

DESCRIPCIÓN

  • Método que devuelve el número de valores posibles de un determinado campo.

PARÁMETROS DE ENTRADA

  • idFormulario: Cadena que contiene el identificador del formulario.
  • campo: Cadena que contiene el identificador del campo.

PARÁMETROS DE SALIDA

  • int: Número de valores del campo múltiple.

EJEMPLO

Obtener el número de valores del campo multivalor 'campoTitulaciones' ubicado en el formulario 'idForm1':

var numero = PLUGIN_FORMULARIOS.getNumeroValoresValorMultiple('idForm1','campoTitulaciones');

La función getNumeroValoresValorMultiple retornará el número 3, lo que indica que ese campo tiene 3 posibles valores.
