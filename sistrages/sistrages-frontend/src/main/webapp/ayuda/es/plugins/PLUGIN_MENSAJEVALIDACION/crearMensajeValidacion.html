crearMensajeValidacion(String codigo): MensajeValidacion

DESCRIPCIÓN

  • Método que permite crear un mensaje de validación a partir de un mensaje definido en pestaña de mensajes de la pantalla de script.
  • El mensaje puede tener parámetros.

PARÁMETROS DE ENTRADA

  • codigo: String con el código del mensaje de validación a crear.

PARÁMETROS DE SALIDA

  • MensajeValidacion: Objecto que modeliza los datos del mensaje de validación.

EJEMPLOS

Pongamos por caso que se desea utilizar en un script un mensaje de validación existente pero particularizando el valor de sus parámetros. Los pasos a seguir serían similares a los siguientes:

Previamente en la pantalla de script, más concretamente en la pestaña de mensajes validación, se ha de definir el mensaje. Por ejemplo podríamos definir un mensaje de validación MENSAJE_001 con el siguiente valor:

var msg = PLUGIN_MENSAJEVALIDACION.crearMensajeValidacion('MENSAJE_001');

y se fijará el valor para los parámetros de dicho mensaje:

msg.addParametroMensaje('param 0');
msg.addParametroMensaje('param 1');

finalmente se recuperará el mensaje en el script de la siguiente forma:

var mensajeFormado = msg.getMensaje();
