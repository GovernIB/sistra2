esNifPersonaFisica(String nif): boolean

DESCRIPCIÓN

  • Valida si el valor introducido posee el formato requerido para ser un NIF de persona física válido (DNI, NIE o NIF Otros).

PARÁMETROS DE ENTRADA

  • nif: cadena que representa el NIF de persona física a validar.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena es un NIF de persona física válido y false si la cadena no es un NIF de persona física válida.

EJEMPLOS

Ejemplo 1 Validar la cadena '62424335N'.

var valido = PLUGIN_UTILS.esNifPersonaFisica('62424335N');

La función esNifPersonaFisica retorna 'true', debido a que la cadena contiene un NIF de persona física válida.


Ejemplo 2 Validar la cadena 'Q6361408E', que se corresponde con un NIF de persona jurídica.

var valido = PLUGIN_UTILS.esNifPersonaFisica('Q6361408E');

La función esNifPersonaFisica retorna 'false', debido a que el valor de la cadena no se corresponde a un NIF de persona física válida.
