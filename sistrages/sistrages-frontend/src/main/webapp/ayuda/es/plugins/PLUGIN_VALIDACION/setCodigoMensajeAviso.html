setCodigoMensajeAviso(String codigoMensajeAviso)

DESCRIPCIÓN

  • Método para establecer el mensaje de aviso a partir de un código de mensaje.

PARÁMETROS DE ENTRADA

  • codigoMensajeAviso: código a establecer.

EJEMPLOS

Suponiendo que en la pestaña de mensajes ya existe uno asociada a 'CODIGO_REPETIDO' (con el literal 'Código introducido repetido'), podemos setear el mensaje a mostrar de la siguiente manera:

PLUGIN_VALIDACION.setCodigoMensajeAviso('CODIGO_REPETIDO');
