getEmail(): String

DESCRIPCIÓN

  • Método que devuelve el email del usuario autenticado.

PARÁMETROS DE SALIDA

  • String: Valor con el email del usuario autenticado.

EJEMPLO

Si se desea obtener el email del usuario de la sesión actual, en este caso 'pedro@caib.es', primero se debe comprobar si la sesión es autenticada, a continuación, obtener el objeto DATOS_USUARIO mediante getUsuario. Una vez obtenido el usuario, podemos obtener su email mediante la función getEmail:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var email = usuario.getEmail();
}

La función getEmail retornará la cadena 'pedro@caib.es'.
