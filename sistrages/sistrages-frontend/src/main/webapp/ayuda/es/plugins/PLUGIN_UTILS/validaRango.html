validaRango(int valor, int minimo, int maximo): boolean

DESCRIPCIÓN

  • Compara si el parámetro 'valor' está dentro de un rango determinado.

PARÁMETROS DE ENTRADA

  • valor: Valor a validar dentro del rango.
  • minimo: Indica el límite mínimo del rango.
  • maximo: Indica el límite máximo del rango.

PARÁMETROS DE SALIDA

  • boolean: true si se encuentra dentro del rango y false si no es así.

EJEMPLOS

Validar si la cadena '25689' se encuentra dentro de los límites mínimo y máximo.

var valido = PLUGIN_UTILS.validaRango(25689, 16498, 31648);

La función validaRango retorna 'true' ya que el valor indicado se encuentra dentro del rango.
