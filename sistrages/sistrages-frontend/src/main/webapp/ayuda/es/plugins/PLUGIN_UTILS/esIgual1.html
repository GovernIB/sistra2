esIgual(String cadena1, String cadena2): boolean

DESCRIPCIÓN

  • Valida si las dos cadenas introducidas son iguales. Se considerarán iguales todas aquellas cadenas que coincidan en todos sus caracteres, o si ambas son nulas.

PARÁMETROS DE ENTRADA

  • cadena1: Primera cadena para comparar.
  • cadena2: Segunda cadena para comparar.

PARÁMETROS DE SALIDA

  • boolean : true si las cadenas son iguales y false si las cadenas no son iguales.

EJEMPLOS

Ejemplo 1 Validar si las cadenas 'abc' y 'abc' son iguales

var iguales = PLUGIN_UTILS.esIgual ('abc', 'abc');

La función esIgual retorna 'true', debido a que el contenido ambas cadenas es el mismo.


Ejemplo 2 Validar si las cadenas 'abc' y 'ABC' son iguales.

var iguales = PLUGIN_UTILS.esIgual ('abc', 'ABC');

La función esIgual retorna 'false', debido a que las cadenas no tienen el mismo contenido.
