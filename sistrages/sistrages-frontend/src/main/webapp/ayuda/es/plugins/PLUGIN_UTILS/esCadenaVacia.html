esCadenaVacia(String cadena): boolean

DESCRIPCIÓN

  • Valida si una cadena es vacía o nula. Se considerará cadena vacía o nula toda cadena que tenga valor nulo o su longitud sea igual a 0.

PARÁMETROS DE ENTRADA

  • cadena: Cadena a validar.

PARÁMETROS DE SALIDA

  • boolean: Devuelve verdadero si la cadena es vacía y falso si la cadena no es vacía.

EJEMPLOS

Ejemplo 1 Verificamos la cadena 'Certificado1'.

var vacio = PLUGIN_UTILS.esCadenaVacia('Certificado1');

La función esCadenaVacia retorna 'false', ya que la cadena no es nula y su longitud es 12.


Ejemplo 2 Verificamos una cadena con un único espacio ' '.

var vacio = PLUGIN_UTILS.esCadenaVacia(' ');

Retorna 'false', ya que la cadena no es nula y su longitud es 1.


Ejemplo 3 Verificamos una cadena sin ningún carácter.

var vacio = PLUGIN_UTILS.esCadenaVacia('');

Retorna 'true', ya que la longitud de la cadena es 0


Ejemplo 4 Verificamos una variable a la que no se ha asignado ningún valor.

var cadena;
var vacio = PLUGIN_UTILS.esCadenaVacia(cadena);

Retorna 'true', ya que el valor de la cadena es un nulo.
