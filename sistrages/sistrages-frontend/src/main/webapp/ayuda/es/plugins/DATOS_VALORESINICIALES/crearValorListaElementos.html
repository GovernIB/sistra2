crearValorListaElementos(): ValorCampoListaElementos

DESCRIPCIÓN

  • Método para crear un valor de lista de elementos para después poder utilizarlo en un campo del formulario actual.

PARÁMETROS DE SALIDA

  • ValorCampoListaElementos: Retorna el objeto ValorCampoListaElementos inicializado, el cual permite añadir nuevos valores.

EJEMPLOS

Por ejemplo, para generar una serie de valores para una lista de elementos:

var datos = DATOS_VALOR.crearValorListaElementos();

De esta manera podemos añadir los nuevos valores a la variable usando el método "addValor(codigo, descripcion)", donde el primer parámetro indica el código y el segundo parámetro el valor:

datos.addValor('codigo', '07073'); datos.addValor('descripcion', 'Mallorca');
