ltrim(String cadena): String

DESCRIPCIÓN

  • Método que elimina los espacios situados a la izquierda en la cadena.

PARÁMETROS DE ENTRADA

  • cadena: Cadena de texto a tratar

PARÁMETROS DE SALIDA

  • String: cadena de texto sin espacios por la izquierda.

EJEMPLOS

Tratar la cadena '  cadena ejemplo    ', para eliminar los espacios de la izquierda.

var pCadenaLimpia = PLUGIN_UTILS.ltrim('  cadena ejemplo    ');

La función ltrim retorna 'cadena ejemplo    '.
