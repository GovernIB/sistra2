getValorElementoCompuesto(listaElementos,numElemento,campo): ValorCampoCompuesto

DESCRIPCIÓN

  • Método que retorna el valor compuesto de un campo de una lista de elementos.

PARÁMETROS DE ENTRADA

  • listaElementos: Identificador del campo de la lista de elementos.
  • numElemento: Número de elemento (empieza en 0).
  • campo: Nombre del campo en la lista de elementos.

PARÁMETROS DE SALIDA

  • ValorCampoCompuesto: Recupera el valor del campo.

Ejemplo

Si se desea obtener el valor del campo compuesto del primer elemento, utilizaremos la función de la siguiente forma:

var valorCompuesto = PLUGIN_DATOSFORMULARIO.getValorElementoCompuesto('ID_LISTAELEMENTOS1',0,'campoProvincia');

La función getValorElementoCompuesto retornará el objeto valorCompuesto. Para obtener los datos del valor compuesto podemos pedir su código o descripción tal como se muestra a continuación:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En este caso la función getCodigo retornará la cadena '07' y la función getDescripcion retornará la cadena 'Illes Balears'.
