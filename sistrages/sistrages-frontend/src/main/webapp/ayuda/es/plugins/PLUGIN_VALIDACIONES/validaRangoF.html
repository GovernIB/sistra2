validaRangoF(float valor, float minimo, float maximo): boolean

DESCRIPCIÓN

  • Compara si el parámetro 'valor' (float) está dentro de un rango marcado.

PARÁMETROS DE ENTRADA

  • valor: Valor a validar dentro del rango. Debe ser un float.
  • minimo: Indica el límite mínimo del rango. Debe ser un float.
  • maximo: Indica el límite máximo del rango. Debe ser un float.

PARÁMETROS DE SALIDA

  • boolean: true si se encuentra dentro del rango y false si no es así.

EJEMPLOS

Comprobar que la cadena '25689.37' se encuentra dentro de los límites mímino y máximo.

var valido = PLUGIN_VALIDACIONES.validaRangoF(parseFloat('25689.37',10), parseFloat('16498.12',10), parseFloat('31648',10));

La función validaRangoF retorna 'true' ya que el valor indicado se encuentra dentro del rango.
