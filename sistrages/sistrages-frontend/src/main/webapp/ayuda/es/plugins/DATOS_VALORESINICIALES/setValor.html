setValor(campo, valor): void

DESCRIPCIÓN

  • Método que permite establecer el valor de un campo simple del formulario actual.

PARÁMETROS DE ENTRADA

  • campo: Campo al que se desea asignar el valor.
  • valor: Valor que se asigna al campo.

EJEMPLOS

Por ejemplo, podemos querer asignar un valor a un campo del formulario, como podría ser establecer el código provincial al formulario actual:

DATOS_VALORESINICIALES.setValor('codProvincia', '07');
