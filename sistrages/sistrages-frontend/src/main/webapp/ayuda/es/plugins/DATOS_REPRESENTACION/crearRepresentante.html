crearRepresentante(): Persona

DESCRIPCIÓN

  • Método que permite indicar representante. De forma opcional se podrá indicar el representante y de no indicarse se tomará el presentador como representante.

EJEMPLOS

Por ejemplo, supongamos deseamos establecer el representante:

var rpte = DATOS_REPRESENTACION.crearRepresentante();
rpte.setDatosPersona('1111111H', 'Pedro', 'Pérez', null);
