setCodigoSIA(String SIA): void

DESCRIPCIÓN

  • Método que permite establecer el código SIA de la entrada de registro. Si no se fija valor se tomará por defecto el código SIA del procedimiento asociado al trámite.

PARÁMETROS DE ENTRADA

  • SIA: Código SIA del registro.

EJEMPLO

Pongamos por caso que queremos establecer el código SIA de la entrada de registro. Para ello incorporaríamos la siguiente línea de código:

DATOS_REGISTRO.setCodigoSIA('12345');
