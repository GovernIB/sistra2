getNumeroValoresValorMultiple(String campo): String

DESCRIPCIÓN

  • Método que devuelve el número valores para un campo con valor múltiple para el formulario actual.

PARÁMETROS DE ENTRADA

  • campo: Cadena que contiene el identificador del campo múltiple.

PARÁMETROS DE SALIDA

  • String: Número de valores del campo múltiple.

EJEMPLO

Obtener el número de valores que tiene el campo múltiple 'campoTitulaciones'

var numero = PLUGIN_DATOSFORMULARIO.getNumeroValoresValorMultiple('campoTitulaciones');

La función getNumeroValoresValorMultiple retornará el total de valores, en este caso 5.
