esFecha(String hora, String formatoHora): boolean

DESCRIPCIÓN

  • Valida que la hora introducida no sea nula ni vacía o no tenga un formato incorrecto.

PARÁMETROS DE ENTRADA

  • hora: Hora a validar.
  • formatoHora: Formato de la hora. Es opcional y el formato por defecto es HH:mm.

PARÁMETROS DE SALIDA

  • boolean : true si la hora es válida y false si la hora no es válida.

EJEMPLOS

Ejemplo 1 Validar una cadena de caracteres vacía ''

var valido = PLUGIN_UTILS.esHora('', null);

La función esHora retorna 'false', debido a que el valor indicado no es una hora válida.


Ejemplo 2 Validar una cadena de caracteres nula.

var valido = PLUGIN_UTILS.esHora(null, null);

La función esHora retorna 'false', debido a que el valor indicado no es una hora válida.


Ejemplo 3 Validar la cadena de caracteres '10:06'.

var valido = PLUGIN_UTILS.esHora('10-06', 'HH:mm');

La función esHora retorna 'false', debido a que el valor indicado no es una hora válida, no sigue el formato HH:mm.


Ejemplo 4 Validar la cadena de caracteres '10:06:42'.

var valido = PLUGIN_UTILS.esHora('10:06:42', 'HH:mm:ss');

La función esHora retorna 'true', debido a que el valor indicado es una hora válida con el formato HH:mm:ss.


Ejemplo 5 Validar la cadena de caracteres '10:06'.

var valido = PLUGIN_UTILS.esFecha('10:06', null);

La función esHora retorna 'true', debido a que el valor indicado es una hora válida.
