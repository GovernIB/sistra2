distanciaHoras(String fecha1, String hora1, String fecha2, String hora2, String formatoFecha, String formatoHora): String

DESCRIPCIÓN

  • Método que devuelve el número de horas, minutos y segundos que hay entre dos fechas y/o dos horas.

PARÁMETROS DE ENTRADA

  • fecha1: Cadena que contiene la fecha inicial.
  • hora2: Cadena que contiene la hora inicial.
  • fecha2: Cadena que contiene la fecha final.
  • hora2: Cadena que contiene la hora final.
  • formatoFecha: Formato de las fechas. Es opcional y el formato por defecto es dd/MM/yyyy.
  • formatoHora: Formato de las horas. Es opcional y el formato por defecto es HH:mm.
Debe indicarse al menos dos fechas o dos horas. Si se indican fechas sin horas, la hora a considerar será 00:00:00.

PARÁMETROS DE SALIDA

  • String: Número de horas, minutos y segundos que hay de distancia entre las fechas y/o las horas, con el formato HH:mm:ss.

EJEMPLOS

Ejemplo 1 Si queremos averiguar el número de horas, minutos y segundos que hay entre 2 fechas (se deben especificar usando los siguientes formatos yyyy-MM-dd y HH:mm:ss), por ejemplo, entre el 23/04/2012 y 13/06/2012 a las 2 de la madrugada, 2 minutos y 2 segundos, podemos usar la siguiente sentencia, con lo que se obtendrá 1226:02:02:

var horas = PLUGIN_UTILS.distanciaHoras('2012-04-23', '', '2012-06-13', '02:02:02', 'yyyy-MM-dd', 'HH:mm:ss');

Ejemplo 2 Si queremos averiguar el número de horas, minutos y segundos que hay entre 2 fechas (especificando el formato dd/MM/yyyy y HH:mm), por ejemplo, entre el 23/04/2012 y 13/06/2012 podemos usar la siguiente sentencia, obteniendo 1224:00:00:

var horas = PLUGIN_UTILS.distanciaDias('23/04/2012', '', '13/06/2012', '', 'dd/MM/yyyy', '');

Ejemplo 3 Si queremos averiguar el número de horas, minutos y segundos que hay entre 2 horas (especificando el formato HH:mm:ss), por ejemplo, entre las 19:20:05 y 23:58:42 podemos usar la siguiente sentencia, obteniendo 04:38:37:

var horas = PLUGIN_UTILS.distanciaDias('', '19:20:05', '', '23:58:42', '', 'HH:mm:ss');
