setPlazoFin(String plazoFin): void

DESCRIPCIÓN

  • Método que permite establecer el fin del plazo en el que está activa la versión de un trámite. La fecha debe tener el formato dd/MM/yyyy HH:mm:ss.

PARÁMETROS DE ENTRADA

  • plazoFin: Cadena con formato de fecha/hora que representará el fin del plazo.

EJEMPLOS

Pongamos que queremos que la versión de un trámite esté disponible durante un periodo concreto. Para ello deberemos establecer la fecha fin de plazo de la manera siguiente:

DATOS_PERSONALIZACION.setPlazoFin('01/04/2019 11:59:59');
