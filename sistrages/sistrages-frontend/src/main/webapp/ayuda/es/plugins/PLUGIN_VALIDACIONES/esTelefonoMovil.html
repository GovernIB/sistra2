esTelefonoMovil(String telefono): boolean

DESCRIPCIÓN

  • Valida si el valor introducido cumple con las reglas de validación necesarias para ser un número de teléfono móvil válido.
  • Se considera válida toda aquella cadena no nula que tenga 9 números y que empiece por 6 o 7.

PARÁMETROS DE ENTRADA

  • telefono: Cadena que representa el número de teléfono móvil.

PARÁMETROS DE SALIDA

  • boolean: true si la cadena representa un teléfono móvil válido y false si la cadena no representa un teléfono móvil válido

EJEMPLOS

Ejemplo 1 Validar la cadena '971177714'.

var valido = PLUGIN_VALIDACIONES.esTelefonoMovil('971177714');

La función esTelefonoMovil retorna 'false', debido a que el teléfono no se corresponde con el formato de un número de teléfono móvil. La cadena corresponde a un número de teléfono fijo.


Ejemplo 2 Validar la cadena '663467284'.

var valido = PLUGIN_VALIDACIONES.esTelefonoMovil('663467284');

La función esTelefonoMovil retorna 'true', debido a que el teléfono se corresponde con el formato de un número de teléfono móvil.
