setPlazoInicio(String plazoInicio): void

DESCRIPCIÓN

  • Método que permite establecer el inicio del plazo en el que está activa la versión de un trámite. La fecha debe tener el formato dd/MM/yyyy HH:mm:ss.

PARÁMETROS DE ENTRADA

  • plazoInicio: Cadena con formato de fecha/hora que representará el inicio del plazo.

EJEMPLOS

Pongamos que queremos que la versión de un trámite esté disponible durante un periodo concreto. Para ello deberemos establecer la fecha inicio de plazo de la manera siguiente:

DATOS_PERSONALIZACION.setPlazoInicio('01/02/2019 11:59:59');
