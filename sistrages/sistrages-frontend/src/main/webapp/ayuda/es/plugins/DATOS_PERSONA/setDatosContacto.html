setDatosContacto(pais, provincia, municipio, direccion, codigoPostal, email, telefono): void

DESCRIPCIÓN

  • Método que permite establecer los datos de contacto de una persona que actuará como interesado en el trámite, ya sea como presentador, representante o como representado.

PARÁMETROS DE ENTRADA

  • pais: Código país (formato SICRES).
  • provincia: Código provincia (formato SICRES).
  • municipio: Código municipio (formato SICRES).
  • direccion: Dirección.
  • codigoPostal: Código postal
  • email: Email
  • telefono: Teléfono

EJEMPLOS

Supongamos por caso que queremos establecer los datos de contacto del presentador de una solicitud. Para ello incluiríamos el código siguiente en el script "Establecer presentador":

DATOS_PERSONA.setDatosContacto("108", "7", "40", "C.\ Principal 1 puerta 1", "07001", "email@dominio.es", "601234567");

De esta manera se establecería los datos de contacto.
