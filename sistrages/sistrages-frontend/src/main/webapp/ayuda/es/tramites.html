TRÁMITES

En la pantalla de 'Trámites' se puede gestionar el alta, baja y edición de áreas, trámites y versiones de trámites. Además se pueden realizar otras operativas relativas a trámite y versión de trámite que se detallan más abajo (Control de acceso, exportación/importación, mover trámite, etc.).
En esta pantalla únicamente se visualizarán las áreas y trámites, contenidos en estas, sobre las que el usuario tenga permisos. Sobre dichos trámites la pantalla permite la realización de búquedas filtradas.

El mantenimiento de las áreas la ha de hacer un usuario que disponga del rol de 'Administrador de entidad'.
El mantenimiento de trámites y versiones de un trámite vinculados a un área únicamente puede ser realizado por un usuario que disponga del rol de 'Administrador de entidad' o en su defecto por un usuario con rol 'Desarrollador' con permisos de desarrollador sobre el área al que pertencen los trámites y o versiones de estos.

OPERATIVA ÁREAS

  • Nuevo: pulsando en el botón se abrirá la ficha para dar de alta un nueva área en la cual tendrá que rellenar los campos mínimos solicitados. Esta operativa únicamente estará disponible para los administradores de entidad.
  • Editar: el botón de edición únicamente se mostrará cuando haya un área seleccionada. Al pulsar dicho botón se abrirá la ficha para poder realizar las modificaciones de los campos deseados.
  • Eliminar: el botón de eliminación únicamente se mostrará cuando haya un área seleccionada a eliminar. Previa a la eliminación del área se mostrará un mensaje de confirmación de la acción. Esta operativa únicamente estará disponible para los administradores de entidad.

CAMPOS TRÁMITES

  • Código del trámite: código unívoco que identifica al trámite en la aplicación
  • Descripción del trámite: texto descriptivo del objeto del trámite
  • Versiones activas: casilla de verificación que indica si el trámite cuenta con versiones activas
  • Área: código del área al que pertenece el trámite. Mediante este campo el usuario, con permiso sobre diferentes áreas, puede distinguir a qué área pertenece el trámite

OPERATIVA TRÁMITES

  • Buscar: introduciendo un texto en el campo 'Buscar' y pulsando el botón con el icono de la lupa se llevará a cabo un filtrado de los trámites que contengan el texto introducido. El criterio de búsqueda aplicará a cualquiera de los campos del trámite, y solo se buscará sobre los trámites de las áreas marcadas. Posteriormente se visualizarán los resultados de la búsqueda filtrándolos en el listado de trámites.
  • Nuevo Trámite: el botón de nuevo trámite únicamente se mostrará cuando haya un área seleccionada. Al pulsar el botón se abrirá la ficha para dar de alta un nuevo trámite en la cual tendrá que rellenar los campos mínimos solicitados para llevar a cabo el alta.
  • Nueva Versión: el botón de nueva versión únicamente se mostrará cuando haya un trámite seleccionado. Al pulsar el botón se abrirá la ficha para indicar el número de versión a crear y el tipo de flujo de dicha versión
  • Editar Trámite: el botón de edición de trámite únicamente se mostrará cuando haya un trámite seleccionado en el listado. Al pulsar el botón se abrirá la ficha para modificar el trámite.
  • Nueva Versión: el botón de nueva versión de trámite únicamente se mostrará cuando haya un trámite seleccionado en el listado. Al pulsar el botón se abrirá la ficha para dar de alta una nueva versión del trámite.
  • Importar Versión: el botón de importación únicamente se mostrará cuando haya un trámite o versión seleccionada, ya que lo que se importa es una versión. Al pulsar el botón se abrirá el asistente de importación.
  • Mover Trámite: el botón de mover trámite únicamente se mostrará cuando haya un trámite seleccionado. Al pulsar el botón se abrirá una ventana que nos permitirá indicar el área destino del trámite.
  • Eliminar Trámite: el botón de eliminar trámite únicamente se mostrará cuando haya un trámite seleccionado. Previo a la eliminación del trámite se mostrará un mensaje de confirmación de la acción.
  • Procedimientos: el botón de procedimientos únicamente se mostrará cuando haya un trámite seleccionado. Al pulsar el botón se presentará una ventana con el listado de procedimientos del catálogo que tienen vinculado el trámite seleccionado.
  • Dominios Área: este botón está disponible cuando se selecciona una única área, y permite acceder al listado de dominios del área seleccionada.
  • Fuentes de Datos Área: este botón está disponible cuando se selecciona una única área, y permite acceder al listado de fuentes de datos del área seleccionada.

CAMPOS VERSIÓN

  • Versión: código unívoco que identifica al trámite en la aplicación
  • Flujo: texto descriptivo del objeto del trámite
  • Activa: casilla de verificación que indica si el trámite cuenta con versiones activas
  • Release: código que identifica unívocamente un conjunto de cambios realizados sobre una versión
  • Bloqueado por: indica el código de usuario que tiene bloqueada para edición la versión
  • Huella: código aleatorio que se genera cada vez que el usuario desbloquea una versión

OPERATIVA VERSIÓN

  • Nueva Versión: el botón de nueva versión únicamente se mostrará cuando haya un trámite seleccionado. Al pulsar el botón se abrirá la ficha para indicar el número de versión a crear y el tipo de flujo de dicha versión
  • Editar Versión: el botón de edición de versión únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón se accederá a la definición de versión, que únicamente será editable si previamente se ha bloqueado.
  • Bloquear: el botón de bloqueo de versión únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón la versión quedará bloqueada por el usuario, de manera que solo él pueda editar la versió hasta que esta sea desbloqueada.
  • Desbloquear: el botón de desbloqueo de versión únicamente se mostrará cuando haya una versión de un trámite seleccionada que esté bloqueada por un usuario. Una versión bloqueada únicamente podrá ser desbloqueada por él mismo o por un administrador de área y/o entidad.
  • Previsualizar Versión: el botón de previsualización únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón se abrirá una ventana que nos permitirá realizar una previsualización, en el asistente de tramitación, de la versión del trámite.
  • Control de acceso: el botón de control de acceso únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón se abrirá una ventana que nos permitirá modificar parámetros relativos al acceso a la versión del trámite desde el asistente de tramitación.
  • Historial: el botón de historial únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón se abrirá una ventana en el que se muestra el histórico de releases de la versión, con el detalle de cada release.
  • Eliminar versión: el botón de eliminar versión únicamente se mostrará cuando haya una versión seleccionada. Previo a la eliminación de la versión se mostrará un mensaje de confirmación de la acción.
  • Exportar versión: el botón de exportación de versión únicamente se mostrará cuando haya una versión seleccionada. Para poder exportar una versión de trámite, esta deberá estar desbloqueada. Al pulsar el botón se presentará una ventana con un resumen de los datos de la versión a exportar que permitirá generar el fichero de exportación.
  • Duplicar versión: el botón de duplicación únicamente se mostrará cuando haya una versión de un trámite seleccionada. Al pulsar el botón se abrirá una ventana que permitirá fijar el área y trámite en el que se duplicará la versión. Es importante tener en cuenta que la versión se duplicará en el trámite destino con un número de versión inmediatamente superior a la versión más reciente.
  • Procedimientos: el botón de procedimientos únicamente se mostrará cuando haya un trámite seleccionado. Al pulsar el botón se presentará una ventana con el listado de procedimientos del catálogo que tienen vinculado el trámite seleccionado.
  • Dominios Área: este botón está disponible cuando se selecciona una única área, y permite acceder al listado de dominios del área seleccionada.
  • Fuentes de Datos Área: este botón está disponible cuando se selecciona una única área, y permite acceder al listado de fuentes de datos del área seleccionada.
