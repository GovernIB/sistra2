MANTENIMIENTO DE PLANTILLAS

En la pantalla de 'Mantenimiento de plantillas' se permite el alta de una plantilla o la edición de una ya existente.

CAMPOS

  • Identificador: script que permite establecer una validación a nivel de página. Este script se ejecutará al guardar una página y permitirá establecer un mensaje de validación, que se mostrará si no se satisface la validación pero permitirá continuar, o un error que impedirá continuar con la tramitación mientras no se pase la validación.
  • Formateador: indica el formateador a utilizar para la renderización de la plantilla. Los valores posibles dependerán de los formateadores disponibles para la entidad. Cada formateador requerirá una plantillas multidioma con un formato o formatos determinados.
  • Descripción: texto descriptivo del objeto de la plantilla de visualización. El objetivo de esta descripción es aportar información a los propios desarrolladores y/o administradores.
  • Por defecto: check que permite indicar si, en caso de haber varias plantillas, se ha de utilizar esta por defecto.

OPERATIVA

  • Aceptar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla descartando los cambios.
