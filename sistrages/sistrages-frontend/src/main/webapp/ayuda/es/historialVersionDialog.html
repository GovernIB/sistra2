HISTORIAL DE VERSIÓN

En la pantalla de 'Historial de versión' se presenta una lista paginada de las acciones realizadas sobre la release de una versión de un trámite.
La tabla permite establecer el número de elementos por página que se desea mostrar.
Al seleccionar un item de la tabla, se mostrará en el campo 'Detalle' de la ventana el texto descriptivo introducido por el usuario respecto a la acción realizada.

CAMPOS

  • Release: número de release de la versión, que permite mantener una trazabilidad de las misma. Este número se incrementa al realizar determinadas acciones (p.ej. Desbloquear)
  • Acción: indica el tipo de acción realizada sobre la versión (Creación, bloqueo, desbloqueo, etc...)
  • Fecha: fecha y hora en que se ha llevado a cabo la acción
  • Usuario: usuario que ha realizado la acción sobre la versión
  • Detalle: texto descriptivo de la acción sobre la versión consignado por el usuario que la ha llevado a cabo.
  • Huella: código aleatorio que se genera cada vez que el usuario desbloquea una versión

OPERATIVA

  • Guardar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla sin guardar los cambios.
