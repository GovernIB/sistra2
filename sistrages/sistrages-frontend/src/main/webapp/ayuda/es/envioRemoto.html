ENVíO REMOTO

En la pantalla de 'Enví Remoto' se pueden mantener los datos correspondientes a un dominio.
Los campos obligatorios se presentan con un asterisco *.

CAMPOS

  • Identificador: código unívoco que identifica el envío y mdeiante el cual se puede invocar el envío
  • Descripción: texto descriptivo de la funcionalidad provista por el envío
  • URL: URL del servicio al cual se quiere realizar el envío remoto. Si existe un placeholder (por haber seleccionado una variable de área), éste será sustituido por el contenido de esa variable, que será insertado al comienzo de este campo. Por ejemplo, si en la variable VAR01 aparece la URL "https://se.caib.es" y la URL de este campo ha de finalizar con "proyectoBase/services/dominios/", en este campo aparecerá el texto "{@@VAR01@@}/proyectoBase/services/dominios/".
  • Selección Variable Área: este botón abre una pantalla que permite seleccionar una variable de área que añadirá un placeholder que será sustituido por la url indicada en esa variable de área a la hora de ejecutar el trámite que utilice este envío remoto.
  • Configuración Autenticación: autenticación que se enviará al servicio en caso de estar securizado.
  • Timeout (segundos): tiempo que esperará SISTRA2 en caso de no obtener respuesta del servicio.

OPERATIVA

  • Guardar: cierra la pantalla guardando los canmbios.
  • Cancelar: cierra la pantalla sin guardar los cambios.
