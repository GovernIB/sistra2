SECCIÓN REUTILIZABLE

En la pantalla de 'Sección reutilizable' se puede llevar a cabo el mantenimiento de una sección reutilizable.
Los campos obligatorios se presentan con un arterisco *.

CAMPOS

  • Identificador: código unívoco que identifica la sección reutilizable y mediante el cual se podrá invocar la sección desde diferentes scripts de la aplicación.
  • Descripción: texto descriptivo de la funcionalidad ofrecida por la sección reutilizable.
  • Check activa: check para indicar si la seccón reutilizable está activa.
  • Editar diseño: para poder editar el diseño de una sección, esta inicialmente se tendrá que bloquear por el usuario que quiera editarla. Una vez bloqueada, clicando sobre el botón de 'Editar diseño' se abrirá el editor de formularios específico para poder editar la sección.
  • Asignar scripts: este botón permite abrir la ventana de asignación de scripts. Al hacer clic, se abrirá una ventana que permitirá editar los scripts que se asignarán a la sección.

OPERATIVA

  • Trámites: permite la consulta de los trámites asociados a la sección.
  • Guardar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla sin guardar los cambios.
