PROPIEDADES DE LA PÁGINA

En la pantalla de 'Propiedades de la página' se pueden editar aspectos relativos a la página seleccionada.

CAMPOS

  • Script de validación: script que permite establecer una validación a nivel de página. Este script se ejecutará al guardar una página y permitirá establecer un mensaje de validación, que se mostrará si no se satisface la validación pero permitirá continuar, o un error que impedirá continuar con la tramitación mientras no se pase la validación.
  • Página final: check que permite indicar si una página será final. Únicamente se permitirá una página final por fomulario.

OPERATIVA

  • Aceptar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla descartando los cambios.
