MANTENIMENTO GESTOR EXTERNO DE FORMULARIOS

En la pantalla de 'Matenimento de gestores externos' puede mantener los datos correspondientes a un gestor externo de formularios.
Los campos obligatorios se presentan con un asterisco *

CAMPOS

  • Identificador: código único identificador del gestor.
  • Descripción: texto descriptivo del gestor.
  • URL: texto que indica la url del gestor externo. Si existe un placeholder (por haber seleccionado una variable de área), éste será sustituido por el contenido de esa variable, que será insertado al comienzo de este campo. Por ejemplo, si en la variable VAR01 aparece la URL "https://se.caib.es" y la URL de este campo ha de finalizar con "proyectoBase/services/dominios/", en este campo aparecerá el texto "{@@VAR01@@}/proyectoBase/services/dominios/".
  • Selección Variable Área: este botón abre una pantalla que permite seleccionar una variable de área que añadirá un placeholder que será sustituido por la url indicada en esa variable de área a la hora de ejecutar el trámite que utilice este gestor externo de formulario.

OPERATIVA

  • Guardar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla sin guardar los cambios.
