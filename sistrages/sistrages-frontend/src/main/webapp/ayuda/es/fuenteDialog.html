MANTENIMENTO FUENTE DE DATOS

En la pantalla de 'Mantenimiento fuente de datos' puede mantener los datos correspondientes a una fuente de datos.
Los campos obligatorios se presentan con un asterisco *

CAMPS

  • Identificador: código único que identifica la fuente de datos. Este identificador será el que se deberá utilizar para referenciar a la fuente de datos desde un dominio
  • Descripción: texto descriptivo del objeto de la fuente de datos.
  • Campos: listado de campos que forma la estructura de la fuente de datos. Las operaciones que se pueden realizar sobre este listado son:
    • Nuevo: permite crear un nuevo campo en la fuente de datos. Para la creación de dicho campo se deberá indicar un código e indicar si el campo ha de actuar o no como clave primaria. Si un campo se marca como clave primaria no podrá contener, a nivel de datos, valores duplicados.
    • Editar: permite editar un campo ya existente en el listado. Antes de pulsar el botón de edición tendrá que seleccionar el campo a editar en el listado.
    • Eliminar: permite eliminar un campo ya existente en el listado. Antes de pulsar el botón de eliminación tendrá que seleccionar la propiedad a eliminar en el listado.
    • Subir: permite subir un campo en el orden del listado de campos. Para subir un campo previamente tendrá que haberlo seleccionado en el listado de campos.
    • Bajar: permite bajar un campo en el orden del listado de campos. Para bajar una campo previamente tendrá que haberlo seleccionado en el listado de campos.

OPERATIVA

  • Guardar: cierra la pantalla guardando los cambios.
  • Cancelar: cierra la pantalla sin guardar los cambios.
