PASO ANEXAR DOCUMENTO

En la pantalla de 'Paso Anexar Documento' se gestionan los documentos anexos de la versión del trámite. Se permite el alta, baja y edición de documentos anexos, así como la ordenación de los mismos para su presentación al ciudadano en el asistente de tramitación.
Esta pantalla permite establecer los documentos anexos a través de una lista fija, o de manera dinámica utilizando el script 'Lista dinámica anexos'.

OPERATIVA

  • Lista fija: tabla que muestra la lista fija de documentos anexos definidos. La operaciones permitidas para este listado son:
    • Nuevo: pulsando en el botón se abrirá la ficha que crear un documento anexo.
    • Editar: pulsando este botón se abrirá la ficha para la edición de las configuraciones del anexo seleccionado.
    • Eliminar: pulsando este botón se eliminará el anexo que se haya seleccionado previamente en el listado. Previo a la eliminación se solicitará confirmación al usuario.
    • Subir: este botón permite subir la posición del anexo seleccionado en el listado de anexos presentados.
    • Bajar: este botón permite subir la posición del anexo seleccionado en el listado de anexos presentados.
  • Lista dinánica anexos: script que permite la definición dinámica de documentos anexos.

CAMPOS

  • Identificador: código unívoco que identifica el anexo en la aplicación. Cualquier referencia al anexo en los scripts de la versión se realizará utilizando este código.
  • Descripción: texto multiidioma de presentación del anexo en el asistente de tramitación. Esta será la denominación del anexo para el tramitador.
  • Obligatoriedad: obligatoriedad del documento anexo. El documento puede ser obligatorio, opcional o dependiente de script.
