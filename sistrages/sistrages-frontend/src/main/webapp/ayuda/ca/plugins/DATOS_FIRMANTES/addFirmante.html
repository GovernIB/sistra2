addFirmante(String nif, String nombre): void

DESCRIPCIÓ

  • Mètode que hi afegeix un signant.

PARÀMETRES D'ENTRADA

  • nif: String amb el NIF del signant. Ha de ser un NIF vàlid.
  • nombre: String amb el nom del signant. Dada obligatòria.

EXEMPLES

Posam per cas que volem establir un signant per a un formulari. En aquest cas emplenarem el script "Establir signants" amb el contingut següent:

DATOS_FIRMANTES.addFirmante('37289620L','Pedro Pérez');

El resultat seria que l'assistent de tramitació sol·licitarà que el formulari sigui signat per la persona amb NIF 37289620L.
