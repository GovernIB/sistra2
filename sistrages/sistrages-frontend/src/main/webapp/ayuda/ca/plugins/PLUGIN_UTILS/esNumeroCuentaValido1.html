esNumeroCuentaValido(String cuenta): boolean

DESCRIPCIÓ

  • Valida que el valor introduït no sigui nul, i posseeixi el format corresponent a un número de compte corrent, complint amb les regles de validació necessàries per a ser un NCC (Número Compte Corrent) Vàlid. Es considera vàlida tota aquella cadena que tingui 20 números que compleixin amb les regles de validació corresponents a entitat, oficina, dígit de control i número de compte.

PARÀMETRES D'ENTRADA

  • cuenta: Codi de 20 dígits del Número de compte..

PARÀMETRES DE SORTIDA

  • boolean: true si la cadena introduïda és un número de compte vàlid i false si la cadena introduïda no és un número de compte vàlid.

EXEMPLES

Exemple 1 Validar la cadena '12345678901234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido('12345678901234567890');

La funció esNumeroCuentaValido retorna 'false' pel fet que la cadena no compleix amb les regles de validació, el dígit de control no és vàlid.


Exemple 2 Validar la cadena '12345678061234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido ('12345678061234567890');

La funció esNumeroCuentaValido retorna 'true', pel fet que la cadena compleix els requisits per a ser un número de compte vàlid.
