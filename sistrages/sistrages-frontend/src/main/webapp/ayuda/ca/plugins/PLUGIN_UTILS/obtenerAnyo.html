obtenerAnyo(String fecha, String formatoFecha): int

DESCRIPCIÓ

  • Mètode que retorna l'any de la data introduïda.

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • int: Any de la data introduïda.

EXEMPLES

Exemple 1 Recuperar l'any de la data 2019-06-22 (sense format especificat).

var anyo = PLUGIN_UTILS.obtenerAnyo('2019-06-22', null);

la funció obtenerAnyo retorna 2019.


Exemple 2 Recuperar l'any de la data 22/06/2019 (amb el format dd/MM/yyyy).

var anyo = PLUGIN_UTILS.obtenerAnyo('22/06/2019', 'dd/MM/yyyy');

la funció obtenerAnyo retorna 2019.
