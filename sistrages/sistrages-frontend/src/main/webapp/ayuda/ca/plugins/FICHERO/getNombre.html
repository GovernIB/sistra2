getNombre():String

DESCRIPCIÓ

  • Mètode que permet recuperar el nom d'un fitxer amb extensió.

PARÀMETRES DE SORTIDA

  • String: String amb el nom i extensió del fitxer.

EXEMPLES

Posam per cas que es vol recuperar el nom d'un fitxer retornat per un domini 'DOMINIO'. Les passes a efectuar són les següents:

Primer es recupera el fitxer mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fitxer = valors.getFichero('PDF');

i es recupera el nom de la manera següent:

var nomFitxer = fitxer.getNombre();
