esNumeroSwiftValido(String banc, String pais, String localitat, String sucursal): boolean

DESCRIPCIÓ

  • Valida que el valor introduït no sigui nul i que compleixi amb les regles de validació necessàries per a ser un SWIFT (codi d'identificació bancària per a realitzar transferències) Vàlid. Es considera vàlida tota aquella cadena que tingui 4 caràcters del banc, 2 caràcters per al país, 2 caràcters per a la localitat i 3 caràcters per a la sucursal (la sucursal pot ser nul i en aquest cas, serà seu central).

PARÀMETRES D'ENTRADA

  • banc: Codi de 4 caràcters del banc.
  • pais: Codi de 2 caràcters per al país (segons la ISO 3166-1-alfa-2).
  • localitat: Codi de 2 caràcters per a la localitat.
  • sucursal: Codi de 3 caràcters de la sucursal.

PARÀMETRES DE SORTIDA

  • boolean: true si banc+pais+localitat+sucursal formen un codi Swift vàlid i false si banc+pais+localitat+sucursal no formen un número Swift vàlid.

EXEMPLES

Exemple 1 Validar els següents valors: banco: 'BBVA', pais: 'ESP', localidad: '88', sucursal: 'XXX'.

var valido = PLUGIN_UTILS.esNumeroSwiftValido('BBVA', 'ESP', '88', 'XXX');

La funció esNumeroSwiftValido retorna 'false' ja que els paràmetres introduïts no formen un codi SWIFT vàlid, el codi de país ha de tenir longitud 2.


Exemple 2 Validar els següents valors: banco: 'BBVA', pais: 'ES', localidad: '88', sucursal: 'XXX'.

var valido = PLUGIN_UTILS.esNumeroSwiftValido ('BBVA', 'ESP', '88', 'XXX');

La funció esNumeroSwiftValido retorna 'true', ja que els paràmetres introduïts formen un codi SWIFT vàlid.
