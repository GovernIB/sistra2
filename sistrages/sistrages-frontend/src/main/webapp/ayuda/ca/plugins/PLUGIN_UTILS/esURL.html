esURL(String url): boolean

DESCRIPCIÓ

  • Valida que el valor introduït no sigui nul i compleixi amb les regles de validació necessàries per a ser una URL vàlida. Es considera vàlida tota aquella cadena que compleixi amb les regles de validació corresponents a una URL.

PARÀMETRES D'ENTRADA

  • url: Cadena que representa la URL (adreça d'internet).

PARÀMETRES DE SORTIDA

  • boolean: true si la cadena representa una URL vàlida i false si la cadena no representa una URL vàlida.

EXEMPLES

Exemple 1 Validar la cadena 'http://www.caib.es/'

var valida = PLUGIN_UTILS.esURL('http://www.caib.es/');

La funció esUrl retorna 'true', pel fet que la cadena representa una URL vàlida.


Exemple 2 Validar la cadena 'http://www.caib.sedeelectronica/'

var valida = PLUGIN_UTILS.esURL('http://www.caib.sedeelectronica');

La funció esUrl retorna 'false', pel fet que la cadena no representa una URL vàlida.
