setNombre(String nom):void

DESCRIPCIÓ

  • Mètode que permet establir el nom amb extensió d'un fitxer.

PARÀMETRES D'ENTRADA

  • nom: String amb el nom i extensió a establir pel fitxer.

EXEMPLES

Posam per cas que es vol renombrar un fitxer retornat per un domini 'DOMINIO'. Les passes a efectuar són les següents:

Primer es recupera el fitxer mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fitxer = valors.getFichero('PDF');

i assignam el nou nom al fitxer de la manera següent:

fitxer.setNombre('nou_nom.pdf');
