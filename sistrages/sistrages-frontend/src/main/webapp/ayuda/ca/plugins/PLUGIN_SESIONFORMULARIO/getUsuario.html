getUsuario(): DATOS_USUARIO

DESCRIPCIÓ

  • Mètode que retorna l'objecte 'DATOS_USUARIO' amb la informació de l'usuari.
  • Només és possible el seu ús en tràmits autenticats.

PARÀMETRES DE SORTIDA

  • DATOS_USUARIO: Retorna l'objecte 'DATOS_USUARIO' amb la informació de l'usuari.

Exemple

Suposem que es desitja obtenir el nom de l'usuari de la sessió actual (només per a tràmits autenticats)

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado(); if (autenticado == true) { var usuario = PLUGIN_SESIONFORMULARIO.getUsuario(); }

La funció getUsuario retorna l'objecte 'DADES_USUARI', a partir del qual es pot extreure la informació que ens interessa, en aquest cas el nom, mitjançant la següent funció:

var nombre = usuario.getNombre();
