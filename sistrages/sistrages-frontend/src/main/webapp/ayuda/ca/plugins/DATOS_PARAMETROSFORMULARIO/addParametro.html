addParametro(String codigo, String valor): void

DESCRIPCIÓ

  • Mètode que permet passar un nou paràmetre a un formulari.

PARÀMETRES D'ENTRADA

  • codigo: String amb el codi del paràmetre a afegir al formulari.
  • valor: String amb el valor del paràmetre a afegir al formulari.

EXEMPLES

Suposam per cas que volem passar un paràmetre província (prov) a un formulari, indicant com a valor el codi 07 pertanyent a les Balears. Per a això inclouríem la següent línia de codi:

var params = DATOS_PARAMETROSFORMULARIO.addParametro('prov','07');
