setExpone(String textoExpone): void

DESCRIPCIÓ

  • Mètode que permet establir un text d'exposició dels fets. Aquest valor únicament té sentit per sol·licituds genèriques.

PARÀMETRES D'ENTRADA

  • textoExpone: cadena de text amb els fet a exposar.

EXEMPLES

Posam per cas que estam desenvolupant una sol·licitud genèrica i volem enviar a registre el text d'exposició. Per fer-ho podriem referenciar un camp (p.ex. EXPONE) del formulari (p.ej. FORM1) al que el ciutadà ha emplenat el text d'exposició dels fets:

var textoExpone = PLUGIN_FORMULARIOS.getValor('FORM1','EXPONE'); DATOS_REGISTRO.setNumeroExpediente(textoExpone);
