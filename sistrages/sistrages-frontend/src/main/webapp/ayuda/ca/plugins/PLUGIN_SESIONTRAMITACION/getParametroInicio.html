getParametroInicio(String idParametro): String

DESCRIPCIÓ

  • Mètode que permet recuperar el valor dels paràmetres enviats a través de la URL en iniciar el tràmit (o establerts dinàmicament mitjançant l'script de paràmetres inicials).
  • Per incloure paràmetres a la URL d'inici del tràmit els paràmetres s'han d'establir amb el següent format: param-_-valorParam (P.e. mitjançant la URL d'inici https://servidor/sistramitfront/asistente/iniciarTramite.html?tramite=TRAM1&version=1&idioma=es&idTramiteCatalogo=123&parametros=param1-_-valor1-_-param2-_-valor2 es passen els paràmetres param1 i param2).
  • Si el paràmetre no existeix, retorna una cadena buida.

PARÀMETRES D'ENTRADA

  • idParametro: Cadena de l'identificador del paràmetre d'entrada.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters amb el valor del paràmetre.

Exemple

Si la URL d'accés és com segueix: https://servidor/sistramitfront/asistente/iniciarTramite.html?tramite=TRAM1&version=1&idioma=es&idTramiteCatalogo=123&parametros=expediente-_-EXP1234

var valor = PLUGIN_SESIONTRAMITACION.getParametroInicio('tasa');

La funció getParametroInicio retornarà la cadena 'EXP1234', i podrem utilitzar aquest valor per recuperar dades d'aquest expedient.
