esHora(String hora, String formatoHora): boolean

DESCRIPCIÓ

  • Valida que l'hora introduïda no sigui nul·la ni buida o no tingui un format incorrecte.

PARÀMETRES D'ENTRADA

  • hora: Hora a validar.
  • formatoHora: Format de l'hora. És opcional i el format per defecte és HH:mm.

PARÀMETRES DE SORTIDA

  • boolean: true si l'hora és vàlida i false si l'hora no és vàlida.

EXEMPLES

Exemple 1 Validar una cadena de caràcters buida ''

var valido = PLUGIN_UTILS.esHora('', null);

La funció esHora retorna 'false', pel fet que el valor indicat no és una hora vàlida.


Exemple 2 Validar una cadena de caràcters nul·la.

var hora;
var valido = PLUGIN_UTILS.esHora(hora, null);

La funció esHora retorna 'false', pel fet que el valor indicat no és una hora vàlida.


Exemple 3 Validar la cadena de caràcters '10:06'.

var valido = PLUGIN_UTILS.esHora('10-06', 'HH:mm');

La funció esHora retorna 'false', pel fet que el valor indicat no és una hora vàlida, no segueix el format HH:mm.


Exemple 4 Validar la cadena de caràcters '10:06:42'.

var valido = PLUGIN_UTILS.esHora('10:06:42', 'HH:mm:ss');

La funció esHora retorna 'true', pel fet que el valor indicat és una hora vàlida amb el format HH:mm:ss


Exemple 5 Validar la cadena de caràcters '10:06'.

var valido = PLUGIN_UTILS.esHora('10:06', null);

La funció esHora retorna 'true', pel fet que el valor indicat és una hora vàlida.
