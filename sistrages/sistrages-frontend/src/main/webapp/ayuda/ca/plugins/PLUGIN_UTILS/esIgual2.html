esIgual(String cadena1, String cadena2, boolean ignoreCase): boolean

DESCRIPCIÓ

  • Valida si les dues cadenes introduïdes són iguals. Es consideraran iguals totes aquelles cadenes que coincideixin en tots els seus caràcters, o si ambdues són nul·les. Mitjançant el paràmetre ignoreCase es configura la funció perquè no faci distinció entre majúscules i minúscules.

PARÀMETRES D'ENTRADA

  • cadena1: Primera cadena per a comparar.
  • cadena2: Segona cadena per a comparar.
  • ignoreCase: Si és true no fa distinció entre majúscules i minúscules. Si és false fa distinció entre majúscules i minúscules.

PARÀMETRES DE SORTIDA

  • boolean : true si les cadenes són iguals i false si les cadenes no són iguals.

EXEMPLES

Exemple 1 Validar si les cadenes 'abc' i 'ABC' són iguals. S'indica que no es faci distinció entre majúscules i minúscules

var iguales = PLUGIN_UTILS.esIgual ('abc', 'ABC', true);

La funció esIgual retorna 'true', pel fet que el valor assignat per a cada cadena és coincident.


Exemple 2 Validar si les cadenes 'abc' i 'ABC' són iguals. S'indica que es faci distinció entre majúscules i minúscules

var iguales = PLUGIN_UTILS.esIgual ('abc', 'ABC', false);

La funció esIgual retorna 'false', pel fet que el valor assignat per a cada cadena NO és coincident.
