obtenerMes(String fecha, String formatoFecha): int

DESCRIPCIÓ

  • Mètode que retorna el mes de la data introduïda.

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.

PARÀMETRES DE SORTIDA

  • int: Mes de la data introduïda.
  • formatoFecha: Format de la data. És opcional i el format per defecte és yyyy-MM-dd.

EXEMPLES

Exemple 1 Recuperar el mes de la data 2019-06-22 (sense format especificat).

var anyo = PLUGIN_UTILS.obtenerMes('2019-06-22', null);

La funció obtenerMes retorna 6.


Exemple 2 Recuperar el mes de la data 22/06/2019 (amb el format dd/MM/yyyy).

var anyo = PLUGIN_UTILS.obtenerMes('22/06/2019', 'dd/MM/yyyy');

La funció obtenerMes retorna 6.
