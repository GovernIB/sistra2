formateaIntACadena(int numero, String separadorMiles): String

DESCRIPCIÓ

  • Mètode per a convertir un número de tipus int en un string

PARÀMETRES D'ENTRADA

  • numero: Número int a convertir en String.
  • separadorMiles:Caràcter amb el qual s'identifica el separador de milers.

PARÀMETRES DE SORTIDA

  • String:Cadena de caràcters que representa el número introduït.

EXEMPLES

Exemple 1 Convertir el número 123110544434 en una cadena de caràcters

var pCadena = PLUGIN_UTILS.formateaIntACadena(123110544434, '.');

La funció formateaIntACadena retorna la cadena '123.110.544.434'.
