sumaDias(String fecha, int dias, String formatoFecha): String

DESCRIPCIÓ

  • Mètode que retorna la data resultant de sumar el nombre de dies indicat en el paràmetre 'dies' a la data indicada en el paràmetre 'data'.

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.
  • dias: Nombre de dies a sumar.
  • formatoFecha: Format de la data. És opcional i el format per defecte és yyyy-MM-dd.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters que representa la data amb els dies sumats.

EXEMPLES

Sumar 3 dies a la data 23/04/2012.

var fecha = PLUGIN_UTILS.sumaDias('23/04/2012', 3, 'dd/MM/yyyy');

La funció sumaDias retorna la cadena '26/04/2012'.
