getRepresentanteUsuario(): : DATOS_USUARIO

DESCRIPCIÓ

  • Mètode que retorna l'objecte 'DATOS_USUARIO' amb la informació del representant de l'usuari autenticat (aquesta informació la retorna el sistema de connexió).
  • Només és possible el seu ús en tràmits autenticats.

PARÀMETRES DE SORTIDA

  • DATOS_USUARIO: Retorna l'objecte 'DATOS_USUARIO' amb la informació de l'representant de l'usuari autenticat.

EJEMPLO

Suposem que volem obtenir el nom del representant l'usuari de la sessió actual (només per a tràmits autenticats)

var autenticado = PLUGIN_SESIONTRAMITACION.isAutenticado(); if (autenticado == true) { var usuario = PLUGIN_SESIONTRAMITACION.getRepresentante(); }

La función getRepresentanteUsuario retorna el objeto 'DATOS_USUARIO' a partir del cual extraer la información que nos interesa, por ejemplo, el nombre mediante la siguiente función:

var nombre = usuario.getNombre();
