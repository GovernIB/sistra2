getFichero(id): Fichero

DESCRIPCIÓ

  • Mètode que permet recuperar un fitxer retornat per un domini.

PARÀMETRES D'ENTRADA

  • Id: Cadena de text que representa l'identificador del fitxer a recuperar.

PARÀMETRES DE SORTIDA

  • Fichero: Objecte Fichero que modelitza el fitxer retornat pel domini.

EXEMPLES

Posam per cas que es vol recuperar un document amb identificador 'PDF' retornat pel domini 'DOMINIO'. Les passes a efecutar són les següents:

Primer s'invoca al domini mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

i es recupera el fitxer, a partir del seu identificador, de la manera següent:

var file = valors.getFichero('PDF');
