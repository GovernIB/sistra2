getNombreApellidos(): String

DESCRIPCIÓ

  • Mètode que retorna el nom complet de l'usuari autenticat amb el següent format 'nom cognom1 cognom2'.

PARÀMETRES DE SORTIDA

  • String:Cadena de caràcters amb el nom complet de l'usuari autenticat.

Exemple

Si es desitja obtenir el nom complet de l'usuari de la sessió actual, en aquest cas 'Jose Pérez Rodriguez', primer s'ha de comprovar si la sessió és autenticada, a continuació obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el nom complet mitjançant la funció getNombreApellidos:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nombreCompleto = usuario.getNombreApellidos();
}

La funció getNombreApellidos retornarà la cadena 'Jose Pérez Rodríguez'.
