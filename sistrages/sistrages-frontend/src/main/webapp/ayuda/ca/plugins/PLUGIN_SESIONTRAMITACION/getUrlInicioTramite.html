getUrlInicioTramite(): String

DESCRIPCIÓ

  • Mètode que retorna la URL amb la qual es va iniciar el tràmit.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que representa la URL amb la qual es va iniciar el tràmit.

Exemple

Obtenir la URL amb la qual es va iniciar el tràmit:

var url = PLUGIN_SESIONTRAMITACION.getUrlInicioTramite();

La funció getUrlInicioTramite retorna una URL similar a: 'www.caib.es/sistramitfront/asistente/iniciarTramite.html?tramite=ID_TRAMITE&version=1&idioma=es&idTramiteCatalogo=32'
