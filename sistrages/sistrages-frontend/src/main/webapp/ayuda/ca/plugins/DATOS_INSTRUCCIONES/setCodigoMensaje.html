setCodigoMensaje(String codigo):void

DESCRIPCIÓ

  • Mètode que permet recuperar la informació a partir dun codi, que haurà d'estar prèviament definit.
  • La informació es defineix a la pantalla de l'script, a la pestanya de missatges validació.

PARÀMETRES D'ENTRADA

  • codigo: String amb l'identificador del missatge d'error.

EXEMPLES

Posem per cas que volem referenciar des del script una informació, prèviament definida a la pestanya de missatges de validació, amb l'identificador 'CodFechas'.

La informació es fixaria a nivell de script mitjançant el codi següent:

DATOS_INSTRUCCIONES.setCodigoMensaje('CodFechas');
