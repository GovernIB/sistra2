esNif(String nif): boolean

DESCRIPCIÓ

  • Valida si el valor introduït és no nul, i posseeix el format i validacions necessàries per a ser un NIF vàlid. Es considera vàlid tota aquella cadena que tingui 8 números i un caràcter alfanumèric que compleixi amb la regla de validació.

PARÀMETRES D'ENTRADA

  • nif: Cadena que representa el NIF a validar.

PARÀMETRES DE SORTIDA

  • boolean:true si la cadena és un NIF vàlid i false si la cadena no és un NIF vàlid.

EXEMPLES

Exemple 1 Validar la cadena '79865432D'.

var valido = PLUGIN_UTILS.esNif('79865432D');

La funció esNif retorna 'true', pel fet que la cadena conté un NIF vàlid.


Exemple 2 Validar la cadena '4854391C.

var valido = PLUGIN_UTILS.esNif('4854391C');

La funció esNif retorna 'false', pel fet que el valor de la cadena no es correspon a un nif vàlid.
