getEmail(): String

DESCRIPCIÓ

  • Mètode que retorna l'email de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String: Valor amb l'email de l'usuari autenticat.

Exemple

Si es desitja obtenir l'email de l'usuari de la sessió actual, en aquest cas 'pedro@caib.es', primer s'ha de comprovar si la sessió és autenticada, a continuació, obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el seu email mitjançant la funció getEmail:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var email = usuario.getEmail();
}

La funció getEmail retornarà la cadena 'pedro@caib.es'.
