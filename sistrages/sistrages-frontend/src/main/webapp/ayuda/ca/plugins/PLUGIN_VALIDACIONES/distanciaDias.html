distanciaDias(String fecha1, String fecha2): int

DESCRIPCIÓ

  • Mètode que retorna el nombre de dies que hi ha entre una data i una altra.

PARÀMETRES D'ENTRADA

  • fecha1: Cadena que conté la data inicial, amb el següent format yyyy-MM-dd.
  • fecha2: Cadena que conté la data final, amb el següent format yyyy-MM-dd.

PARÀMETRES DE SORTIDA

  • int: Nombre de dies que hi ha de distància entre una data i una altra.

EXEMPLES

Si volem esbrinar el nombre de dies que hi ha entre 2 dates (s'han d'especificar usant el següent format yyyy-MM-dd), per exemple, entre el 23/04/2012 i 13/06/2012 podem usar la següent sentència:

var dias = PLUGIN_VALIDACIONES.distanciaDias('2012-04-23', '2012-06-13');
