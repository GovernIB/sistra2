isError():boolean

DESCRIPCIÓ

  • Mètodo que permet comprovar si la invocació d'un domini retorna o no un error.
  • Aquest mètode està disponible des del mètode PLUGIN_DOMINIO.invocarDominio (accesible des de tots els scripts de versió, passa i formulari).

PARÀMETRES DE SORTIDA

  • boolean: Retorna vertader (true) si la invocació del domini retorna un error, i fals (false) si el domini es resol correctament.

EXEMPLES

Posam per cas que es vol comprovar si la invocació del domini 'DOMINIO' retorna un error. Les passes a efectuar serien les següents:

Primer s'invoca el domini mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

i posteriorment es comprova si es retorna un error:

var hayErr = valors.isError();
