getDescripcion():String

DESCRIPCIÓ

  • Mètode que permet recuperar la descripció d'un fitxer.

PARÀMETRES DE SORTIDA

  • String: String amb la descripció del fitxer.

EXEMPLES

Posam per cas que es vol recuperar la descripció d'un fitxer retornat per un domini 'DOMINIO'. Les passes a efectuar són les següents:

Primer es recupera el fitxer mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');
var fitxer = valors.getFichero('PDF');

i es recupera la descripció de la manera següent:

var descripcio = fitxer.getDescripcion();
