esTelefono(String telefono): boolean

DESCRIPCIÓ

  • Valida si el valor introduït compleix amb les regles de validació necessàries per a ser un número de telèfon fix o mòbil vàlid.
  • Es considera vàlida tota aquella cadena no nul·la que tingui 9 números i que comenci per 6, 7 o 9.

PARÀMETRES D'ENTRADA

  • telefono: Cadena que representa el número de telèfon.

PARÀMETRES DE SORTIDA

  • boolean: true si la cadena representa un telèfon vàlid i false si la cadena no representa un telèfon vàlid

EXEMPLES

Exemple 1 Validar la cadena '9711714'.

var valido = PLUGIN_VALIDACIONES.esTelefono('9711714');

La funció esTelefono retorna 'false', pel fet que el número de telèfon ha de tenir com a mínim 9 dígits.


Exemple 2 Validar la cadena '971177714'.

var valido = PLUGIN_VALIDACIONES.esTelefono('971177714');

La funció esTelefono retorna 'true', pel fet que el telèfon es correspon amb el format d'un número de telèfon fix.


Exemple 3 Validar la cadena '663467284'.

var valido = PLUGIN_VALIDACIONES.esTelefono('663467284');

La funció esTelefono retorna 'true', pel fet que el telèfon es correspon amb el format d'un número de telèfon mòbil.
