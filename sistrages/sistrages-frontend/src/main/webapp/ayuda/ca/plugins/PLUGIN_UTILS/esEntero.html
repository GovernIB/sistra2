esEntero(String numero): boolean

DESCRIPCIÓ

  • Verifica que el nombre introduït no sigui nul, ni buit i que sigui un nombre enter.

PARÀMETRES D'ENTRADA

  • numero: Cadena de caràcters que conté el nombre enter a validar.

PARÀMETRES DE SORTIDA

  • boolean : True si el número és un nombre enter i fals si no és un nombre enter.

EXEMPLES

Exemple 1 Validar una cadena de caràcters buida.

var valido = PLUGIN_UTILS.esEntero('');

La funció esEntero retorna 'false', pel fet que el valor introduït no és un número.


Exemple 2 Validar una cadena de caràcters nula.

var entero; var valido = PLUGIN_UTILS.esEntero(entero);

La funció esEntero retorna 'false', pel fet que el valor introduït es nul.


Exemple 3 Validar la cadena de caràcters '3A'.

var valido = PLUGIN_UTILS.esEntero('3A');

La funció esEntero retorna 'false', pel fet que el valor introduït no és un nombre enter.


Exemple 4 Validar la cadena de caràcters '12,45'.

var valido = PLUGIN_UTILS.esEntero('12,45');

La funció esEntero retorna 'false', pel fet que el valor introduït no és un nombre enter.


Exemple 5 Validar la cadena de caràcters '12'.

var valido = PLUGIN_UTILS.esEntero('12');

La funció esEntero retorna 'true', pel fet que el valor introduït és un nombre enter.


Exemple 6 Validar la cadena de caràcters '-23'.

var valido = PLUGIN_UTILS.esEntero('-23');

La funció esEntero retorna 'true', pel fet que el valor introduït és un nombre enter.
