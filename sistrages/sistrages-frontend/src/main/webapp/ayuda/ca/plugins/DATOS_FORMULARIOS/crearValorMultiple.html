crearValorMultiple(): ValorCampoMultiple

DESCRIPCIÓ

  • Mètode per a crear un valor múltiple per a després poder utilitzar-lo en un camp del formulari actual.

PARÀMETRES DE SORTIDA

  • ValorCampoMultiple: Crea el valor del camp múltiple.

EXEMPLES

Per exemple, si desitjam generar un conjunt de dades geogràfiques predefenides, primer crearíem la variable:

var datos = DATOS_FORMULARIOS.crearValorMultiple();

i després afegiríem les dades a la variable, essent el primer paràmetre d'entrada el codi i el segon paràmetre el valor:

datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');
