validaFechaActual(String fecha, String formatoFecha): int

DESCRIPCIÓ

  • Mètode per a comprovar si la data indicada és la data actual. La data ha de tenir el format dd/MM/yyyy.

PARÀMETRES D'ENTRADA

  • fecha: Cadena de caràcters a comparar amb la data actual.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • int: Nombre sencer, els possibles valors a retornar són:
    1. 0: Si data és igual a la data actual.
    2. 1: Si data és superior a la data actual.
    3. -1: Si data és menor a la data actual.

EXEMPLES

Exemple 1 Validar si la cadena de caràcters '09/01/2019' (sense format especificat) correspon amb la data actual (suposant que avui és 9 de gener de 2019).

var valida = PLUGIN_UTILS.validaFechaActual('09/01/2019');

La funció validaFechaActual retorna 0, ja que la data indicada correspon amb la data actual.


Exemple 2 Validar si la cadena de caràcters '2019-01-09' (con el format 'yyyy-MM-dd') correspon amb la data actual (suposant que avui és 9 de gener de 2019).

var valida = PLUGIN_UTILS.validaFechaActual('2019-01-09', 'yyyy-MM-dd');

La funció validaFechaActual retorna -1, ja que la data indicada és menor que la data actual.


Exemple 3 Validar si la cadena de caràcters '2019-01-12' (amb el formate yyyy-MM-dd) correspon amb la data actual (suposant que avui és 9 de gener de 2019).

var valida = PLUGIN_UTILS.validaFechaActual('2019-01-12', 'yyyy-MM-dd');

La funció validaFechaActual retorna 1, ja que la data indicada és major que la data actual.
