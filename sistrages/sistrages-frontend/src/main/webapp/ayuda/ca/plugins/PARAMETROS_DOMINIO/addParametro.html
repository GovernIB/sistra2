addParametro(String codi, String parametre): void

DESCRIPCIÓ

  • Mètode que permet afegir paràmetres requerits per la invocació d'un domini.

PARÀMETRES D'ENTRADA

  • codi: String amb el codi de paràmetre a afegir.
  • parametre: String amb el valor del paràmetre a afegir.

EXEMPLES

Posem per cas que volem invocar un domini amb identificador 'MUNICIPIS' que requereix passar per paràmetre la província, el codi de paràmetre del qual és 'prov' i al que assignarem el valor '07', que és el codi INE d'Illes Balears. Per incorporar aquest paràmetre hauriem de procedir de la manera següent:

var params = PARAMETROS_DOMINIO.addParametro('prov','07');
