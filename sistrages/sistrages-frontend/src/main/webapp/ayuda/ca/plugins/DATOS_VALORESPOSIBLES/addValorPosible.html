addValorPosible(String codigo, String descripcion): void

DESCRIPCIÓ

  • Mètode que permet afegir un nou valor al camp actual.

PARÀMETRES D'ENTRADA

  • codigo: Cadena de text que representa el codi de l'element
  • descripcion: Cadena de text que representa la descripció de l'element.

EXEMPLES

Afegir valors possibles al camp actual, per exemple, si el camp actual és un selector de municipis:

DATOS_VALORESPOSIBLES.addValorPosible('46','Provincia de Valencia'); DATOS_VALORESPOSIBLES.addValorPosible('03','Provincia de Alicante');

D'aquesta manera aquests valors quedaran afegits al camp selector.
