getNombreColumnas(): List<String>

DESCRIPCIÓ

  • Mètode que permet obtenir una llista amb el nom de les columnes que retorna un domini.

PARÀMETRES DE SORTIDA

  • List<String>: Retorna una llista d'objectes String que representen els noms de les columnes que retorna el domini.

EXEMPLES

Posam per cas que es vol recuperar el nom de les columnes que retorna el domini 'DOMINIO'. Les passes a efecutar són les següents:

Primer s'invoca el domini mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

i es recupera la llista de noms de les columnes de la manera següent:

var listColumns = valores.getNombreColumnas();

Una vegada obtinguda la llista, es pot recuperar el nom de cada columna com s'indica a continuació:

for (i=0; i<listColumns.size(); i++) {
var nomcol = listColumns.get(i);
....
}
