validaRango(int valor, int minimo, int maximo): boolean

DESCRIPCIÓ

  • Compara si el paràmetre 'valor' està dins d'un rang determinat.

PARÀMETRES D'ENTRADA

  • valor: Valor a validar dins del rang.
  • minimo: Indica el límit mínim del rang.
  • maximo: Indica el límit màxim del rang.

PARÀMETRES DE SORTIDA

  • boolean: true si se encuentra dentro del rango y false si no es así.

EXEMPLES

Validar si la cadena '25689' es troba dins dels límits mínim i màxim.

var valido = PLUGIN_VALIDACIONES.validaRango(25689, 16498, 31648);

La funció validaRango retorna 'true' ja que el valor indicat es troba dins del rang.
