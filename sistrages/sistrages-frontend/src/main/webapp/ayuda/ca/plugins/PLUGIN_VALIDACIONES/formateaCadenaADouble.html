formateaCadenaADouble(String numero, String separadorMiles, String separadorDecimales): double

DESCRIPCIÓ

  • Mètode per a convertir un string en un número double.

PARÀMETRES D'ENTRADA

  • numero: Cadena a convertir a doble.
  • separadorMiles: Caràcter amb el qual s'identifica el separador de milers.
  • separadorDecimales: Caràcter amb el qual s'identifica el separador de decimals. Dada obligatòria.

PARÀMETRES DE SORTIDA

  • double: Número en format double. Si no es pot convertir, llança un error.

EXEMPLES

Exemple 1 Convertir la cadena '123.110.544.434,58' a double.

var nDoble = PLUGIN_VALIDACIONES.formateaCadenaADouble('123.110.544.434,58', '.', ',')

La funció formateaCadenaADouble retorna el número 123110544434.58 en format double


Exemple 2 Convertir la cadena '123,110,544,434.58' a double.

var nDoble = PLUGIN_VALIDACIONES.formateaCadenaADouble('123,110,544,434.58', ',', '.')

La funció formateaCadenaADouble retorna el número 123110544434.58 en format double.
