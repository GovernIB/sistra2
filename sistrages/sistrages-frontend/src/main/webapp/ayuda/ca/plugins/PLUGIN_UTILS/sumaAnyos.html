sumaAnyos(String fecha, int anyos, String formatoFecha): String

DESCRIPCIÓ

  • Mètode que retorna la data resultant de sumar el nombre de anys indicat en el paràmetre 'anyos' a la data indicada en el paràmetre 'data'.

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.
  • anyos: Nombre de anys a sumar.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters que representa la data amb els anys sumats.

EXEMPLES

Sumar 3 anys a la data 23/04/2012.

var fecha = PLUGIN_UTILS.sumaAnyos('23/04/2012', 3, 'dd/MM/yyyy');

La funció sumaAnys retorna la cadena '23/04/2015'.
