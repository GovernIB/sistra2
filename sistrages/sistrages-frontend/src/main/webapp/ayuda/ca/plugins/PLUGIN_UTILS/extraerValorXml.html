extraerValorXml(String xml, String xpath): String

DESCRIPCIÓ

  • Extreu el valor del node XML representat pel xpath indicat.
  • El XML ha d'estar ben format, i el xpath ha de contenir una ruta vàlida a un node del XML

PARÀMETRES D'ENTRADA

  • xml: Cadena amb estructura de XML. Ha de contenir un XML ben format
  • xpath:Cadena que conté el xpath. Conté una seqüència de tags del XML, que marquen la ruta cap al node del qual volem extreure el valor.

PARÀMETRES DE SORTIDA

  • String: Valor del node indicat en el paràmetre xpath. Si la ruta indicada en el xpath no existeix retorna un error.

EXEMPLES

Exemple 1 Extreure el valor del node 'dni' en el XML. Per a això, hem de revisar l'estructura completa del XML, i obtenir el xpath adequat. En aquest cas el xpath serà'/xml/persona/dni'

var xml = '<xml><persona><nombre>juan</nombre><dni>79865432D</dni></persona></xml>';
var dni = PLUGIN_UTILS.extraerValorXml(xml, '/xml/persona/dni');

La funció extraerValorXml retorna '79865432D', que és el valor del node dni.


Exemple 2 Pot ocórrer que el node del qual intentem obtenir el valor no existeixi en el XML. Per exemple, si intentem usar el següent xpath '/xml/persona/apellido', fent servir el mateix XML de l'anterior exemple:

var xml = '<xml><persona><nombre>juan</nombre><dni>79865432D</dni></persona></xml>';
var persona = PLUGIN_UTILS.extraerValorXml(xml, '/xml/persona/apellido');

La funció extraerValorXml retorna un error, ja que '/xml/persona/apellido' no existeix com a node en el XML facilitat.
