esCif(String cif): boolean

DESCRIPCIÓ

  • Verifica si el cif introduït és un cif legalment vàlid. Es considerarà CIF vàlid tota aquella cadena que compleixi amb el format de nombre de caràcters correcte i compleixi amb les regles respectives a la correspondència alfanumèrica.

PARÀMETRES D'ENTRADA

  • cif: Valor del Cif.

PARÀMETRES DE SORTIDA

  • boolean: true si el cif és vàlid i false si el cif no és vàlid.

EXEMPLES

Exemple 1 Es verifica una cadena buida ''.

var valido = PLUGIN_VALIDACIONES.esCif('');

La funció esCif retorna 'false', ja que el valor passat com cif no es correspon amb el format d'un cif.


Exemple 2 Es verifica una cadena amb valor nul.

var cif;
var valido = PLUGIN_VALIDACIONES.esCif(cif);

La funció esCif retorna 'false', ja que el valor passat com cif, és nul i per tant no es correspon amb el format d'un cif.


Exemple 3 Es verifica el valor 'C23568945'.

var valido = PLUGIN_VALIDACIONES.esCif('C23568945');

La funció esCif retorna 'false', ja que el valor passat com a cif no té el format correcte (no té la lletra de control).


Exemple 4 Es verifica el valor 'C6141464E'.

var valido = PLUGIN_VALIDACIONES.esCif('C6141464E');

La funció esCif retorna 'true', ja que el valor passat com cif es correspon amb el format correcte.
