crearRepresentante(): Persona

DESCRIPCIÓ

  • Mètode que permiteix indicar representant. De forma opcional es podrà indicar el representant i de no indicar-se es prendrà el presentador com a representant.

EXEMPLES

Per exemple, suposem desitgem establir el representant:

var rpte = DATOS_REPRESENTACION.crearRepresentante();
rpte.setDatosPersona('1111111H', 'Pedro', 'Pérez', null);
