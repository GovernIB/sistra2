setValorListaElementos(valor): void

DESCRIPCIÓ

  • Mètode que estableix un valor llista elements a un camp del formulari actual.

PARÀMETRES DE SORTIDA

  • valorCampoListaElementos : Objecte de tipus ValorListaElementos el qual conté els diferents valors a assignar al camp.

EXEMPLES

S'inicialitza una variable per a assignar-la a un camp multiregistre del formulari actual i anar afegint valors mitjançant la funció addValor:

var datos = DATOS_VALOR.crearValorListaElementos();
var valorElemento=datos.addElemento();
valorElemento.addValor('codigo', '07073');
valorElemento.addValor('descripcion', 'Mallorca');

Una vegada creada la variable, creat l'element i assignats els valors, es podrà assignar al camp:

DATOS_VALOR.setValorListaElementos(datos);
