DATOS_REPRESENTACION

DESCRIPCIÓ

  • Objecte que permet establir les dades representació. En cas d'existir representació, caldrà establir el representat. De forma opcional es podrà indicar el representant i de no indicar-se es prendrà el presentador com a representant.
  • Si no s'estableix l'script i s'accedeix amb certificat de representació, s'establirà per defecte aquesta representació.
