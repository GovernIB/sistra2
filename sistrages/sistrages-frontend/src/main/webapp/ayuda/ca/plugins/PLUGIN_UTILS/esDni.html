esDni(String dni): boolean

DESCRIPCIÓ

  • Valida si el valor introduït posseeix el format per a ser un DNI vàlid. Es considera vàlid tota aquella cadena que tingui 8 números i un caràcter alfanumèric que compleixi amb la regla de validació.

PARÀMETRES D'ENTRADA

  • dni: Cadena que representa el DNI a validar.

PARÀMETRES DE SORTIDA

  • boolean:true si la cadena és un DNI vàlid i false si la cadena no és un DNI vàlid.

EXEMPLES

Exemple 1 Validar la cadena '79865432D'.

var valido = PLUGIN_UTILS.esDni('79865432D');

La funció esDni retorna 'true' pel fet que la cadena conté un DNI vàlid.


Exemple 2 Validar la cadena '4854391C.

var valido = PLUGIN_UTILS.esDni('4854391C');

La funció esDni retorna 'false' ja que el valor de la cadena no es correspon a un dni vàlid.
