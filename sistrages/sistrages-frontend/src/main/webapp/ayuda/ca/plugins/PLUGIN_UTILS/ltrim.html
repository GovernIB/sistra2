ltrim(String cadena): String

DESCRIPCIÓ

  • Mètode que elimina els espais situats a l'esquerra en la cadena.

PARÀMETRES D'ENTRADA

  • cadena: Cadena de text a tractar

PARÀMETRES DE SORTIDA

  • String: Cadena de text sense espais per l'esquerra.

EXEMPLES

Tractar la cadena '  cadena ejemplo    ', per a eliminar els espais de l'esquerra.

var pCadenaLimpia = PLUGIN_UTILS.ltrim('  cadena ejemplo    ');

La funció ltrim retorna 'cadena ejemplo    '.
