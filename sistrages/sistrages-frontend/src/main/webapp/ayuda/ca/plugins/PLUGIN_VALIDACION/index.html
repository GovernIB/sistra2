PLUGIN_VALIDACION

DESCRIPCIÓN

  • Plugin para generar un mensaje de aviso.
  • Este plugin estara disponible en los scripts que implementen validación y servirá para mostrar el resultado de la validación mediante un mensaje de aviso (INFO / WARNING / ERROR). Si se indica que el mensaje es de tipo ERROR no pasará la validación.

MÉTODOS

  • setExisteAviso(existeAviso,tipo)
  • addParametroMensajeAviso(parametro)
  • setCodigoMensajeAviso(codigoMensajeAviso)
  • setTextoMensajeAviso(textoMensajeAviso)
