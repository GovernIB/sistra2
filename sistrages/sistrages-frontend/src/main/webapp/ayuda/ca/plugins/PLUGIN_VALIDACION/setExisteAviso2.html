setExisteAviso(boolean existeAviso, String tipo, String campo)

DESCRIPCIÓ

  • Mètode per establir si es genera avís per emprar-se a l'script de validació de pàgina i poder indicar el camp del formulari al qual aplica la validació.

PARÀMETRES D'ENTRADA

  • existeAviso: booleà quer permet establir si es genera avís.
  • tipo: INFO / WARNING / ERROR
  • campo: Identificador camp formulari

EXEMPLE

Podem indicar que existeix un avís de tipus WARNING al camp txtValor de la següent manera:

PLUGIN_VALIDACION.setExisteAviso(true, 'WARNING', 'txtValor');
