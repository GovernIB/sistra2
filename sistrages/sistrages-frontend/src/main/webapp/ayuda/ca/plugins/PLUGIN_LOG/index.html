PLUGIN_LOG

DESCRIPCIÓN

  • Plugin que permite utilizar el log de los scripts
  • Para que aparezca estos mensajes deberá estar habilitado el check de 'Habilitar debug' en la pantalla de control de acceso de la versión del trámite
  • Estos mensajes son visibles si en la misma sesión del navegador (abriendo otra pestaña en la ventana donde estamos tramitando), accedemos a la siguiente dirección: /debug/logSesionTramitacion.html.
  • Este plugin está disponible para todos los scritps.
