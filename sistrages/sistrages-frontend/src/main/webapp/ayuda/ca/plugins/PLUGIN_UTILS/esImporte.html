esImporte(String importe): boolean

DESCRIPCIÓ

  • Valida si el valor introduït és no nul, numèric positiu i amb entre zero i dos decimals. Ej: '0,00'

PARÀMETRES D'ENTRADA

  • importe: Import a validar.

PARÀMETRES DE SORTIDA

  • boolean: true si l'import és vàlid i false si l'import no és vàlid.

EXEMPLES

Exemple 1 Validar una cadena de caràcters nul·la.

var importe;
var valido = PLUGIN_UTILS.esImporte(importe);

La funció esImporte retorna 'false', pel fet que el valor no és un import vàlid.


Exemple 2 Validar la cadena de caràcters 45,20.

var valido = PLUGIN_UTILS.esImporte('45,20');

La funció esImporte retorna 'true', pel fet que el valor és un import vàlid.


Exemple 3 Validar la cadena de caracteres 98,444.

var valido = PLUGIN_UTILS.esImporte('98,444');

La funció esImporte retorna 'false', pel fet que el valor no està correctament format.
