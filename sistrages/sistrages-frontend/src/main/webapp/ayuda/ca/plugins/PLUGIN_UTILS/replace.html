replace(String texto, String oldChar, String newChar): String

DESCRIPCIÓ

  • Mètode usat per a reemplaçar totes les ocurrències d'una determinada cadena de text per una altra cadena de text.

PARÀMETRES D'ENTRADA

  • texto: Cadena de text.
  • oldChar Cadena a reemplaçar
  • newChar: Cadena de reemplaçament.

PARÀMETRES DE SORTIDA

  • string: cadena de text en la qual totes les ocurrències de oldChar han estat reemplaçades per newchar.

EXEMPLES

Exemple 1 Reemplaçar, en la cadena de text 'aba', la cadena 'a' per la cadena 'z'.

var cadenaFinal = PLUGIN_UTILS.replace('aba', 'a', 'z');

La funció replace retorna la cadena'zbz'.


Exemple 2Reemplaçar, en la cadena de text 'adbad', la cadena 'ad' per la cadena 'mnm'.

var cadenaFinal = PLUGIN_UTILS.replace('adbad', 'ad', 'mnm');

La funció replace retorna la cadena 'mnmbmnm'.
