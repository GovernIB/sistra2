esNifPersonaFisica(String nif): boolean

DESCRIPCIÓ

  • Valida si el valor introduït posseeix el format requerit per a ser un NIF de persona física vàlida (DNI, NIE o NIF Altres).

PARÀMETRES D'ENTRADA

  • nif: Cadena que representa el NIF de persona física a validar.

PARÀMETRES DE SORTIDA

  • boolean:true si la cadena és un NIF de persona física vàlida i false si la cadena no és un NIF de persona física vàlida.

EXEMPLES

Exemple 1 Validar la cadena 'B88018098'.

var valido = PLUGIN_UTILS.esNifPersonaFisica('B88018098');

La funció esNifPersonaFisica retorna 'true', pel fet que la cadena conté un NIF de persona física vàlida.


Exemple 2 Validar la cadena 'A76381906.

var valido = PLUGIN_UTILS.esNifPersonaFisica('A76381906');

La funció esNifPersonaFisica retorna 'false', pel fet que el valor de la cadena no es correspon a un NIF de persona física vàlida.
