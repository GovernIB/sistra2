distanciaHoras(String fecha1, String hora1, String fecha2, String hora2, String formatoFecha, String formatoHora): String

DESCRIPCIÓ

  • Mètode que retorna el nombre d'hores, minuts i segons que hi ha entre dues dates i/o dues hores.

PARÀMETRES D'ENTRADA

  • fecha1: Cadena que conté la data inicial.
  • hora1: Cadena que conté l'hora inicial.
  • fecha2: Cadena que conté la data final.
  • hora2: Cadena que conté l'hora final.
  • formatoFecha: Format de les dates. És opcional i el format per defecte és dd/MM/yyyy.
  • formatoFecha: Format de les hores. És opcional i el format per defecte és HH:mm.

PARÀMETRES DE SORTIDA

  • String: Nombre d'hores, minuts i segons que hi ha de distància entre les dates i/o les hores, amb el format HH:mm:ss.

EXEMPLES

Exemple 1 Si volem esbrinar el nombre d'hores, minuts i segons que hi ha entre 2 dates (s'han d'especificar usant el següent format yyyy-MM-dd i HH:mm:ss), per exemple, entre el 23/04/2012 i 13/06/2012 a les 2 de la matinada, 2 minuts i 2 segons, podem usar la següent sentència, de manera que s'obtindrà 1226:02:02:

var horas = PLUGIN_UTILS.distanciaHoras('2012-04-23', '', '2012-06-13', '02:02:02', 'yyyy-MM-dd', 'HH:mm:ss');

Exemple 2 Si volem esbrinar el nombre d'hores, minuts i segons que hi ha entre 2 dates (especificant el format dd/MM/yyyy i HH:mm), per exemple, entre el 23/04/2012 i 13/06/2012 podem usar la següent sentència, obtenint 1224:00:00:

var horas = PLUGIN_UTILS.distanciaDias('23/04/2012', '', '13/06/2012', '', 'dd/MM/yyyy', '');

Exemple 3 Si volem esbrinar el nombre d'hores, minuts i segons que hi ha entre 2 hores (especificant el format HH:mm:ss), per exemple, entre les 19:20:05 i 23:58:42 podem usar la següent sentència, obtenint 04:38:37:

var horas = PLUGIN_UTILS.distanciaDias('', '19:20:05', '', '23:58:42', '', 'HH:mm:ss');
