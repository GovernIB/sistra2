formateaNombreApellidos(String formato, String nombre, String apellido1, String apellido2): String

DESCRIPCIÓ

  • Mètode que permet combinar el nom i els cognoms en funció del paràmetre format.

PARÀMETRES D'ENTRADA

  • formato: Cadena de caràcters que indica el format a usar. Els valors possibles són:
    • AN : cognom1 cognom2, nom
    • NA : nom cognom1 cognom2
  • nombre: Nom complet.
  • apellido1: Primer cognom.
  • apellido2: Segon cognom.

PARÀMETRES DE SORTIDA

  • String: Cadena amb el nom i els cognoms combinats segons el paràmetre format.

EXEMPLES

Exemple 1 Suposem que es desitja donar format (primer_cognom segon_cognom, nom) a les següents dades: Nom:Pedro, Cognom1: Roses Cognom2: Fuentes.

var pCadena = PLUGIN_UTILS.formateaNombreApellidos('AN', 'Pedro', 'Rosas', 'Fuentes');

La funció formateaNombreApellidos retorna la següent cadena de caràcters 'Roses Fuentes, Pedro'.


Exemple 2Suposem que es desitja donar format (nom primer_cognom segon_cognom) a les següents dades: Nom:Sonia, Cognom1: Arizmendi, Cognom2: Cardenas.

var pCadena = PLUGIN_UTILS.formateaNombreApellidos('NA', 'Sonia', 'Arizmendi', 'Cardenas');

La funció formateaNombreApellidos retorna la següent cadena de caràcters 'Sonia Arizmendi Cardenas'.
