validaFechaHoraActual(String fecha, String formatoFecha): int

DESCRIPCIÓ

  • Mètode que permet comparar, la data indicada amb la data actual.

PARÀMETRES D'ENTRADA

  • fecha: Cadena de text que representa la data a comparar amb la data actual.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy HH:mm:ss.

PARÀMETRES DE SORTIDA

  • int: Nombre enter, els possibles valors a retornar són:
    1. 0: Si la data és igual a la data/hora actual.
    2. 1: Si la data és superior a la data/hora actual.
    3. -1: Si la data és menor a la data/hora actual.

EXEMPLES

Exemple 1 Comparar la cadena de text '2019-01-09 09.53:03' (sense format especificat) amb la data/hora actual (suposant que avui és 9 de gener de 2019 09.53:03).

var valida = PLUGIN_UTILS.validaFechaHoraActual('2019-01-09 09:53:03', null);

La funció validaFechaHoraActual retorna 0, ja que la data indicada es correspon amb la data/hora actual.


Exemple 2 Comparar la cadena de text '02/01/2019 07:43:03' (amb el format especificat 'dd/MM/yyyy HH:mm:ss') amb la data/hora actual (suposant que avui és 9 de gener de 2019 09.53:03).

var valida = PLUGIN_UTILS.validaFechaHoraActual('02/01/2019 07:43:03','dd/MM/yyyy HH:mm:ss');

La funció validaFechaHoraActual retorna -1, ja que la data indicada és menor que la data/hora actual.


Exemple 3 Comparar la cadena de text '2019-01-12 19:13:03' (sense format especificat) amb la data/hora actual (suposant que avui és 9 de gener de 2019 09.53:03).

var valida = PLUGIN_UTILS.validaFechaHoraActual('2019-01-12 19:13:03', null);

La funció validaFechaHoraActual retorna 1, ja que la data indicada és major que la data/hora actual.
