getApellidos(): String

DESCRIPCIÓ

  • Mètode que retorna els cognoms de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters amb els cognoms de l'usuari autenticat (primer i segon cognom, concatenats amb un espai).

Exemple

Si es desitja obtenir els cognoms de l'usuari de la sessió actual, en aquest cas 'Jose Pérez Rodriguez', primer s'ha de comprovar si la sessió és autenticada, a continuació obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir els cognoms mitjançant la funció getApellidos:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellidos = usuario.getApellidos();
}

La funció getApellidos retornarà la cadena 'Pérez Rodriguez'.
