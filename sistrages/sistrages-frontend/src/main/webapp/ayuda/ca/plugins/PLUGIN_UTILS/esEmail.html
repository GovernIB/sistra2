esEmail(String email): boolean

DESCRIPCIÓ

  • Valida si la direcció d'email introduïda és una direcció vàlida. Es considerarà adreça d'e-mail vàlida tota aquella cadena que compleixi amb el format i les regles que regeixen el format d'un correu electrònic.

PARÀMETRES D'ENTRADA

  • email: Adreça de correu electrònic a validar.

PARÀMETRES DE SORTIDA

  • boolean: true si l'email està correctament format i false si l'email no està correctament format.

EXEMPLES

Exemple 1 Validar la cadena de caràcters ''.

var valido = PLUGIN_UTILS.esEmail('');

La funció esEmail retorna 'false', pel fet que l'email introduït no té un format vàlid.


Exemple 2 Validar una cadena de caràcters nula.

var email;
var valido = PLUGIN_UTILS.esEmail(email);

La funció esEmail retorna 'false', pel fet que l'email introduït no té un format vàlid.


Exemple 3 Validar la cadena de caràcters 'pepe@gmail'

var valido = PLUGIN_UTILS.esEmail('pepe@gmail');

La funció esEmail retorna 'false', pel fet que l'email introduït no té un format vàlid.


Exemple 4 Validar la cadena de caràcters 'pepe@gmail.com'.

var valido = PLUGIN_UTILS.esEmail('pepe@gmail.com');

La funció esEmail retorna 'true', pel fet que l'email introduït té un format vàlid.
