convierteFechaFormatoDefecto(String fechaEntrada, String formatoFechaEntrada): int

DESCRIPCIÓ

  • Mètode que converteix una data d'entrada amb un determinat format a el format per defecte (dd/MM/yyyy).

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.
  • formatoFecha: Format de la data.

PARÀMETRES DE SORTIDA

  • String: Data amb format per defecte.

EXEMPLES

Exemple 1 Recuperar el dia de la data 2019-06-22.

var fecha = PLUGIN_UTILS.convierteFechaFormatoDefecto('2019-06-22', 'yyyy-MM-dd');

La funció retorna '22/06/2019'.
