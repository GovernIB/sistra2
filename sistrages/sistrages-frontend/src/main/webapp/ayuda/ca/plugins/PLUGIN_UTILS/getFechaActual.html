getFechaActual(String formatoFecha): String

DESCRIPCIÓ

  • Mètode que retorna la data actual del sistema. Indicar que si es vol passar a camps tipus 'Data', el format de la data ha de ser el que ve per defecte. D'altra banda, si es vol passar a un camp tipus 'Text', la data pot venir en el format que es desitgi.

PARÀMETRES D'ENTRADA

  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que representa la data actual del sistema, amb el format especificat (per defecte dd/MM/yyyy).

EXEMPLES

Exemple 1 Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema (amb el format per defecte)

var hoy = PLUGIN_UTILS.getFechaActual(null);

La funció getFechaActual retornarà '07/01/2019'.


Exemple 2 Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema (amb el format yyyy-MM-dd)

var hoy = PLUGIN_UTILS.getFechaActual('yyyy-MM-dd');

La funció getFechaActual retornarà '2019-01-07'.


Exemple 3 Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema (amb el format 'MM/dd/yyyy')

var hoy = PLUGIN_UTILS.getFechaActual('MM/dd/yyyy');

La funció getFechaActual retornarà '01/07/2019'.


Exemple 4 Suposem que avui és 7 de gener de 2019 a les 10:53:12, i desitgem recuperar la data actual del sistema (amb el format 'dd/MM/yyyy HH:mm:ss')

var hoy = PLUGIN_UTILS.getFechaActual('dd/MM/yyyy HH:mm:ss');

La funció getFechaActual retornarà '07/01/2019 10:53:12'.


Exemple 5 Aquesta funció també és capaç de retornar només la data amb la precisió que volem. Per exemple, si volem obtenir l'hora i els minuts actuals:

var hoy = PLUGIN_UTILS.getFechaActual('HH:mm');

La funció getFechaActual retornarà l'hora i els minuts actuals.
