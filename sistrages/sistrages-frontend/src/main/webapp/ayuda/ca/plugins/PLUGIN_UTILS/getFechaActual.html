getFechaActual(String formatoFecha): String

DESCRIPCIÓ

  • Mètode que retorna la data actual del sistema.

PARÀMETRES D'ENTRADA

  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que representa la data actual del sistema, amb el format especificat (per defecte dd/MM/yyyy).

EXEMPLES

Exemple 1 Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema (i no passem cap format)

var hoy = PLUGIN_UTILS.getFechaActual(null);

La funció getFechaActual retornarà '2019-01-07'.


Exemple 2 Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema (en el formato 'dd/MM/yyyy')

var hoy = PLUGIN_UTILS.getFechaActual('dd/MM/yyyy');

La funció getFechaActual retornarà '01/07/2019'.
