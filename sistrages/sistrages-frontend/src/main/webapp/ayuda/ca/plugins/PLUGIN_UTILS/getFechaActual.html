getFechaActual(): String

DESCRIPCIÓ

  • Mètode que retorna la data actual del sistema.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que representa la data actual del sistema, amb el format yyyy-MM-dd.

EXEMPLES

Suposem que avui és 7 de gener de 2019, i desitgem recuperar la data actual del sistema

var hoy = PLUGIN_UTILS.getFechaActual();

La funció getFechaActual retornarà '2019-01-07'.
