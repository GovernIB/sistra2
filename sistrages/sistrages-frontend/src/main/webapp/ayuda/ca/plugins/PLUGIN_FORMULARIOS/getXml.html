getXml(String idFormulario): String

DESCRIPCIÓ

  • Mètode que retorna el XML amb els valors del formulari.

PARÀMETRES D'ENTRADA

  • idFormulario: Cadena que conté l'identificador del formulari.

PARÀMETRES DE SORTIDA

  • string:Cadena de caràcters que representa el XML amb els valors del formulari.

Exemple

Obtenir el XML del formulari amb identificador 'Form1':

var xml = PLUGIN_DATOSFORMULARIO.getXml('Form1');

La funció getXml retornarà una cadena de text on el contingut serà el XML del formulari amb els seus valors.
D'aquest XML es podria extreure el valor d'un determinat camp, per Exemple si sabem que en el XML existeix un camp dni amb el següent xpath '/xml/persona/dni' podríem obtenir el valor del camp DNI mitjançant:

var dni = PLUGIN_VALIDACIONES.extraerValorXml(xml, '/xml/persona/dni');

La funció extraerValorXml retornarà el valor del camp dni dins del XML, en aquest cas '11111111H'.
