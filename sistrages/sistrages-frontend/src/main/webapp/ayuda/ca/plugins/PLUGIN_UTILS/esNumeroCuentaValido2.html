esNumeroCuentaValido(String entidad, String sucursal, String dc, String cuenta): boolean

DESCRIPCIÓ

  • Valida si els valors introduïts no són nuls, i compleix amb les regles de validació necessàries per a ser un NCC (Número Compte Corrent) Vàlid. Es consideren vàlides tota aquella cadena que tinguin 4 números per a l'entitat, 4 números per a la sucursal, 2 números per als dígits de control i 10 números per al compte corrent i que compleixin amb les regles de validació corresponents a la combinació d'entitat, oficina, dígit de control i número de compte.

PARÀMETRES D'ENTRADA

  • entidad: Codi de 4 dígits de l'entitat bancària.
  • sucursal: Codi de 4 dígits del Número de sucursal.
  • dc: Codi de 2 dígits per als digits de control.
  • cuenta: Codi de 10 dígits Número de compte.

PARÀMETRES DE SORTIDA

  • boolean: true si entitat+sucursal+dc+compte formen un número de compte vàlid i false si entitat+sucursal+dc+compte no formen un número de compte vàlid.

EXEMPLES

Exemple 1 Validar els següents valors: entitat: '1234', sucursal: '5678', dc: '90', compte: '1234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido('1234', '5678', '90', '1234567890');

La función esNumeroCuentaValido retorna 'false' ya que los parámetros introducidos no forman un número de cuenta válido, el dígito de control no es correcto..


Exemple 2 Validar els següents valors: entitat: '1234', sucursal: '5678', dc: '06', compte: '1234567890'.

var valido = PLUGIN_UTILS.esNumeroCuentaValido ('1234', '5678', '06', '1234567890');

La función esNumeroCuentaValido retorna 'true', ja que els paràmetres introduïts formen un número de compte vàlid.
