distanciaDias(String fecha1, String fecha2, String formatoFecha): int

DESCRIPCIÓ

  • Mètode que retorna el nombre de dies que hi ha entre una data i una altra.

PARÀMETRES D'ENTRADA

  • fecha1: Cadena que conté la data inicial.
  • fecha2: Cadena que conté la data final.
  • formatoFecha: Format de les dates. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • int: Nombre de dies que hi ha de distància entre una data i una altra.

EXEMPLES

Exemple 1 Si volem esbrinar el nombre de dies que hi ha entre 2 dates (s'han d'especificar usant el següent format yyyy-MM-dd), per exemple, entre el 23/04/2012 i 13/06/2012 podem usar la següent sentència:

var dias = PLUGIN_UTILS.distanciaDias('2012-04-23', '2012-06-13', 'yyyy-MM-dd');

Exemple 2 Si volem esbrinar el nombre de dies que hi ha entre 2 dates (especificant el format dd/MM/yyyy), per exemple, entre el 23/04/2012 i 13/06/2012 podem usar la següent sentència:

var dias = PLUGIN_UTILS.distanciaDias('23/04/2012', '13/06/2012', 'dd/MM/yyyy');
