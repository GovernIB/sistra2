getValorCompuesto(String idFormulario, String campo): VALOR_CAMPOCOMPUESTO

DESCRIPCIÓ

  • Mètode que retorna el valor d'un camp compost.

PARÀMETRES D'ENTRADA

  • idFormulario:Cadena que conté l'identificador del formulari.
  • campo: Cadena que conté l'identificador del camp.

PARÀMETRES DE SORTIDA

  • VALOR_CAMPOCOMPUESTO: Retorna l'objecte VALOR_CAMPOCOMPUESTO. Aquest objecte està format per 2 camps, codi i descripció, els valors dels quals es poden obtenir mitjançant les funcions getCodigo() i getDescripcion() respectivament.

Exemple

Obtenir el valor del camp compost 'campoProvincia' del formulari amb id 'Form1':

var valorCompuesto = PLUGIN_FORMULARIOS.getValorCompuesto('Form1','campoProvincia');

La funció getValorCompuesto retornarà l'objecte valorCompuesto. Per a obtenir les dades del valor compost podem demanar el seu codi o descripció tal com es mostra a continuació:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En aquest cas la funció getCodigo retornarà la cadena '07' i la funció getDescripcion retornarà la cadena 'Illes Balears'.
