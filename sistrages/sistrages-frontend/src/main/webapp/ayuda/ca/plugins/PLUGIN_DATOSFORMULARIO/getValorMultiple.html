getValorMultiple(String campo, String indice): VALOR_CAMPOCOMPUESTO

DESCRIPCIÓ

  • Mètode que retorna el valor d'un camp múltiple.

PARÀMETRES D'ENTRADA

  • campo:Cadena que conté l'identificador del camp.

PARÀMETRES DE SORTIDA

  • VALOR_CAMPOCOMPUESTO: Retorna l'objecte VALOR_CAMPOCOMPUESTO situat en l'índex del camp múltiple. Aquest objecte està format per 2 camps, codi i descripció, els valors dels quals es poden obtenir mitjançant les funcions getCodigo() i getDescripcion() respectivament.

Exemple

Obtenir l'últim valor del camp múltiple 'campoTitulaciones', per a això primer s'ha d'obtenir el total de valors:

var total = PLUGIN_DATOSFORMULARIO.getNumeroValoresValorMultiple('campoTitulaciones');

La funció getNumeroValoresValorMultiple retornarà 8 (total de valors d'aquest camp). Amb aquest valor podem calcular l'índex de l'últim element de la següent manera:

if (numero > 0) { var valorCompuesto = PLUGIN_DATOSFORMULARIO.getValorMultiple('campoTitulaciones', total -1); }

La funció getValorMultiple retornarà l'objecte valorCompuesto situat en la posició indicada. Per a obtenir les dades del valor compost podem demanar el seu codi o descripció tal com es mostra a continuació:

var descripcion = valorCompuesto.getDescripcion();
var codigo = valorCompuesto.getCodigo();

En aquest cas la funció getCodigo retornarà la cadena 'GRAD.SUP.INFOR' i la funció getDescripcion retornarà la cadena 'Grau superior en informàtica'.
