getNumeroFilas(): int

DESCRIPCIÓ

  • Mètode que permet obtenir el nombre de files retornats com a resultat de la invocació d'un domini.

PARÀMETRES DE SORTIDA

  • int: Enter que representa el nombre de files retornades pel domini.

EXEMPLES

Posam per cas que es vol recuperar el nombre de files que retorna el domini 'DOMINIO'. Les passes a efecutar són les següents:

Primer s'invoca el domini mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

i es recupera el nombre de files de la manera següent:

var numFilas = valors.getNumeroFilas();
