DATOS_ANEXOS_DINAMICOS

DESCRIPCIÓ

  • Objecte que permet establir els annexes de manera dinàmica, indicant-hi la següent informació per annexe: identificador, descripció, extensions permeses, mida màxima, url plantilla, obligatorietat, convertir a PDF i signar.
  • Aquest objecte està disponible per al script de Llista dinàmica d'annexes de la passa annexar documents.

MÈTODES

  • addAnexo(String identificador, String descripcion, String extensiones, String tamanyoMax, String urlPlantilla, boolean obligatorio, boolean convertirPDF, boolean firmar)
