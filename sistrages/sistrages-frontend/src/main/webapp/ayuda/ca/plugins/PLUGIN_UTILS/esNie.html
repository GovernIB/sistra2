esNie(String nie): boolean

DESCRIPCIÓ

  • Valida si el valor introduït posseeix el format requerit per a ser un NIE (Número Identificació d'Estranger) vàlid. Es considera vàlida tota aquella cadena que tingui 9 caràcters alfanumèrics i compleixi amb la regla de validació.

PARÀMETRES D'ENTRADA

  • nie: cadena que representa el NIE que es vol validar.

PARÀMETRES DE SORTIDA

  • boolean : true si un NIE és vàlid i false si no es un NIE vàlid.

EXEMPLES

Exemple 1 Validar la cadena 'Y23459781X'.

var valido = PLUGIN_UTILS.esNie('Y23459781X');

La funció esNie retorna 'true' pel fet que el valor indicat correspon a un NIE.


Exemple 2 Validar la cadena 'A623459781.

var valido = PLUGIN_UTILS.esNie('A623459781');

La funció esNie retorna 'false' ja que el valor indicat no correspon a un NIE vàlid.
