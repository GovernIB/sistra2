rtrim(String cadena): String

DESCRIPCIÓ

  • Mètode que elimina els espais situats a la dreta en la cadena.

PARÀMETRES D'ENTRADA

  • cadena: Cadena de text a tractar

PARÀMETRES DE SORTIDA

  • String: Cadena de text sense espais per la dreta.

EXEMPLES

Tractar la cadena '  cadena ejemplo    ', per a eliminar els espais de la dreta.

var pCadenaLimpia = PLUGIN_VALIDACIONES.rtrim('  cadena ejemplo    ');

La funció rtrim retorna '  cadena ejemplo'.
