addParametroMensaje(String parametro):void

DESCRIPCIÓ

  • Mètode que permet afegir el valor dels paràmetres a substituir a la informació. D'aquesta manera es pot dotar de dinamisme aquesta informació.
  • Els paràmetres s'identifiquen a la informació mitjançant claus ({0} {1} ...).
  • Si el missatge té més d'un paràmetre, cal tenir present que importa l'ordre en què s'hi afegeixen.

PARÀMETRES D'ENTRADA

  • paràmetre: String amb el valor del paràmetre que cal fixar.

EXEMPLES

Posem per exemple que a la informació de Cal Saber volem mostrar un NIF i una data

Prèviament, a la pantalla de script s'ha de definir la informació a la pestanya de missatges de validació, per exemple amb l'identificador CodFechas i el valor següent:
El DNI {0} entra a data {1}.
Des de l'script, es fa referència a aquest missatge de la manera següent:

DATOS_INSTRUCCIONES.setCodigoMensaje('CodFechas');
DATOS_INSTRUCCIONES.addParametroMensaje('12345678T');
DATOS_INSTRUCCIONES.addParametroMensaje('10/09/2012');

Com a resultat de l'execució de l'script s'obtindrà el resultat següent:
El DNI 12345678T entra a data 10/09/2012.
