addAnexo(String identificador, String descripcion, String extensiones, String tamanyoMax, String urlPlantilla, boolean obligatorio, boolean convertirPDF, boolean firmar): void

DESCRIPCIÓ

  • Mètode que permet afegir un nou annex de forma dinàmica a la versió del tràmit.

PARÀMETRES D'ENTRADA

  • identificador: Identificador de l'annex.
  • descripcion: Descripció de l'annex.
  • extensiones: Extensions permeses per a l'annex. Llista de valors separats per comes. És necessari indicar almenys una extensió.
  • tamanyoMax: Mida màxima de l'annex. És opcional. Si s'indica, s'ha d'afegir la unitat al final del mateix. Valors possibles:
    1. KB
    2. MB
  • urlPlantilla: Url en la qual es pot descarregar la plantilla de l'annex. Dada opcional.
  • obligatorio: Variable booleana que permet marcar si l'annex és obligatori o no.
  • convertirPDF: Variable booleana que permet marcar si l'annex s'ha de convertir a pdf.
  • firmar: Variable booleana que permet marcar si s'ha de signar o no l'annex.

EXEMPLES

Si desitjam afegir un nou annexe de manera dinàmica a la versió d'un tràmit, executaríem el següent:

DATOS_ANEXOS_DINAMICOS.addAnexo('ANE1', 'Documento de conformidad', 'doc,pdf', '1024 KB', 'http://caib.es/anexos/plantillaConformodidad.pdf', true, true, true);

El resultat de l'execució del script "Llista dinàmica Annexos" amb aquesta sentència serà un annex que és obligatori, que ha de pujar-se en format doc/pdf, amb una grandària màxima de 1024 KB (1MB), que ha de ser signat i que es convertirà a pdf.
