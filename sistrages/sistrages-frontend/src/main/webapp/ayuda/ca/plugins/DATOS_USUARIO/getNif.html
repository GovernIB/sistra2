getNif(): String

DESCRIPCIÓ

  • Mètode que retorna el NIF de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String:Cadena de caràcters amb el NIF de l'usuari autenticat.

Exemple

Si es desitja obtenir el NIF de l'usuari de la sessió actual, en aquest cas '11111111H', primer s'ha de comprovar si la sessió és autenticada, a continuació, obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el seu NIF mitjançant la funció getNif:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nif = usuario.getNif();
}

La funció getNif retornarà la cadena '11111111H'.
