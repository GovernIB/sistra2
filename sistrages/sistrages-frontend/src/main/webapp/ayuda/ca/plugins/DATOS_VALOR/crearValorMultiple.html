crearValorMultiple(): ValorCampoMultiple

DESCRIPCIÓ

  • Mètode per a crear un valor múltiple per a després poder utilitzar-lo en un camp del formulari actual.

PARÀMETRES DE SORTIDA

  • ValorCampoMultiple: Retorna l'objecte ValorCampoMultiple inicialitzat, el qual permet afegir nous valors.

EXEMPLES

Per exemple, per a generar un conjunt de dades geogràfiques predefinides, usarem aquesta funció per a inicialitzar la variable:

var datos = DATOS_VALOR.crearValorMultiple();

d'aquesta manera podem afegir els nous valors a la variable usant el mètode "addValor(codigo, descripcion)", on el primer paràmetre indica el codi i el segon paràmetre el valor:

datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');
