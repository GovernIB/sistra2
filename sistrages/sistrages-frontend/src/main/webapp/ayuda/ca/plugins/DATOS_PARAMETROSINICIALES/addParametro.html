addParametro(String codigo, String valor): void

DESCRIPCIÓ

  • Mètode que permet afegir un nou paràmetre a la versió d'un tràmit.

PARÀMETRES D'ENTRADA

  • codigo: String amb el codi del paràmetre a afegir a la versió del tràmit.
  • valor: String con el valor del parámetro a añadir a la versión del trámite.

EXEMPLES

Suposam que volem afegir un paràmetre, per exemple província, a la versió d'un tràmit. A aquest paràmetre li volem assignar el codi 07 referit a la província Illes Balears. Per a això faríem el següent:

var params = DATOS_PARAMETROS_INICIALES.addParametro('prov','07');
