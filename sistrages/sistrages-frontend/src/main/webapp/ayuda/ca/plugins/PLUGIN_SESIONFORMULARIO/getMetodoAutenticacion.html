getMetodoAutenticacion(): String

DESCRIPCIÓ

  • Mètode que retorna el mètode d'autenticació.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que conté el mètode d'autenticació. Els valors possibles són:
    • 'ANONIMO'
    • 'CLAVE_CERTIFICADO'
    • 'CLAVE_PIN'
    • 'CLAVE_PERMANENTE'

Exemple

Obtenir el mètode d'autenticació de l'actual sessió, en aquest cas és un usuari no autenticat:

var metodo = PLUGIN_SESIONFORMULARIO.getMetodoAutenticacion();

La funció getMetodoAutenticacion retornarà la cadena 'ANONIMO'.
