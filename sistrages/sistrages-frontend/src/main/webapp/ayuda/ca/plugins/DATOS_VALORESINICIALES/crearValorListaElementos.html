crearValorListaElementos(): ValorCampoListaElementos

DESCRIPCIÓ

  • Mètode per a crear un valor de llista d'elements per a després poder utilitzar-lo en un camp del formulari.

PARÀMETRES DE SORTIDA

  • ValorCampoListaElementos: Retorna l'objecte ValorCampoListaElementos inicialitzat, el qual permet afegir nous valors.

EXEMPLES

Per exemple, per a generar una sèrie de valors per a una llista d'elements:

var datos = DATOS_VALOR.crearValorListaElementos();

D'aquesta manera podem afegir els nous valors a la variable fent servir el mètode "addValor(codigo, descripcion)", on el primer paràmetre indica el codi i el segon paràmetre el valor:

datos.addValor('codigo', '07073'); datos.addValor('descripcion', 'Mallorca');
