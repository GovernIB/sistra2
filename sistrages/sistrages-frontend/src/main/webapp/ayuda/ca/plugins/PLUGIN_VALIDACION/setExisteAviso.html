setExisteAviso(boolean existeAviso, String tipo)

DESCRIPCIÓ

  • Mètode per establir si es genera avís.

PARÀMETRES D'ENTRADA

  • existeAviso: booleà que permet establir si es genera avís.
  • tipo: INFO / WARNING / ERROR

EXEMPLES

Per indicar que existeix un avís de tipo WARNING incorporarem el codi següent:

PLUGIN_VALIDACION.setExisteAviso(true, 'WARNING');
