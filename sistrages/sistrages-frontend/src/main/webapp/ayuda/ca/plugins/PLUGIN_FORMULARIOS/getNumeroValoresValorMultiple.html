getNumeroValoresValorMultiple(String idFormulario, String campo): int

DESCRIPCIÓ

  • Mètode que retorna el nombre de valors possibles d'un determinat camp.

PARÀMETRES D'ENTRADA

  • idFormulario: Cadena que conté l'identificador del formulari.
  • campo: Cadena que conté l'identificador del camp.

PARÀMETRES DE SORTIDA

  • int: Nombre de valors del camp múltiple.

Exemple

Obtenir el nombre de valors del camp multivalor 'campoTitulaciones' situat en el formulari 'idForm1':

var numero = PLUGIN_FORMULARIOS.getNumeroValoresValorMultiple('idForm1','campoTitulaciones');

La funció getNumeroValoresValorMultiple retornarà el número 3, la qual cosa indica que aquest camp té 3 possibles valors.
