esIgual(String cadena1, String cadena2): boolean

DESCRIPCIÓ

  • Valida si les dues cadenes introduïdes són iguals. Es consideraran iguals totes aquelles cadenes que coincideixin en tots els seus caràcters, o si ambdues són nul·les.

PARÀMETRES D'ENTRADA

  • cadena1: Primera cadena per a comparar.
  • cadena2: Segona cadena per a comparar.

PARÀMETRES DE SORTIDA

  • boolean : true si les cadenes són iguals i false si les cadenes no són iguals.

EXEMPLES

Exemple 1 Validar si les cadenes 'abc' i 'abc' són iguals

var iguales = PLUGIN_VALIDACIONES.esIgual ('abc', 'abc');

La funció esIgual retorna 'true', pel fet que el contingut de les dues cadenes és el mateix.


Exemple 2 Validar si les cadenes 'abc' i 'ABC' són iguals.

var iguales = PLUGIN_VALIDACIONES.esIgual ('abc', 'ABC');

La funció esIgual retorna 'false', pel fet que les cadenes no tenen el mateix contingut.
