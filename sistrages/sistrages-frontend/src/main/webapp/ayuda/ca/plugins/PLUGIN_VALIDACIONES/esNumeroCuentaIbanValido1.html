esNumeroCuentaIbanValido(String cuenta): boolean

DESCRIPCIÓ

  • Valida que el valor introduït no sigui nul, i posseeixi el format corresponent a un número de compte IBAN, complint amb les regles de validació necessàries per a ser un IBAN (International Bank Account Number) Vàlid. Es considera vàlida tota aquella cadena que tingui 24 números que compleixin amb les regles de validació corresponents a clau del pais, codi de control i identificador NCC (entitat, oficina, dígit de control i número de compte).

PARÀMETRES D'ENTRADA

  • cuenta: Codi de 24 dígits del Número de compte.

PARÀMETRES DE SORTIDA

  • boolean: true si la cadena introduïda és un número de compte vàlid i false si la cadena introduïda no és un número de compte vàlid

EXEMPLES

Exemple 1 Validar la cadena 'ES6621000418461234567891'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaIbanValido('ES6621000418461234567891');

La funció esNumeroCuentaIbanValido retorna 'false' pel fet que la cadena no compleix amb les regles de validació, el dígit de control no és vàlid (el CC és 40, no 46).


Exemple 2 Validar la cadena 'ES6621000418401234567891'.

var valido = PLUGIN_VALIDACIONES.esNumeroCuentaIbanValido ('ES6621000418401234567891');

La funció esNumeroCuentaIbanValido retorna 'true', pel fet que la cadena compleix els requisits per a ser un número de compte IBAN vàlid.
