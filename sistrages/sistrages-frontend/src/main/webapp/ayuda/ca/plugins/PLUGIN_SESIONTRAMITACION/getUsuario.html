getUsuario(): : DATOS_USUARIO

DESCRIPCIÓ

  • Mètode que retorna l'objecte'DATOS_USUARIO' amb la informació de l'usuari autenticat.
  • Només és possible el seu ús en tràmits autenticats.

PARÀMETRES DE SORTIDA

  • DATOS_USUARIO: Retorna l'objecte DATOS_USUARIO amb la informació de l'usuari.

Exemple

Suposem que desitgem obtenir el nom de l'usuari de la sessió actual (només per a tràmits autenticats)

var autenticado = PLUGIN_SESIONTRAMITACION.isAutenticado(); if (autenticado == true) { var usuario = PLUGIN_SESIONTRAMITACION.getUsuario(); }

La funció getUsuario retorna l'objecte DATOS_USUARIO a partir del qual extreure la informació que ens interessa, per exemple, el nom, mitjançant la següent funció:

var nombre = usuario.getNombre();
