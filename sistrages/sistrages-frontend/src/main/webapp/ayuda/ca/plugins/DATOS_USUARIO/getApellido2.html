getApellido2(): String

DESCRIPCIÓ

  • Mètode que retorna el segon cognom de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters amb el segon cognom de l'usuari autenticat.

Exemple

Si es desitja obtenir el segon cognom de l'usuari de la sessió actual, en aquest cas 'Jose Pérez Rodriguez', primer s'ha de comprovar si la sessió és autenticada, a continuació obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el seu segon cognom mitjançant la funció getApellido2:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellido = usuario.getApellido2();
}

La funció getApellido2 retornarà la cadena 'Rodriguez'.
