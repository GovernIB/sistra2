sumaMeses(String fecha, int meses, String formatoFecha): String

DESCRIPCIÓ

  • Mètode que retorna la data resultant de sumar el nombre de mesos indicat en el paràmetre 'meses' a la data indicada en el paràmetre 'data'.

PARÀMETRES D'ENTRADA

  • fecha: Cadena que conté la data.
  • meses: Nombre de mesos a sumar.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters que representa la data amb els mesos sumats.

EXEMPLES

Sumar 3 mesos a la data 23/04/2012.

var fecha = PLUGIN_UTILS.sumaMeses('23/04/2012', 3, 'dd/MM/yyyy');

La funció sumaMeses retorna la cadena '23/07/2012'.
