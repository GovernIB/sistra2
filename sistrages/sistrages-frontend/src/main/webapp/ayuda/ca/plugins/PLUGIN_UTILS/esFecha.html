esFecha(String fecha, String formatoFecha): boolean

DESCRIPCIÓ

  • Valida que la data introduïda no sigui nul·la ni buida.

PARÀMETRES D'ENTRADA

  • fecha: Data a validar.
  • formatoFecha: Format de la data. És opcional i el format per defecte és dd/MM/yyyy.

PARÀMETRES DE SORTIDA

  • boolean: true si la data és vàlida i false si la data no és vàlida.

EXEMPLES

Exemple 1 Validar una cadena de caràcters buida ''

var valido = PLUGIN_UTILS.esFecha('', null);

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida.


Exemple 2 Validar una cadena de caràcters nul·la.

var fecha;
var valido = PLUGIN_UTILS.esFecha(fecha, null);

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida.


Exemple 3 Validar la cadena de caràcters '10/06/2012'.

var valido = PLUGIN_UTILS.esFecha('10/06/2012', null);

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida, no segueix el format dd/MM/yyyy


Exemple 4 Validar la cadena de caràcters '10/06/2012'.

var valido = PLUGIN_UTILS.esFecha('10/06/2012', 'dd/MM/yyyy');

La funció esFecha retorna 'true', pel fet que el valor indicat és una data vàlida amb el format dd/MM/yyyy


Exemple 5 Validar la cadena de caràcters '2012-12-09'.

var valido = PLUGIN_UTILS.esFecha('2012-12-09', null);

La funció esFecha retorna 'true', pel fet que el valor indicat és una data vàlida.
