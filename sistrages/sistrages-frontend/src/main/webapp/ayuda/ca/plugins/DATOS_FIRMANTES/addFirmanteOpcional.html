addFirmanteOpcional(String nif, String nombre): void

DESCRIPCIÓ

  • Mètode que hi afegeix un signant opcional. Dels signants que s'afegeixin amb el mètode addFirmanteOpcional(), almenys un haurà de signar obligatòriament.

PARÀMETRES D'ENTRADA

  • nif: String amb el NIF del signant. Ha de ser un NIF vàlid.
  • nombre: String amb el nom del signant. Dada obligatòria.

EXEMPLES

Exemple 1.- Posam per cas que volem establir dos signant per a un formulari, un obligatori (A) i un altre opcional (B). En aquest cas emplenarem el script "Establir signants" amb el contingut següent:

DATOS_FIRMANTES.addFirmante(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniB, nomB);

El resultat seria que l'assistent de tramitació sol·licitarà que el formulari sigui signat per A obligatoriament.

Exemple 2.- Si volem establir dos signant per a un formulari i es indiferent quin es el signant obligatori (A o B), emplenarem el script "Establir signants" amb el contingut següent:

DATOS_FIRMANTES.addFirmanteOpcional(dniA, nomA); DATOS_FIRMANTES.addFirmanteOpcional(dniB, nomB);

El resultat seria que l'assistent de tramitació sol·licitarà que el formulari sigui signat per A o per B obligatoriament.
