getIdioma(): String

DESCRIPCIÓ

  • Mètode que retorna l'idioma de tramitació.

PARÀMETRES DE SORTIDA

  • string: Cadena de caràcters que conté l'idioma de tramitació, sent el valor 'es' per a l'espanyol, 'ca' per al català o 'en' per a l'anglès.

Exemple

Obtenir l'idioma amb el qual s'està tramitant. En aquest cas Català

var idioma = PLUGIN_SESIONTRAMITACION.getIdioma();

La funció getIdioma retornarà 'ca'.
