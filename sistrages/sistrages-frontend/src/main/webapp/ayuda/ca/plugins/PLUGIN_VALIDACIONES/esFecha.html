esFecha(String fecha): boolean

DESCRIPCIÓ

  • Valida que la data introduïda no sigui nul·la ni buida, i que estigui formada de la següent manera: yyyy-MM-dd.

PARÀMETRES D'ENTRADA

  • fecha: Data a validar.

PARÀMETRES DE SORTIDA

  • boolean: true si la data és vàlida i false si la data no és vàlida.

EXEMPLES

Exemple 1 Validar una cadena de caràcters buida ''

var valido = PLUGIN_VALIDACIONES.esFecha('');

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida.


Exemple 2 Validar una cadena de caràcters nul·la.

var fecha;
var valido = PLUGIN_VALIDACIONES.esFecha(fecha);

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida.


Exemple 3 Validar la cadena de caràcters '10/06/2012'.

var valido = PLUGIN_VALIDACIONES.esFecha('10/06/2012');

La funció esFecha retorna 'false', pel fet que el valor indicat no és una data vàlida, no segueix el format yyyy-MM-dd


Exemple 4 Validar la cadena de caràcters '2012-12-09'.

var valido = PLUGIN_VALIDACIONES.esFecha('2012-12-09');

La funció esFecha retorna 'true', pel fet que el valor indicat és una data vàlida.
