DATOS_PARAMETROSINICIALES

DESCRIPCIÓ

  • Per defecte els paràmetres inicials seran els passats el paràmetre "parametros" a la url d'inici del tràmit. En aquest script es permet afegir dinàmicament paràmetres inicials (p.e. calculats a partir dels paràmetres que vénen a la url d'inici).
  • Els paràmetres inicials estaran disponibles per a la resta d'scripts del flux a través de PLUGIN_SESIONTRAMITACION.getParametroInicio (idParametro).

MÈTODES
