formateaCadenaAInt(String numero, String separadorMiles): int

DESCRIPCIÓ

  • Mètode per a convertir un string en un número int.

PARÀMETRES D'ENTRADA

  • numero: Cadena a convertir a int.
  • separadorMiles: Caràcter amb el qual s'identifica el separador de milers.

PARÀMETRES DE SORTIDA

  • int: Número en format int. Si no es pot convertir, llança un error.

EXEMPLES

Exemple 1 Convertir la cadena '123.110.544.434' a int.

var nInt = PLUGIN_UTILS.formateaCadenaAInt('123.110.544.434', '.')

La funció formateaCadenaAInt retorna el número 123110544434 en format int


Exemple 2 Convertir la cadena '123,110,544,434' a int.

var nInt = PLUGIN_UTILS.formateaCadenaAInt('123,110,544,434', ',')

La funció formateaCadenaAInt retorna el número 123110544434 en format int.
