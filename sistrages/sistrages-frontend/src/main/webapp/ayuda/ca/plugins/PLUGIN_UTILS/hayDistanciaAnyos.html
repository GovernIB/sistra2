hayDistanciaAnyos(String fechaIni, String fechaFin, String formatoFecha, int distanciaAnyos): boolean

DESCRIPCIÓ

  • Valida que la distància mínima entre dues dates sigui com a mínim del paràmetre de distància que es passa per entrada.

PARÀMETRES D'ENTRADA

  • fechaIni: Data inicial menor que la fechaFin.
  • fechaFin: Fecha final mayor que la fechaIni.
  • formatoFecha: Paràmetre opcional que indica el format de les dates per a convertir-lo en una variable de tipus data. Si es passa nul, el format data és 'dd/MM/YYYY' (per exemple, una data en aquest format seria '30/12/2019').
  • distanciaAnyosNombre mínim d'anys de distància.

PARÀMETRES DE SORTIDA

  • boolean: true si la distància en anys entre totes dues dates és com a mínim el paràmetre añosDistancia, false si no compleix el mínim en anyos de distància entre tots dos.

EXEMPLES

Exemple 1 Validar si la distància entre dues dates és d'almenys 3 anys.

var valido = PLUGIN_UTILS.hayDistanciaAnyos('01/06/2019','05/07/2015', null,3);

La funció hayDistanciaAnyos retorna 'true' perquè la distància d'anys entre totes dues dates és major o igual a 3 anys.


Exemple 2 Validar si la distància entre dues dates és d'almenys 5 anys.

var valido = PLUGIN_UTILS.hayDistanciaAnyos('01/01/2018','05/07/2015','dd/MM/YYYY',5);

La funció hayDistanciaAnyos retorna 'false' perquè la distància d'anys entre totes dues dates no és major i igual a 5 anys.
