getValorXml(String xpath): String

DESCRIPCIÓ

  • Mètode que retorna el valor d'un camp del XML annexat. Aquest mètode únicament funcionarà per annexes amb format XML.

PARÀMETRES D'ENTRADA

  • xpath: xpath del camp del XML annexat.

PARÀMETRES DE SORTIDA

  • String: Valor del camp del XML annexat. Si el camp no existeix, retorna una cadena buida.

EXEMPLE

Per a poder fer ús d'aquesta funcionalitat, és necessari conèixer per endavant l'estructura del XML.

Suposem que desitjam validar que un determinat XML compleix amb una estructura concreta, validant per exemple que inclou un camp 'dni'. En aquest cas podríem emprar la sentència següent per a obtenir el camp 'dni' situat en el XML, i posteriorment verificar que sigui un DNI vàlid.

<datos>
<dni>11111111H</dni>
<nombre>Nombre Apellido1 Apellido2</nombre>
</datos>

var dni = PLUGIN_ANEXO.getValorXml('//datos/dni');
