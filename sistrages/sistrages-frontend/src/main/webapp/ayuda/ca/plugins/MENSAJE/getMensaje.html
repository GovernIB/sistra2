getMensaje(): String

DESCRIPCIÓ

  • Mètode que permet recuperar el valor del missatge de validació.

PARÀMETRES DE SORTIDA

  • String: String amb el text del missatge de validació.

EXEMPLES

Posam per cas que volem recuperar el text d'un missatge de validació, per exemple MENSAJE_001, prèviament creat. Les passes a seguir serien les següents:

A la pantalla de script es defineix un missatge, a la pipella de missatges de validació, amb l'identificador MENSAJE_001 amb valor:

El valor del certificat no es correspon amb el valor cancel·lat.

Des del script es pot referenciar aquest missatge de la manera següent:

var msg = PLUGIN_MENSAJES.crearMensaje('MENSAJE_001');

Per obtenir el valor del missatge al script:

var missatge = msg.getMensaje();
