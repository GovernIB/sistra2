getApellido1(): String

DESCRIPCIÓ

  • Mètode que retorna el primer cognom de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String: Cadena de caràcters amb el primer cognom de l'usuari autenticat.

Exemple

Si es desitja obtenir el primer cognom de l'usuari de la sessió actual, en aquest cas 'Jose Pérez Rodriguez', primer s'ha de comprovar si la sessió és autenticada, a continuació obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el seu primer cognom mitjançant la funció getApellido1:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var apellido = usuario.getApellido1();
}

La funció getApellido1 retornarà la cadena 'Pérez'.
