setTextoMensaje(String texto):void

DESCRIPCIÓ

  • Mètode que permet fixar directament per script el text de la informació.

PARÀMETRES D'ENTRADA

  • texto: String amb el text pertinent a la informació.

EXEMPLES

Posem per cas que volem establir una instrucció directament des del script. Ho podríem fer a través de l'script de la manera següent:

DATOS_INSTRUCCIONES.setTextoMensaje('Ha d'emplenar totes les pases');
