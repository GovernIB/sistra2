crearValorMultiple(): ValorCampoMultiple

DESCRIPCIÓ

  • Mètode que permet crear un valor múltiple per a després poder utilitzar-lo en un camp del formulari actual.

PARÀMETRES DE SORTIDA

  • ValorCampoMultiple: Crea el valor del campo múltiple.

EXEMPLES

Per exemple, si desitjam generar un conjunt de dades geogràfiques predefenides en un formulari, primer crearíem la variable:

var datos = DATOS_VALORESINICIALES.crearValorMultiple();

i posteriorment afegiríem les dades a la variable, sent el primer paràmetre d'entrada el codi i el segon el valor:

datos.addValor('codProvincia', '07'); datos.addValor('codMunicipio', '07073'); datos.addValor('codPostal', '07406');
