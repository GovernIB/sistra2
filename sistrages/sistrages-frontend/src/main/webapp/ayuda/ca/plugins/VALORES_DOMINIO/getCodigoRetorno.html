getCodigoRetorno():String

DESCRIPCIÓ

  • En la recuperació del domini es pot establir de forma opcional un codi de retorn, sempre que es produeixi correctament la resolució del domini

PARÀMETRES DE SORTIDA

  • String: Objecte String que modelitza el codi de retorn.

EXEMPLES

Posam per cas que es vol recuperar el codi de retorno que retorna el domini 'DOMINIO'. Les passes a efecutar són les següents:

Primer s'invoca el domini mitjançant PLUGIN_DOMINIOS:

var valors = PLUGIN_DOMINIOS.invocarDominio('DOMINIO');

i es recupera el codi de retorn de la manera següent:

var retCode = valors.getCodigoRetorno();
