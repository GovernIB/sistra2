setDetallePago(String modelo, String concepto, String tasa, int importe): void

DESCRIPCIÓ

  • Mètode que permet establir el detall del pagament.

PARÀMETRES D'ENTRADA

  • modelo: Model de pagament.
  • concepto: Concepte de pagament.
  • tasa: Codi de la taxa.
  • importe: Import del pagament en cèntims. Ha de ser un nombre natural (numèric, sencer i positiu).

EXEMPLES

Per exemple, si es desitja establir el detall per a una taxa definida en la versió d'un tràmit, s'hauria de fer de la manera següent:

DATOS_PAGO.setDetallePago('Modelo1', 'Tasa de admisión a examen', 'TASA1', '8950');

El resultat és un pagament del model 'Modelo1' amb un import de 89 euros amb 50 cèntims, amb el codigo de taxa TASA1 i el concepte 'Tasa de admisión a examen'. El model i el codi de taxa a indicar dependrà de la plataforma de pagament a emprar
