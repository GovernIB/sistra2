esNulo(Object objeto): boolean

DESCRIPCIÓ

  • Valida si el paràmetre objecte és nul.

PARÀMETRES D'ENTRADA

  • objeto: Objecte a validar.

PARÀMETRES DE SORTIDA

  • boolean : true si l'objecte és nul i false si l'objecte no és nul.

EXEMPLES

Exemple 1 Validar una variable inicialitzada a null.

Object object = null;
var vacio = PLUGIN_UTILS.esNulo(object);

La funció esNulo retorna 'true', pel fet que el contingut de la variable és null.


Exemple 2 Validar una cadena de caràcters buida.

var vacio = PLUGIN_UTILS.esNulo('');

La funció esNulo retorna 'false', pel fet que l'objecte no és null.
