esNatural(String numero): boolean

DESCRIPCIÓ

  • Valida si el valor introduït és no nul, numèric, sencer i positiu.

PARÀMETRES D'ENTRADA

  • numero: Nombre natural a validar.

PARÀMETRES DE SORTIDA

  • boolean : true si la cadena és un nombre natural i false si la cadena no és un nombre natural.

EXEMPLES

Exemple 1 Validar si la cadena 4,0 és un nombre natural.

var valido = PLUGIN_UTILS.esNatural('4,0');

La funció esNatural retorna 'false', pel fet que la cadena no representa un nombre natural.


Exemple 2 Validar si la cadena -77 és un nombre natural.

var valido = PLUGIN_UTILS.esNatural('-77');

La funció esNatural retorna 'false', pel fet que la cadena no representa un nombre natural.


Exemple 3 Validar si la cadena 5 és un nombre natural

var valido = PLUGIN_UTILS.esNatural('5');

la funció esNatural retorna 'true', pel fet que la cadena representa un nombre natural.
