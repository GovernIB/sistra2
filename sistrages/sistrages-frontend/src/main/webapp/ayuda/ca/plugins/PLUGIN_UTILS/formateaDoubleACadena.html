formateaDoubleACadena(double numero, String separadorMiles, String separadorDecimales, int numeroDecimales): String

DESCRIPCIÓ

  • Mètode per a convertir un número de tipus double en un string

PARÀMETRES D'ENTRADA

  • numero: Número double a convertir en String.
  • separadorMiles:Caràcter amb el qual s'identifica el separador de milers.
  • separadorDecimales: Caràcter amb el qual s'identifica el separador de decimals. Dada obligatòria.
  • separadorDecimales: Nombre de decimals.

PARÀMETRES DE SORTIDA

  • String:Cadena de caràcters que representa el número introduït.

EXEMPLES

Exemple 1 Convertir el número 123110544434.58 en una cadena de caràcters

var pCadena = PLUGIN_UTILS.formateaDoubleACadena(123110544434.58, '.', ',', 2);

La funció formateaDoubleACadena retorna la cadena '123.110.544.434,58'.


Exemple 2 Convertir el número 123110544434.58 en una cadena de caràcters sense incloure els decimals

var pCadena = PLUGIN_UTILS.formateaDoubleACadena(123110544434.58, '.', ',', 0);

La funció formateaDoubleACadena retorna la cadena '123.110.544.435', sense incloure els decimals.
