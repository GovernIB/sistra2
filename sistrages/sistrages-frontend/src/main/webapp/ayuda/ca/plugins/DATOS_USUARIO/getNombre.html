getNombre(): String

DESCRIPCIÓ

  • Mètode que retorna el nom de l'usuari autenticat.

PARÀMETRES DE SORTIDA

  • String: cadena de caràcters amb el nom de l'usuari autenticat.

Exemple

Si es desitja obtenir el nom de l'usuari de la sessió actual, en aquest cas 'Jose Pérez Rodriguez', primer s'ha de comprovar si la sessió és autenticada, a continuació, obtenir l'objecte DADES_USUARI mitjançant getUsuario. Una vegada obtingut l'usuari, podem obtenir el seu nom mitjançant la funció getNombre:

var autenticado = PLUGIN_SESIONFORMULARIO.isAutenticado();
if (autenticado == true) {
var usuario = PLUGIN_SESIONFORMULARIO.getUsuario();
var nombre = usuario.getNombre();
}

La funció getNombre retornarà la cadena'Jose'.
