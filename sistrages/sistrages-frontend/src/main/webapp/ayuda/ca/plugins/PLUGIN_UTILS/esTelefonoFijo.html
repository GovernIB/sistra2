esTelefonoFijo(String telefono): boolean

DESCRIPCIÓ

  • Valida si el valor introduït compleix amb les regles de validació necessàries per a ser un número de telèfon fix vàlid.
  • Es considera vàlida tota aquella cadena no nul·la que tingui 9 números i que comenci per 9.

PARÀMETRES D'ENTRADA

  • telefono: Cadena que representa el número de telèfon.

PARÀMETRES DE SORTIDA

  • boolean:true si la cadena representa un telèfon fix vàlid i false si la cadena no representa un telèfon fix vàlid

EXEMPLES

Exemple 1 Validar la cadena '971177714'.

var valido = PLUGIN_UTILS.esTelefonoFijo('971177714');

La funció esTelefonoFijo retorna 'true', pel fet que el telèfon es correspon amb el format d'un número de telèfon fix.


Exemple 2 Validar la cadena '663467284'.

var valido = PLUGIN_UTILS.esTelefonoFijo('663467284');

La funció esTelefonoFijo retorna 'false', pel fet que el telèfon no es correspon amb el format d'un número de telèfon fix. La cadena correspon a un número de telèfon mòbil.
