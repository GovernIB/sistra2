PLUGIN_ERROR

DESCRIPCIÓ

  • Plugin que permet establir errors a l'execució d'un script.
  • Si el script és de validació, es permetrà continuar amb el tràmit però es mostrarà al navegador un missatge d'alerta amb el text fixat mitjançant aquest plugin. Els scripts de validació són:
    • script de personalització (accessible des de Propietats de la versió d'un tràmit).
    • script de Postguadar formulari (per un Formulari de la passa 'Emplenar formulari').
    • script de validació (per un annex de la passa 'Anexar document').
    • script de Permet registrar (a la passa 'Registrar formulari').
  • Per la resta de scripts, en cas s'estableixi un error es forçarà la sortida del tràmit indicant un missatge d'error genèric. El missatge d'error establert a través del plugin es visualitzarà a la finestra de debug (des del navegador amb la sessió oberta de la tramitació fallida, s'ha d'obrir una altra pipella al navegador i accedir a l'adreça següent: /sistramitfront/debug/logSesionTramitacion.html)
  • Aquest plugin està disponible per tots els scripts de versió, passa i de formulari.
