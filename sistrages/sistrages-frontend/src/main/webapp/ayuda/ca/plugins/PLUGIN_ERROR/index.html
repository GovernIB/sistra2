PLUGIN_ERROR

DESCRIPCIÓ

  • Plugin que permet establir errors a l'execució d'un script.
  • Es forçarà la sortida del tràmit indicant un missatge d'error genèric. El missatge d'error establert a través del plugin es visualitzarà a la finestra de debug (des del navegador amb la sessió oberta de la tramitació fallida, s'ha d'obrir una altra pipella al navegador i accedir a l'adreça següent: /sistramitfront/debug/logSesionTramitacion.html)
