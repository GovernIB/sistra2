setTextoMensajeAviso(String textoMensajeAviso)

DESCRIPCIÓ

  • Mètode per establir directament el missatge d'avís.

PARÀMETRES D'ENTRADA

  • textoMensajeAviso: missatge a establir.

EXEMPLES

Suposam que volem mostrar un text de missatge per script i no mitjançant els missatges de pantalla:

PLUGIN_VALIDACION.setTextoMensajeAviso('Codi introduït repetit');
