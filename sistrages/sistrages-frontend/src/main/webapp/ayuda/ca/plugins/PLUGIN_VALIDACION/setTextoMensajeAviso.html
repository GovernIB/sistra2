setTextoMensajeAviso(String textoMensajeAviso)

DESCRIPCIÓN

  • Método para establecer directamente el mesaje de aviso.

PARÁMETROS DE ENTRADA

  • textoMensajeAviso: mensaje a establecer.

EJEMPLOS

Suponiendo que deseamos mostrar un texto de mensaje por script y no por la pestaña de mensajes:

PLUGIN_VALIDACION.setTextoMensajeAviso('Código introducido repetido');
