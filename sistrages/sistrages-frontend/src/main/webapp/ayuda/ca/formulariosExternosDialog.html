MANTENIMENT GESTOR EXTERN DE FORMULARIS

A la pantalla de 'Manteniment de gestors externs ' pot mantenir les dades corresponents a un gestor extern de formularis.
Els camps obligatoris es presenten amb un asterisc *

CAMPS

  • Identificador: codi único identificador del gestor.
  • Descripció: text descriptiu del gestor.
  • URL: text que indica la url del gestor extern. Si existeix un placeholder (per haver seleccionat una variable d'àrea), aquest serà substituït pel contingut d'aquesta variable, que serà inserit al començament d'aquest camp. Per exemple, si en la variable VAR01 apareix la URL "https://se.caib.es" i la URL d'aquest camp ha de finalitzar amb "proyectoBase/services/dominios/", en aquest camp apareixerà el text "{@@VAR01@@}/proyectoBase/services/dominios/".
  • Selecció Variable Àrea: aquest botó obre una pantalla que permet seleccionar una variable d'àrea que afegirà un placeholder que serà substituït per la url indicada en aquesta variable d'àrea a l'hora d'executar el tràmit que utilitzi aquest gestor extern de formulari.

OPERATIVA

  • Desar: cierra la pantalla guardando los cambios.
  • Cancel·lar: tanca la pantalla sense desar els canvis.
