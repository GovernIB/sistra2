ESBORRAT DE SCRIPTS

En la pantalla de 'Esborrat de scripts' pot seleccionar els scripts dels respectius passos que desitja esborrar de la versió.

SCRIPTS SELECCIONABLES

  • Esborrar scripts propietats: esborra els scripts de l'apartat de propietats d'una versió.
  • Esborrar scripts pas omplir: esborra els scripts del pas d'emplenar i dels formularis.
  • Esborrar scripts pas annex: esborra els scripts del pas d'annex i dels annexos.
  • Esborrar scripts pas taxes: esborra els scripts del pas de taxes i de les taxes.
  • Esborrar scripts pas registrar: esborra els scripts del pas de registrar.
  • Esborrar script pas captura: esborra els scripts del pas de captura.

OPERATIVA

  • Acceptar: tanca la pantalla esborrant els scripts seleccionats.
  • Cancel·lar: tanca la pantalla sense esborrar cap script.
